# 9장. 충돌 감지
게임에서 두 오브젝트가 서로 만났는지 판단하는 것은 매우 중요하다. 플레이어가 아이템을 줍는지, 적이 플레이어를 맞히는지, 벽에 부딪히는지 등 모든 상호작용이 충돌 감지를 기반으로 한다. 이 장에서는 게임에서 가장 많이 사용되는 두 가지 충돌 감지 방식인 사각형 충돌(AABB)과 원형 충돌을 학습하고, 충돌이 감지되었을 때 어떻게 대응할지를 배운다.
  

## 9.1 사각형 충돌 (AABB)

### AABB 충돌의 개념
AABB는 Axis-Aligned Bounding Box의 약자로, 축에 정렬된 경계 상자라는 뜻이다. 즉, 회전하지 않은 사각형 두 개가 겹치는지를 판단하는 방식이다. 이 방식은 계산이 간단하고 빠르므로 대부분의 게임에서 사용된다.

```
┌─────────────────────────────────────┐
│        AABB 충돌 감지 원리            │
│                                     │
│  충돌하지 않음:                       │
│  ┌────┐        ┌────┐               │
│  │    │        │    │               │
│  └────┘        └────┘               │
│                                     │
│  충돌함:                             │
│      ┌────┐                         │
│      │┌──┼┐                         │
│      ││  ││                         │
│      └┼──┼┘                         │
│       └────┘                        │
│                                     │
│  한 점에서 만남 (충돌 아님):           │
│  ┌────┐┌────┐                       │
│  │    ││    │                       │
│  └────┘└────┘                       │
└─────────────────────────────────────┘
```

AABB 충돌 감지의 원리는 간단하다. 두 사각형이 겹치려면 X축과 Y축 모두에서 겹쳐야 한다. 하나의 축에서도 겹치지 않으면 충돌하지 않은 것이다.

### 기본 AABB 충돌 감지
가장 기본적인 AABB 충돌 감지를 구현해보자.

```csharp
public class GameObject
{
    public Vector2 Position { get; set; }
    public int Width { get; set; }
    public int Height { get; set; }
    public Texture2D Texture { get; set; }

    public Rectangle GetBounds()
    {
        return new Rectangle(
            (int)Position.X,
            (int)Position.Y,
            Width,
            Height
        );
    }

    public bool Intersects(GameObject other)
    {
        return GetBounds().Intersects(other.GetBounds());
    }
}
```

MonoGame의 Rectangle 클래스는 이미 Intersects 메서드를 제공하므로, 우리는 이를 활용할 수 있다. GetBounds 메서드는 오브젝트의 현재 위치와 크기를 기반으로 Rectangle을 반환한다.

```csharp
// 충돌 감지 사용 예제
GameObject player = new GameObject();
GameObject enemy = new GameObject();

if (player.Intersects(enemy))
{
    // 충돌했을 때 처리
    player.Health -= 10;
}
```

### 중심점 기반 AABB
많은 경우 오브젝트의 위치가 중심점이 아닌 왼쪽 위 모서리로 표현된다. 이 경우 GetBounds를 조정해야 한다.

```csharp
public class GameObject
{
    public Vector2 Position { get; set; }  // 중심점
    public int Width { get; set; }
    public int Height { get; set; }
    public Texture2D Texture { get; set; }

    // 중심점 기반 경계 상자
    public Rectangle GetBounds()
    {
        return new Rectangle(
            (int)(Position.X - Width / 2),
            (int)(Position.Y - Height / 2),
            Width,
            Height
        );
    }

    public bool Intersects(GameObject other)
    {
        return GetBounds().Intersects(other.GetBounds());
    }
}
```

### 충돌 깊이 계산
두 사각형이 얼마나 깊이 겹쳐있는지를 계산하는 것이 유용할 때가 있다. 이를 통해 충돌 응답 처리를 더 정밀하게 할 수 있다.

```csharp
public class CollisionInfo
{
    public bool IsColliding { get; set; }
    public float OverlapX { get; set; }
    public float OverlapY { get; set; }
    public Vector2 Overlap => new Vector2(OverlapX, OverlapY);
}

public static class CollisionDetection
{
    public static CollisionInfo GetAABBCollision(GameObject obj1, GameObject obj2)
    {
        Rectangle rect1 = obj1.GetBounds();
        Rectangle rect2 = obj2.GetBounds();

        if (!rect1.Intersects(rect2))
        {
            return new CollisionInfo { IsColliding = false };
        }

        // 겹치는 영역 계산
        int overlapLeft = Math.Max(rect1.Left, rect2.Left);
        int overlapRight = Math.Min(rect1.Right, rect2.Right);
        int overlapTop = Math.Max(rect1.Top, rect2.Top);
        int overlapBottom = Math.Min(rect1.Bottom, rect2.Bottom);

        float overlapX = overlapRight - overlapLeft;
        float overlapY = overlapBottom - overlapTop;

        return new CollisionInfo
        {
            IsColliding = true,
            OverlapX = overlapX,
            OverlapY = overlapY
        };
    }
}
```

### AABB 충돌과 화면 경계
화면 경계 충돌도 AABB로 처리할 수 있다.

```csharp
public class GameObject
{
    public Vector2 Position { get; set; }
    public Vector2 Velocity { get; set; }
    public int Width { get; set; }
    public int Height { get; set; }

    public Rectangle GetBounds()
    {
        return new Rectangle(
            (int)Position.X,
            (int)Position.Y,
            Width,
            Height
        );
    }

    public void CheckScreenCollision(int screenWidth, int screenHeight)
    {
        Rectangle bounds = GetBounds();

        // 좌측 경계
        if (bounds.Left < 0)
        {
            Position.X = 0;
            Velocity.X = 0;  // 또는 속도를 반사시킬 수 있음
        }

        // 우측 경계
        if (bounds.Right > screenWidth)
        {
            Position.X = screenWidth - Width;
            Velocity.X = 0;
        }

        // 상단 경계
        if (bounds.Top < 0)
        {
            Position.Y = 0;
            Velocity.Y = 0;
        }

        // 하단 경계
        if (bounds.Bottom > screenHeight)
        {
            Position.Y = screenHeight - Height;
            Velocity.Y = 0;
        }
    }
}
```

## 9.2 원형 충돌

### 원형 충돌의 개념
원형 충돌은 두 원이 겹치는지를 판단한다. 이 방식은 AABB보다 정확하지만 계산이 조금 더 복잡하다. 특히 공, 행성, 폭발 같은 원형 오브젝트에 적합하다.

```
┌─────────────────────────────────────┐
│        원형 충돌 감지 원리             │
│                                     │
│  충돌하지 않음:                       │
│       ○              ○              │
│    (r1)            (r2)             │
│  distance > r1 + r2                 │
│                                     │
│  충돌함:                             │
│        ○○                           │
│      (r1)(r2)                       │
│  distance < r1 + r2                 │
│                                     │
│  한 점에서 만남:                      │
│        ○ ○                          │ 
│      (r1)(r2)                       │
│  distance = r1 + r2                 │
└─────────────────────────────────────┘
```

원형 충돌의 원리는 두 원의 중심 사이 거리를 계산하고, 이를 두 원의 반지름의 합과 비교한다. 거리가 반지름의 합보다 작거나 같으면 충돌한 것이다.

### 기본 원형 충돌 감지

```csharp
public class CircleObject
{
    public Vector2 Position { get; set; }  // 중심점
    public float Radius { get; set; }
    public Texture2D Texture { get; set; }

    // 두 원 사이의 거리 기반 충돌 감지
    public bool Intersects(CircleObject other)
    {
        float distance = Vector2.Distance(Position, other.Position);
        return distance < (Radius + other.Radius);
    }

    // 정확한 충돌 깊이 계산
    public float GetCollisionDepth(CircleObject other)
    {
        float distance = Vector2.Distance(Position, other.Position);
        float minDistance = Radius + other.Radius;
        return minDistance - distance;
    }
}
```

### AABB와 원형 혼합 충돌
게임에서는 AABB와 원형 충돌을 함께 사용하는 경우가 많다. 이 경우 사각형과 원이 충돌하는지를 판단해야 한다.

```csharp
public static class CollisionDetection
{
    // 사각형과 원의 충돌 감지
    public static bool CircleRectangleIntersects(
        Vector2 circleCenter, float circleRadius,
        Rectangle rectangle)
    {
        // 원의 중심에서 사각형까지의 최단 거리 찾기
        float closestX = MathHelper.Clamp(
            circleCenter.X,
            rectangle.Left,
            rectangle.Right
        );

        float closestY = MathHelper.Clamp(
            circleCenter.Y,
            rectangle.Top,
            rectangle.Bottom
        );

        // 최단 거리 계산
        float distanceX = circleCenter.X - closestX;
        float distanceY = circleCenter.Y - closestY;

        float distanceSquared = (distanceX * distanceX) +
                               (distanceY * distanceY);

        return distanceSquared < (circleRadius * circleRadius);
    }
}
```

### 원형 충돌의 법선 벡터
충돌 응답 처리를 위해서는 충돌 시점의 법선 벡터를 계산해야 한다. 원형 충돌에서 법선 벡터는 두 원의 중심을 연결하는 벡터다.

```csharp
public class CircleObject
{
    public Vector2 Position { get; set; }
    public float Radius { get; set; }
    public Vector2 Velocity { get; set; }

    // 충돌 법선 벡터 계산
    public Vector2 GetCollisionNormal(CircleObject other)
    {
        Vector2 direction = other.Position - Position;
        if (direction.Length() > 0)
        {
            return Vector2.Normalize(direction);
        }
        return Vector2.UnitX;
    }

    // 충돌 시점 계산 (두 원이 만나는 지점)
    public Vector2 GetCollisionPoint(CircleObject other)
    {
        Vector2 normal = GetCollisionNormal(other);
        return Position + normal * Radius;
    }
}
```

## 9.3 충돌 응답 처리

### 충돌 후 처리 전략
충돌이 감지되었을 때 어떻게 대응할지는 게임의 종류에 따라 다르다. 주요 충돌 응답 방식은 다음과 같다.

```
┌────────────────────────────────────┐
│     충돌 응답 처리 전략               │
│                                    │
│ 1. 무시 (Ignore)                    │
│    - 충돌을 감지하지만 반응 없음       │
│    - 예: 투명한 벽                   │
│                                    │
│ 2. 반사 (Bounce)                    │
│    - 속도 방향 반대로 반사            │
│    - 예: 탁구공, 공 게임             │
│                                    │
│ 3. 관통 불가 (Solid)              │
│    - 위치를 조정하여 겹침 방지      │
│    - 예: 벽, 플랫폼                │
│                                    │
│ 4. 상호작용 (Interaction)         │
│    - 특정 로직 실행                │
│    - 예: 아이템 획득, 데미지       │
└────────────────────────────────────┘
```

### 기본 충돌 응답

```csharp
public class Player : GameObject
{
    public int Health { get; set; } = 100;

    public void OnCollisionWithEnemy(Enemy enemy)
    {
        // 접촉 데미지
        Health -= 10;

        // 플레이어 튕겨내기
        Vector2 knockback = Vector2.Normalize(Position - enemy.Position) * 200;
        Velocity = knockback;
    }

    public void OnCollisionWithItem(Item item)
    {
        // 아이템 효과 적용
        switch (item.Type)
        {
            case ItemType.HealthPotion:
                Health = Math.Min(Health + 50, 100);
                break;
            case ItemType.SpeedBoost:
                // 속도 증가 효과
                break;
        }

        // 아이템 제거
        item.IsActive = false;
    }
}
```

### 벽 충돌 처리
벽과의 충돌은 위치 조정이 필요하다. 오브젝트가 벽을 통과하지 않도록 해야 한다.

```csharp
public static class CollisionResponse
{
    // 움직이는 오브젝트와 벽의 충돌 응답
    public static void ResolveWallCollision(
        GameObject movingObject,
        Rectangle wall)
    {
        Rectangle bounds = movingObject.GetBounds();
        CollisionInfo collision = GetAABBCollision(movingObject, wall);

        if (!collision.IsColliding)
            return;

        // 겹침이 적은 축 방향으로 밀어냄
        if (collision.OverlapX < collision.OverlapY)
        {
            // X축 충돌 처리
            if (bounds.Center.X < wall.Center.X)
            {
                // 왼쪽에서 충돌
                movingObject.Position.X = wall.Left - bounds.Width / 2;
            }
            else
            {
                // 오른쪽에서 충돌
                movingObject.Position.X = wall.Right + bounds.Width / 2;
            }
            movingObject.Velocity.X = 0;
        }
        else
        {
            // Y축 충돌 처리
            if (bounds.Center.Y < wall.Center.Y)
            {
                // 위에서 충돌
                movingObject.Position.Y = wall.Top - bounds.Height / 2;
            }
            else
            {
                // 아래에서 충돌
                movingObject.Position.Y = wall.Bottom + bounds.Height / 2;
            }
            movingObject.Velocity.Y = 0;
        }
    }
}
```

### 탄성 충돌 (원형 오브젝트)
두 공이 충돌할 때 물리적으로 정확한 탄성 충돌을 구현할 수 있다.

```csharp
public static class PhysicsCollision
{
    // 두 원형 오브젝트의 탄성 충돌
    public static void ResolveElasticCollision(
        CircleObject obj1,
        CircleObject obj2)
    {
        // 충돌 법선
        Vector2 normal = obj2.Position - obj1.Position;
        if (normal.Length() == 0)
            normal = Vector2.UnitX;
        else
            normal = Vector2.Normalize(normal);

        // 상대 속도
        Vector2 relativeVelocity = obj1.Velocity - obj2.Velocity;

        // 법선 방향 상대 속도
        float velocityAlongNormal = Vector2.Dot(relativeVelocity, normal);

        // 이미 분리 중이면 충돌 안 함
        if (velocityAlongNormal > 0)
            return;

        // 반발 계수 (탄성 정도)
        float restitution = 0.8f;

        // 충격 크기 계산 (같은 질량 가정)
        float impulseMagnitude = -(1 + restitution) * velocityAlongNormal / 2;

        // 충격 벡터
        Vector2 impulse = normal * impulseMagnitude;

        // 속도 업데이트
        obj1.Velocity += impulse;
        obj2.Velocity -= impulse;

        // 겹침 분리
        float minDistance = obj1.Radius + obj2.Radius;
        float distance = Vector2.Distance(obj1.Position, obj2.Position);
        float overlap = minDistance - distance;

        if (overlap > 0)
        {
            Vector2 separation = normal * (overlap / 2);
            obj1.Position -= separation;
            obj2.Position += separation;
        }
    }
}
```

### 종합 충돌 처리 예제
다음은 여러 종류의 충돌을 종합적으로 처리하는 예제다.  

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Graphics;
using System;
using System.Collections.Generic;

namespace CollisionHandlingExample
{
    public class CollisionManager
    {
        private List<GameObject> _gameObjects;
        private List<Rectangle> _walls;

        public CollisionManager()
        {
            _gameObjects = new List<GameObject>();
            _walls = new List<Rectangle>();
        }

        public void Update()
        {
            // 1. 모든 오브젝트 간의 충돌 감지
            for (int i = 0; i < _gameObjects.Count; i++)
            {
                for (int j = i + 1; j < _gameObjects.Count; j++)
                {
                    GameObject obj1 = _gameObjects[i];
                    GameObject obj2 = _gameObjects[j];

                    if (obj1.Intersects(obj2))
                    {
                        HandleObjectCollision(obj1, obj2);
                    }
                }
            }

            // 2. 오브젝트와 벽의 충돌 감지
            foreach (GameObject obj in _gameObjects)
            {
                foreach (Rectangle wall in _walls)
                {
                    Rectangle bounds = obj.GetBounds();
                    if (bounds.Intersects(wall))
                    {
                        HandleWallCollision(obj, wall);
                    }
                }
            }

            // 3. 화면 경계 충돌
            foreach (GameObject obj in _gameObjects)
            {
                obj.CheckScreenCollision(800, 600);
            }
        }

        private void HandleObjectCollision(GameObject obj1, GameObject obj2)
        {
            // 타입에 따른 충돌 처리
            if (obj1 is Player && obj2 is Enemy)
            {
                ((Player)obj1).OnCollisionWithEnemy((Enemy)obj2);
            }
            else if (obj1 is Enemy && obj2 is Player)
            {
                ((Player)obj2).OnCollisionWithEnemy((Enemy)obj1);
            }
            else if (obj1 is Player && obj2 is Item)
            {
                ((Player)obj1).OnCollisionWithItem((Item)obj2);
            }
            else if (obj1 is Item && obj2 is Player)
            {
                ((Player)obj2).OnCollisionWithItem((Item)obj1);
            }
            // 추가 충돌 타입들...
        }

        private void HandleWallCollision(GameObject obj, Rectangle wall)
        {
            Rectangle bounds = obj.GetBounds();

            // 겹침이 적은 축으로 분리
            float overlapX = Math.Min(bounds.Right, wall.Right) -
                            Math.Max(bounds.Left, wall.Left);
            float overlapY = Math.Min(bounds.Bottom, wall.Bottom) -
                            Math.Max(bounds.Top, wall.Top);

            if (overlapX < overlapY)
            {
                // X축 충돌
                if (bounds.Center.X < wall.Center.X)
                {
                    obj.Position.X = wall.Left - bounds.Width / 2;
                }
                else
                {
                    obj.Position.X = wall.Right + bounds.Width / 2;
                }
                obj.Velocity.X = 0;
            }
            else
            {
                // Y축 충돌
                if (bounds.Center.Y < wall.Center.Y)
                {
                    obj.Position.Y = wall.Top - bounds.Height / 2;
                    // 위에서 착지
                    obj.IsGrounded = true;
                }
                else
                {
                    obj.Position.Y = wall.Bottom + bounds.Height / 2;
                }
                obj.Velocity.Y = 0;
            }
        }

        public void AddGameObject(GameObject obj)
        {
            _gameObjects.Add(obj);
        }

        public void AddWall(Rectangle wall)
        {
            _walls.Add(wall);
        }
    }

    // 기본 GameObject 클래스
    public abstract class GameObject
    {
        public Vector2 Position { get; set; }
        public Vector2 Velocity { get; set; }
        public int Width { get; set; }
        public int Height { get; set; }
        public bool IsGrounded { get; set; }
        public Texture2D Texture { get; set; }

        public Rectangle GetBounds()
        {
            return new Rectangle(
                (int)(Position.X - Width / 2),
                (int)(Position.Y - Height / 2),
                Width,
                Height
            );
        }

        public bool Intersects(GameObject other)
        {
            return GetBounds().Intersects(other.GetBounds());
        }

        public void CheckScreenCollision(int width, int height)
        {
            Rectangle bounds = GetBounds();
            if (bounds.Left < 0)
                Position.X = Width / 2;
            if (bounds.Right > width)
                Position.X = width - Width / 2;
            if (bounds.Top < 0)
                Position.Y = Height / 2;
            if (bounds.Bottom > height)
                Position.Y = height - Height / 2;
        }

        public abstract void Update(GameTime gameTime);
        public abstract void Draw(SpriteBatch spriteBatch);
    }

    // Player 클래스
    public class Player : GameObject
    {
        public int Health { get; set; } = 100;

        public void OnCollisionWithEnemy(Enemy enemy)
        {
            Health -= 10;
            Vector2 knockback = Vector2.Normalize(Position - enemy.Position) * 300;
            Velocity = knockback;
        }

        public void OnCollisionWithItem(Item item)
        {
            Health = Math.Min(Health + 50, 100);
            item.IsActive = false;
        }

        public override void Update(GameTime gameTime)
        {
            // 플레이어 업데이트 로직
        }

        public override void Draw(SpriteBatch spriteBatch)
        {
            if (Texture != null)
            {
                spriteBatch.Draw(Texture, Position, Color.Blue);
            }
        }
    }

    // Enemy 클래스
    public class Enemy : GameObject
    {
        public override void Update(GameTime gameTime)
        {
            // 적 업데이트 로직
        }

        public override void Draw(SpriteBatch spriteBatch)
        {
            if (Texture != null)
            {
                spriteBatch.Draw(Texture, Position, Color.Red);
            }
        }
    }

    // Item 클래스
    public class Item : GameObject
    {
        public bool IsActive { get; set; } = true;

        public override void Update(GameTime gameTime)
        {
            // 아이템 업데이트 로직
        }

        public override void Draw(SpriteBatch spriteBatch)
        {
            if (Texture != null && IsActive)
            {
                spriteBatch.Draw(Texture, Position, Color.Yellow);
            }
        }
    }
}
```

이 코드는 전체적인 충돌 관리 시스템을 보여준다. CollisionManager 클래스는 모든 게임 오브젝트를 관리하고 충돌을 감지한다. Update 메서드에서 세 가지 종류의 충돌을 검사한다.

첫째, 모든 오브젝트 간의 충돌을 확인한다. 이중 루프를 사용하여 각 오브젝트 쌍을 비교한다. O(n²) 시간이 걸리지만, 오브젝트 개수가 적을 때는 문제없다.

둘째, 각 오브젝트와 벽의 충돌을 확인한다. 벽과 충돌할 때는 겹침이 적은 축 방향으로 오브젝트를 밀어낸다. X축과 Y축 겹침 크기를 비교하여 어느 방향에서 충돌했는지 판단한다.

셋째, 화면 경계 충돌을 확인한다. 오브젝트가 화면을 벗어나지 않도록 위치를 조정한다.

HandleObjectCollision 메서드는 충돌한 두 오브젝트의 타입을 확인하고, 해당하는 충돌 처리를 실행한다. Player와 Enemy가 충돌하면 플레이어가 데미지를 받고 튕겨난다.

HandleWallCollision 메서드는 벽 충돌 응답을 처리한다. 겹침의 크기를 비교하여 어느 축에서 충돌했는지 판단하고, 그에 따라 오브젝트 위치를 조정한다. Y축 충돌인 경우 아래에서 착지했을 때 IsGrounded를 true로 설정하여 점프 등의 로직에 사용할 수 있다.

### 과제 1: AABB 충돌 감지 구현
다음 기능을 가진 CollisionDetector 클래스를 만들자.

```csharp
public class CollisionDetector
{
    // 두 Rectangle의 충돌 감지
    public static bool IsColliding(Rectangle rect1, Rectangle rect2)
    
    // 충돌 깊이 반환 (음수면 미충돌)
    public static Vector2 GetCollisionDepth(Rectangle rect1, Rectangle rect2)
    
    // 충돌 방향 반환 (위, 아래, 왼쪽, 오른쪽)
    public static CollisionDirection GetCollisionDirection(
        Rectangle rect1, Rectangle rect2)
    
    // 점이 사각형 내에 있는지 확인
    public static bool IsPointInRectangle(Vector2 point, Rectangle rect)
}
```

**요구사항:**
- CollisionDirection은 enum으로 정의한다
- 충돌 깊이는 겹치는 영역의 크기를 나타낸다
- 충돌 방향은 rect1의 관점에서 rect2가 어느 방향에서 충돌했는지를 나타낸다

### 과제 2: 원형 충돌 감지 구현
원형 충돌을 감지하고 처리하는 다음 기능들을 구현하자.

```csharp
public class CircleCollisionDetector
{
    // 두 원의 충돌 감지
    public static bool IsColliding(Vector2 center1, float radius1, 
                                   Vector2 center2, float radius2)
    
    // 충돌 깊이 계산
    public static float GetCollisionDepth(Vector2 center1, float radius1,
                                         Vector2 center2, float radius2)
    
    // 원과 사각형의 충돌 감지
    public static bool CircleRectangleCollision(Vector2 circleCenter, 
                                               float radius,
                                               Rectangle rectangle)
}
```

**요구사항:**
- 거리 계산에 Vector2.Distance 사용
- 원과 사각형의 충돌은 사각형의 가장 가까운 점을 찾아 판정한다

### 과제 3: 충돌 응답 시스템
충돌이 감지된 후 처리하는 시스템을 만들자.

**요구사항:**
- CollisionHandler 클래스를 만든다
- 다양한 타입의 충돌 응답을 처리한다
  - 벽 충돌: 위치 조정, 속도 제거
  - 아이템 수집: 아이템 비활성화, 점수 증가
  - 적과의 접촉: 데미지, 넉백
- 충돌 이벤트를 발생시킨다

### 과제 4: 플랫포머 게임 충돌 처리
플랫폼 게임의 중력과 플랫폼 충돌을 구현하자.

**요구사항:**
- 플레이어에게 중력을 적용한다
- 플랫폼에 착지하면 IsGrounded를 true로 설정한다
- 플레이어는 IsGrounded가 true일 때만 점프할 수 있다
- 플랫폼 밑에서 부딪히면 IsGrounded를 false로 설정하고 속도를 0으로 한다
- 여러 플랫폼을 배치하고 플레이어가 이동할 수 있게 한다

### 과제 5: 게임 구현 - 간단한 슛 게임
이 장에서 배운 충돌 감지를 사용하여 다음 게임을 만들자.

- 플레이어는 화면 하단에 위치하고 왼쪽/오른쪽으로 이동한다
- 마우스 클릭으로 탄환을 발사한다
- 화면 상단에서 적들이 내려온다
- 탄환이 적과 충돌하면 적이 제거된다
- 적이 화면 하단에 도달하면 생명 감소
- 생명이 0이 되면 게임 오버

**요구사항:**
- Player, Enemy, Bullet 클래스를 만든다
- 모든 충돌을 감지하고 처리한다
- 충돌 시 시각적 효과를 표현한다 (색상 변경 등)
- 현재 생명, 점수, 처치한 적 수를 표시한다
- 웨이브 시스템을 구현한다 (점수에 따라 적의 속도와 개수 증가)
