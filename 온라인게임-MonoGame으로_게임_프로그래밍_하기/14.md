지난 두 장에서는 벽돌깨기 게임의 기본 요소들을 구현했다. 이제 게임을 완성하기 위해 게임 오버와 클리어 조건 처리, UI 구성, 사운드 통합, 그리고 최종 테스트를 진행하겠다. 이 장을 마치면 완전히 동작하는 벽돌깨기 게임을 갖게 될 것이다.

## 14.1 게임 오버와 클리어
게임의 종료 조건은 두 가지다. 플레이어의 생명이 모두 떨어지면 게임 오버가 되고, 모든 벽돌을 깨뜨리면 게임을 클리어한다. 이런 상태 변화를 관리하는 것이 중요하다.

### 게임 상태 열거형
게임의 여러 상태를 관리하기 위해 열거형을 정의한다.

```csharp
public enum GameStateType
{
    Playing,      // 게임 진행 중
    GameOver,     // 게임 오버
    Clear,        // 게임 클리어
    Paused        // 일시 중지
}
```

### 게임 상태 관리 클래스 확장
이전 장의 `GameState` 클래스를 확장하여 게임 상태와 타이머를 관리하자.

```csharp
public class GameState
{
    public int Score { get; set; }
    public int Lives { get; set; }
    public int Level { get; set; }
    public int BricksDestroyed { get; set; }
    public int TotalBricks { get; set; }
    public GameStateType CurrentState { get; set; }
    public double StateChangeTime { get; set; }

    public GameState()
    {
        Score = 0;
        Lives = 3;
        Level = 1;
        BricksDestroyed = 0;
        TotalBricks = 0;
        CurrentState = GameStateType.Playing;
        StateChangeTime = 0;
    }

    public void AddScore(int points)
    {
        Score += points;
    }

    public void DecreaseLives()
    {
        if (Lives > 0)
        {
            Lives--;
        }
    }

    public bool IsGameOver()
    {
        return Lives <= 0;
    }

    public bool IsClear()
    {
        return BricksDestroyed == TotalBricks && TotalBricks > 0;
    }

    public void SetGameOver()
    {
        CurrentState = GameStateType.GameOver;
        StateChangeTime = 0;
    }

    public void SetClear()
    {
        CurrentState = GameStateType.Clear;
        StateChangeTime = 0;
    }

    public void SetPlaying()
    {
        CurrentState = GameStateType.Playing;
    }

    public void SetPaused()
    {
        if (CurrentState == GameStateType.Playing)
        {
            CurrentState = GameStateType.Paused;
        }
    }

    public void Resume()
    {
        if (CurrentState == GameStateType.Paused)
        {
            CurrentState = GameStateType.Playing;
        }
    }

    public void Reset()
    {
        Score = 0;
        Lives = 3;
        Level = 1;
        BricksDestroyed = 0;
        CurrentState = GameStateType.Playing;
        StateChangeTime = 0;
    }

    public void NextLevel()
    {
        Level++;
        BricksDestroyed = 0;
        CurrentState = GameStateType.Playing;
    }

    public void UpdateStateTime(GameTime gameTime)
    {
        StateChangeTime += gameTime.ElapsedGameTime.TotalSeconds;
    }
}
```

**코드 설명:**

`GameState` 클래스에 게임 상태를 추적하는 기능을 추가했다. `CurrentState` 속성은 게임의 현재 상태(플레이 중, 게임 오버, 클리어, 일시 중지)를 나타낸다. `StateChangeTime`은 현재 상태가 지속된 시간을 기록하여, 예를 들어 게임 오버 화면을 일정 시간 표시한 후 자동으로 다음 동작을 수행할 때 사용한다.

`SetGameOver`, `SetClear`, `SetPlaying` 메서드들은 게임 상태를 변경한다. `Reset` 메서드는 게임을 처음부터 시작할 때 모든 상태를 초기화한다. `NextLevel` 메서드는 다음 레벨로 진행할 때 벽돌 파괴 카운트를 초기화하고 레벨을 증가시킨다.

### 게임 오버와 클리어 처리
게임의 메인 `Update` 메서드를 수정하여 게임 상태 변화를 처리한다.

```csharp
private Paddle paddle;
private Ball ball;
private List<Brick> bricks;
private GameState gameState;
private SpriteFont gameFont;

protected override void Update(GameTime gameTime)
{
    if (GamePad.GetState(PlayerIndex.One).Buttons.Back == ButtonState.Pressed 
        || Keyboard.GetState().IsKeyDown(Keys.Escape))
        Exit();

    // 일시 중지 처리
    KeyboardState keyboardState = Keyboard.GetState();
    if (keyboardState.IsKeyDown(Keys.P))
    {
        gameState.SetPaused();
    }
    if (keyboardState.IsKeyDown(Keys.R) && 
        (gameState.CurrentState == GameStateType.GameOver || 
         gameState.CurrentState == GameStateType.Clear))
    {
        gameState.Reset();
        InitializeBricks();
    }

    // 게임 진행 중일 때만 업데이트
    if (gameState.CurrentState == GameStateType.Playing)
    {
        paddle.Update(gameTime);
        ball.Update(gameTime);

        // 공과 패들의 충돌 감지
        if (ball.IsLaunched && 
            ball.GetBounds().Intersects(paddle.GetBounds()))
        {
            ball.ReflectOffPaddle(paddle);
        }

        // 공과 벽돌의 충돌 감지
        foreach (Brick brick in bricks)
        {
            if (!brick.IsDestroyed && 
                CollisionDetection.CircleRectangleCollision(
                    ball.Position, ball.Radius, brick.GetBounds()))
            {
                brick.Destroy();
                gameState.AddScore(brick.Points);
                gameState.BricksDestroyed++;
                
                CollisionDetection.CollisionDirection direction = 
                    CollisionDetection.GetCollisionDirection(
                        ball.Position, ball.Radius, brick.GetBounds());
                
                ReflectBallFromBrick(direction);
            }
        }

        // 공이 화면 하단으로 떨어졌을 때
        if (ball.Position.Y > 720)
        {
            gameState.DecreaseLives();
            
            if (gameState.IsGameOver())
            {
                gameState.SetGameOver();
            }
            else
            {
                ball.ResetBall();
            }
        }

        // 모든 벽돌이 파괴되었을 때
        if (gameState.IsClear())
        {
            gameState.SetClear();
        }
    }
    else if (gameState.CurrentState == GameStateType.Paused)
    {
        // 일시 중지 상태에서는 R 키로 재개
        if (keyboardState.IsKeyDown(Keys.R))
        {
            gameState.Resume();
        }
    }
    else if (gameState.CurrentState == GameStateType.GameOver || 
             gameState.CurrentState == GameStateType.Clear)
    {
        // 게임 오버 또는 클리어 상태 지속 시간 업데이트
        gameState.UpdateStateTime(gameTime);

        // 2초 후 자동으로 게임 재시작 요청 표시
        if (gameState.StateChangeTime > 2.0)
        {
            // UI에서 "R을 눌러 재시작"이라는 메시지를 표시함
        }
    }

    base.Update(gameTime);
}
```

**코드 설명:**

게임의 핵심 로직이 `if (gameState.CurrentState == GameStateType.Playing)` 블록 안에 들어가므로, 게임이 진행 중일 때만 업데이트가 일어난다. 게임 오버나 클리어 상태에서는 게임 로직이 중지되고, 플레이어의 입력만 처리된다.

공이 화면 하단으로 떨어졌을 때, 단순히 `ball.ResetBall()`을 호출하는 대신, 먼저 생명을 감소시키고 생명이 0이 되었는지 확인한다. 생명이 남아 있으면 공을 리셋하고, 생명이 0이 되면 게임 오버 상태로 전환한다.

모든 벽돌이 파괴되었는지는 매 프레임마다 확인하고, 조건을 만족하면 게임 클리어 상태로 전환한다.

## 14.2 UI 추가
게임의 사용자 경험을 높이기 위해 다양한 UI 요소를 추가해야 한다. 점수, 생명, 게임 상태 메시지 등을 화면에 표시하자.

### UI 관리 클래스
UI를 관리하는 별도의 클래스를 만들어 모든 화면 출력을 중앙에서 관리한다.

```csharp
public class GameUI
{
    private SpriteFont _font;
    private SpriteFont _largeFont;
    private GraphicsDevice _graphicsDevice;

    public GameUI(SpriteFont font, SpriteFont largeFont, 
        GraphicsDevice graphicsDevice)
    {
        _font = font;
        _largeFont = largeFont;
        _graphicsDevice = graphicsDevice;
    }

    public void DrawGameplayUI(SpriteBatch spriteBatch, GameState gameState)
    {
        // 점수 표시 (왼쪽 위)
        string scoreText = $"Score: {gameState.Score}";
        spriteBatch.DrawString(_font, scoreText, new Vector2(10, 10), Color.White);

        // 생명 표시 (오른쪽 위)
        string livesText = $"Lives: {gameState.Lives}";
        spriteBatch.DrawString(_font, livesText, new Vector2(1100, 10), Color.White);

        // 레벨 표시 (중앙 위)
        string levelText = $"Level: {gameState.Level}";
        spriteBatch.DrawString(_font, levelText, new Vector2(560, 10), Color.White);

        // 안내 메시지
        string controlText = "P: Pause | Space: Launch Ball";
        spriteBatch.DrawString(_font, controlText, 
            new Vector2(10, 690), Color.Gray);

        // 남은 벽돌 수 표시
        int remainingBricks = gameState.TotalBricks - gameState.BricksDestroyed;
        string bricksText = $"Bricks: {remainingBricks}/{gameState.TotalBricks}";
        spriteBatch.DrawString(_font, bricksText, 
            new Vector2(500, 690), Color.White);
    }

    public void DrawGameOverScreen(SpriteBatch spriteBatch, GameState gameState)
    {
        // 반투명 검은색 오버레이
        DrawSemiTransparentRectangle(spriteBatch, 
            new Rectangle(0, 0, 1280, 720), Color.Black, 0.7f);

        // "GAME OVER" 텍스트
        string gameOverText = "GAME OVER";
        Vector2 gameOverSize = _largeFont.MeasureString(gameOverText);
        spriteBatch.DrawString(_largeFont, gameOverText,
            new Vector2(640 - gameOverSize.X / 2, 250), Color.Red);

        // 최종 점수
        string finalScoreText = $"Final Score: {gameState.Score}";
        Vector2 scoreSize = _font.MeasureString(finalScoreText);
        spriteBatch.DrawString(_font, finalScoreText,
            new Vector2(640 - scoreSize.X / 2, 350), Color.White);

        // 재시작 안내
        string restartText = "Press R to Restart";
        Vector2 restartSize = _font.MeasureString(restartText);
        spriteBatch.DrawString(_font, restartText,
            new Vector2(640 - restartSize.X / 2, 450), Color.Yellow);
    }

    public void DrawClearScreen(SpriteBatch spriteBatch, GameState gameState)
    {
        // 반투명 검은색 오버레이
        DrawSemiTransparentRectangle(spriteBatch, 
            new Rectangle(0, 0, 1280, 720), Color.Black, 0.7f);

        // "CLEAR!" 텍스트
        string clearText = "CLEAR!";
        Vector2 clearSize = _largeFont.MeasureString(clearText);
        spriteBatch.DrawString(_largeFont, clearText,
            new Vector2(640 - clearSize.X / 2, 250), Color.Green);

        // 클리어 점수
        string scoreText = $"Level Score: {gameState.Score}";
        Vector2 scoreSize = _font.MeasureString(scoreText);
        spriteBatch.DrawString(_font, scoreText,
            new Vector2(640 - scoreSize.X / 2, 350), Color.White);

        // 다음 레벨 안내
        string nextLevelText = "Press R for Next Level";
        Vector2 nextLevelSize = _font.MeasureString(nextLevelText);
        spriteBatch.DrawString(_font, nextLevelText,
            new Vector2(640 - nextLevelSize.X / 2, 450), Color.Yellow);
    }

    public void DrawPausedScreen(SpriteBatch spriteBatch)
    {
        // 반투명 검은색 오버레이
        DrawSemiTransparentRectangle(spriteBatch, 
            new Rectangle(0, 0, 1280, 720), Color.Black, 0.5f);

        // "PAUSED" 텍스트
        string pausedText = "PAUSED";
        Vector2 pausedSize = _largeFont.MeasureString(pausedText);
        spriteBatch.DrawString(_largeFont, pausedText,
            new Vector2(640 - pausedSize.X / 2, 300), Color.White);

        // 재개 안내
        string resumeText = "Press R to Resume";
        Vector2 resumeSize = _font.MeasureString(resumeText);
        spriteBatch.DrawString(_font, resumeText,
            new Vector2(640 - resumeSize.X / 2, 400), Color.Yellow);
    }

    private void DrawSemiTransparentRectangle(SpriteBatch spriteBatch,
        Rectangle rect, Color color, float alpha)
    {
        // 반투명 직사각형을 그리기 위해 임시 텍스처 생성
        Texture2D texture = new Texture2D(_graphicsDevice, 1, 1);
        Color[] data = new Color[1];
        data[0] = color * alpha;
        texture.SetData(data);

        spriteBatch.Draw(texture, rect, color * alpha);
    }
}
```

**코드 설명:**

`GameUI` 클래스는 게임의 모든 UI 요소를 담당한다. 생성자에서 폰트와 그래픽스 장치를 받아 저장하고, 여러 메서드를 통해 상황에 맞는 UI를 그린다.

`DrawGameplayUI` 메서드는 게임 진행 중에 필요한 정보들을 화면에 표시한다. 점수, 생명, 레벨, 조작 방법, 남은 벽돌 수 등을 다양한 위치에 배치하여 플레이어가 한눈에 게임 상태를 파악할 수 있게 한다.

`DrawGameOverScreen`, `DrawClearScreen`, `DrawPausedScreen` 메서드들은 각각의 상황에 맞는 메시지를 표시한다. 이들은 모두 반투명 오버레이를 사용하여 배경 게임을 흐릿하게 처리한 후, 중앙에 크게 메시지를 표시한다.

`DrawSemiTransparentRectangle` 메서드는 알파 값을 적용한 반투명 직사각형을 그린다. 이는 게임 화면 위에 반투명한 검은색 오버레이를 만들어서 사용자의 주의를 메시지에 집중시킨다.

### UI 통합
게임의 `Draw` 메서드를 수정하여 UI를 표시한다.

```csharp
private GameUI gameUI;

protected override void LoadContent()
{
    _spriteBatch = new SpriteBatch(GraphicsDevice);
    
    // 폰트 로드
    SpriteFont regularFont = Content.Load<SpriteFont>("Arial");
    SpriteFont largeFont = Content.Load<SpriteFont>("ArialLarge");
    
    gameUI = new GameUI(regularFont, largeFont, GraphicsDevice);
    
    gameState = new GameState();
    
    paddle = new Paddle(
        position: new Vector2(640, 650),
        width: 120,
        height: 20
    );
    
    ball = new Ball(
        position: new Vector2(640, 620),
        radius: 8,
        paddle: paddle
    );

    InitializeBricks();
    gameState.TotalBricks = bricks.Count;
}

protected override void Draw(GameTime gameTime)
{
    GraphicsDevice.Clear(Color.Black);

    _spriteBatch.Begin();
    
    // 벽돌 그리기
    foreach (Brick brick in bricks)
    {
        brick.Draw(_spriteBatch);
    }
    
    // 게임 오브젝트 그리기
    paddle.Draw(_spriteBatch);
    ball.Draw(_spriteBatch);
    
    // 게임 상태에 따른 UI 그리기
    if (gameState.CurrentState == GameStateType.Playing || 
        gameState.CurrentState == GameStateType.Paused)
    {
        gameUI.DrawGameplayUI(_spriteBatch, gameState);
    }

    if (gameState.CurrentState == GameStateType.GameOver)
    {
        gameUI.DrawGameOverScreen(_spriteBatch, gameState);
    }

    if (gameState.CurrentState == GameStateType.Clear)
    {
        gameUI.DrawClearScreen(_spriteBatch, gameState);
    }

    if (gameState.CurrentState == GameStateType.Paused)
    {
        gameUI.DrawPausedScreen(_spriteBatch);
    }
    
    _spriteBatch.End();

    base.Draw(gameTime);
}
```

## 14.3 사운드 통합
게임에 음향 효과와 배경 음악을 추가하면 플레이어의 몰입감이 크게 향상된다. 여러 상황에서 다른 사운드를 재생하도록 구현하자.

### 사운드 관리 클래스
게임의 모든 사운드를 관리하는 클래스를 만든다.

```csharp
public class SoundManager
{
    private SoundEffect _paddleHitSound;
    private SoundEffect _brickHitSound;
    private SoundEffect _wallHitSound;
    private SoundEffect _gameOverSound;
    private SoundEffect _clearSound;
    private Song _backgroundMusic;

    public SoundManager(ContentManager content)
    {
        // 사운드 이펙트 로드
        // (Content 폴더에 해당 파일이 있어야 함)
        try
        {
            _paddleHitSound = content.Load<SoundEffect>("Sounds/paddle_hit");
            _brickHitSound = content.Load<SoundEffect>("Sounds/brick_hit");
            _wallHitSound = content.Load<SoundEffect>("Sounds/wall_hit");
            _gameOverSound = content.Load<SoundEffect>("Sounds/game_over");
            _clearSound = content.Load<SoundEffect>("Sounds/level_clear");
            _backgroundMusic = content.Load<Song>("Music/background");
        }
        catch (ContentLoadException e)
        {
            System.Diagnostics.Debug.WriteLine($"사운드 파일 로드 실패: {e.Message}");
        }
    }

    public void PlayPaddleHit()
    {
        _paddleHitSound?.Play(volume: 0.7f, pitch: 0f, pan: 0f);
    }

    public void PlayBrickHit()
    {
        _brickHitSound?.Play(volume: 0.5f, pitch: 0f, pan: 0f);
    }

    public void PlayWallHit()
    {
        _wallHitSound?.Play(volume: 0.5f, pitch: 0f, pan: 0f);
    }

    public void PlayGameOver()
    {
        _gameOverSound?.Play(volume: 0.8f, pitch: 0f, pan: 0f);
    }

    public void PlayClear()
    {
        _clearSound?.Play(volume: 0.8f, pitch: 0f, pan: 0f);
    }

    public void PlayBackgroundMusic()
    {
        if (_backgroundMusic != null)
        {
            MediaPlayer.IsRepeating = true;
            MediaPlayer.Volume = 0.5f;
            MediaPlayer.Play(_backgroundMusic);
        }
    }

    public void StopBackgroundMusic()
    {
        MediaPlayer.Stop();
    }

    public void SetBackgroundMusicVolume(float volume)
    {
        MediaPlayer.Volume = MathHelper.Clamp(volume, 0f, 1f);
    }
}
```

**코드 설명:**

`SoundManager` 클래스는 게임의 모든 사운드를 중앙에서 관리한다. 생성자에서 Content 폴더에서 사운드 파일들을 로드하고, 각 상황에 맞는 메서드를 제공한다.

각 `Play` 메서드에서는 사운드 이펙트의 볼륨을 조절한다. 패들 충돌은 0.7, 벽돌 충돌은 0.5, 게임 오버와 클리어는 0.8로 설정하여 플레이어에게 적절한 청각 피드백을 제공한다.

`PlayBackgroundMusic` 메서드는 배경 음악을 반복 재생한다. `MediaPlayer.IsRepeating = true`를 설정하여 음악이 끝나면 자동으로 처음부터 다시 재생된다.

Try-catch 블록을 사용하여 사운드 파일이 없을 때의 에러를 처리한다. 이렇게 하면 사운드 파일이 없어도 게임이 제대로 작동한다.

### Ball 클래스에 사운드 통합
공의 움직임과 충돌 시 사운드를 재생하도록 Ball 클래스를 수정한다.

```csharp
public class Ball : GameObject
{
    public float Radius { get; set; }
    private float _speed;
    private bool _isLaunched;
    private Paddle _paddle;
    private SoundManager _soundManager;

    public Ball(Vector2 position, float radius, Paddle paddle, 
        SoundManager soundManager) 
        : base(position)
    {
        Radius = radius;
        _speed = 350f;
        _isLaunched = false;
        _paddle = paddle;
        _soundManager = soundManager;
        Velocity = Vector2.Zero;
    }

    public override void Update(GameTime gameTime)
    {
        float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;

        if (!_isLaunched)
        {
            Position = new Vector2(_paddle.Position.X, _paddle.Position.Y - 30);

            if (Keyboard.GetState().IsKeyDown(Keys.Space))
            {
                LaunchBall();
            }
        }
        else
        {
            Position += Velocity * deltaTime;

            // 좌우 벽 충돌 처리
            if (Position.X - Radius < 0 || Position.X + Radius > 1280)
            {
                Velocity.X *= -1;
                Position.X = MathHelper.Clamp(Position.X, Radius, 1280 - Radius);
                _soundManager?.PlayWallHit();
            }

            // 상단 벽 충돌 처리
            if (Position.Y - Radius < 0)
            {
                Velocity.Y *= -1;
                Position.Y = Radius;
                _soundManager?.PlayWallHit();
            }

            // 하단 벗어남 처리
            if (Position.Y - Radius > 720)
            {
                ResetBall();
            }
        }
    }

    public void LaunchBall()
    {
        _isLaunched = true;
        float angle = MathHelper.PiOver4;
        Velocity = new Vector2(
            (float)Math.Cos(angle) * _speed,
            -(float)Math.Sin(angle) * _speed
        );
    }

    public void ResetBall()
    {
        _isLaunched = false;
        Velocity = Vector2.Zero;
    }

    public void ReflectOffPaddle(Paddle paddle)
    {
        float paddleCenter = paddle.Position.X;
        float hitOffset = Position.X - paddleCenter;
        float normalizedOffset = hitOffset / (paddle.Width / 2);
        normalizedOffset = MathHelper.Clamp(normalizedOffset, -1, 1);

        float reflectAngle = normalizedOffset * MathHelper.PiOver3;
        
        Velocity = new Vector2(
            (float)Math.Sin(reflectAngle) * _speed,
            -(float)Math.Abs(Math.Cos(reflectAngle)) * _speed
        );

        Position.Y = paddle.Position.Y - paddle.Height / 2 - Radius - 1;
        _soundManager?.PlayPaddleHit();
    }

    public override void Draw(SpriteBatch spriteBatch)
    {
        DrawCircle(spriteBatch, Position, Radius, Color.Yellow);
    }

    private void DrawCircle(SpriteBatch spriteBatch, Vector2 center, 
        float radius, Color color)
    {
        int segments = (int)(radius * 2);
        for (int i = 0; i < segments; i++)
        {
            float angle1 = (float)i / segments * MathHelper.TwoPi;
            float angle2 = (float)(i + 1) / segments * MathHelper.TwoPi;

            Vector2 p1 = center + new Vector2(
                (float)Math.Cos(angle1) * radius,
                (float)Math.Sin(angle1) * radius
            );
            Vector2 p2 = center + new Vector2(
                (float)Math.Cos(angle2) * radius,
                (float)Math.Sin(angle2) * radius
            );

            spriteBatch.DrawLine(p1, p2, color, 2);
        }
    }

    public Rectangle GetBounds()
    {
        return new Rectangle(
            (int)(Position.X - Radius),
            (int)(Position.Y - Radius),
            (int)(Radius * 2),
            (int)(Radius * 2)
        );
    }

    public bool IsLaunched => _isLaunched;
}
```

**과제 1: 다단계 난이도 시스템**

현재는 모든 게임이 같은 난이도로 진행된다. 레벨이 올라갈수록 게임이 어려워지도록 구현해보자. 다음과 같은 항목들을 레벨에 따라 조정하자.

- 공의 속도를 레벨당 10% 증가
- 벽돌의 배치 패턴을 레벨에 따라 다르게 구성
- 강화된 벽돌의 비율을 높은 레벨에서 증가

`InitializeBricks(int level)` 메서드를 수정하여 레벨에 따른 벽돌 생성을 구현하자.

**과제 2: 파워업 시스템**

게임의 재미를 높이기 위해 파워업 아이템을 추가해보자. 벽돌을 파괴할 때 일정 확률로 파워업 아이템이 떨어지고, 패들로 받으면 특정 효과를 얻도록 구현하자.

예를 들어:
- 느린 공: 공의 속도를 50% 감소
- 넓은 패들: 패들의 너비를 150% 증가
- 느린 시간: 게임을 1초간 일시 중지
