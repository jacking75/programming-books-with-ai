# 11장. 사운드 이펙트와 음악
게임을 만들 때 시각적 요소만큼 중요한 것이 바로 소리이다. 플레이어가 점프할 때의 효과음, 적을 처치했을 때의 소리, 그리고 배경에서 계속 흘러나오는 음악은 게임의 몰입감을 크게 높인다. 이 장에서는 MonoGame에서 사운드 이펙트와 배경 음악을 어떻게 로드하고 재생하는지, 그리고 볼륨과 반복 재생을 어떻게 제어하는지 배운다.
  

## 11.1 SoundEffect 로드와 재생
MonoGame에서 사운드를 사용하려면 먼저 오디오 파일을 Content 폴더에 준비해야 한다. .wav 형식의 파일이 일반적으로 가장 잘 지원되며, 게임의 효과음이나 짧은 사운드에 사용된다.

### 사운드 파일 준비
게임 프로젝트의 Content 폴더에 Sound라는 새로운 폴더를 만들고, 거기에 오디오 파일을 배치한다. 예를 들어 "jump.wav", "hit.wav", "victory.wav" 같은 파일들을 준비하면 된다.

Content.mgcb 파일을 열어서 이 오디오 파일들을 프로젝트에 추가하면, MonoGame이 자동으로 처리해준다. Content 폴더 구조는 다음과 같이 구성된다.

```
Content/
├── Sound/
│   ├── jump.wav
│   ├── hit.wav
│   └── victory.wav
└── Content.mgcb
```

### SoundEffect 로드
SoundEffect 객체를 만들어 오디오 파일을 로드할 수 있다. 다음은 Game 클래스에서 사운드를 로드하는 방법이다.

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Audio;
using Microsoft.Xna.Framework.Graphics;

public class MyGame : Game
{
    private GraphicsDeviceManager graphics;
    private SpriteBatch spriteBatch;
    
    // 사운드 이펙트 저장
    private SoundEffect jumpSound;
    private SoundEffect hitSound;
    private SoundEffect victorySound;
    
    public MyGame()
    {
        graphics = new GraphicsDeviceManager(this);
        Content.RootDirectory = "Content";
    }
    
    protected override void LoadContent()
    {
        spriteBatch = new SpriteBatch(GraphicsDevice);
        
        // Content.Load 메서드로 오디오 파일 로드
        // 경로에서 Content/ 폴더는 생략하고 작성한다.
        jumpSound = Content.Load<SoundEffect>("Sound/jump");
        hitSound = Content.Load<SoundEffect>("Sound/hit");
        victorySound = Content.Load<SoundEffect>("Sound/victory");
    }
    
    protected override void Update(GameTime gameTime)
    {
        // 스페이스 바를 누르면 점프 소리 재생
        var keyboardState = Microsoft.Xna.Framework.Input.Keyboard.GetState();
        if (keyboardState.IsKeyDown(Microsoft.Xna.Framework.Input.Keys.Space))
        {
            jumpSound.Play();
        }
        
        base.Update(gameTime);
    }
    
    protected override void Draw(GameTime gameTime)
    {
        GraphicsDevice.Clear(Color.Black);
        
        spriteBatch.Begin();
        // 게임 그리기
        spriteBatch.End();
        
        base.Draw(gameTime);
    }
}
```

위 코드에서 주목할 점은 Content.Load<SoundEffect>를 사용한다는 것이다. 제네릭 매개변수로 SoundEffect 타입을 지정하고, 괄호 안에는 파일 경로를 문자열로 작성한다. Content 폴더는 자동으로 루트이므로 생략하고, 확장자도 적지 않는다.

### 사운드 이펙트 재생
SoundEffect는 Play 메서드를 가지고 있으며, 간단하게 호출하면 된다.

```csharp
// 기본 재생
jumpSound.Play();

// 볼륨, 피치, 팬을 지정하여 재생
// 볼륨: 0.0f(무음) ~ 1.0f(최대)
// 피치: -1.0f(낮음) ~ 0.0f(기본) ~ 1.0f(높음)
// 팬: -1.0f(왼쪽) ~ 0.0f(중앙) ~ 1.0f(오른쪽)
hitSound.Play(volume: 0.8f, pitch: 0.0f, pan: 0.0f);
```

Play 메서드는 SoundEffectInstance를 반환한다. 이를 통해 더 세밀한 제어가 가능하다.

```csharp
// SoundEffectInstance로 더 세밀한 제어
SoundEffectInstance jumpInstance = jumpSound.CreateInstance();
jumpInstance.Volume = 0.5f;
jumpInstance.Pitch = 0.2f;
jumpInstance.Pan = -0.5f; // 왼쪽 스피커에서 더 크게
jumpInstance.Play();

// 재생 중인지 확인
if (jumpInstance.State == SoundState.Playing)
{
    // 재생 중
}

// 일시 중지 또는 정지
jumpInstance.Pause();
jumpInstance.Resume();
jumpInstance.Stop();
```

### 사운드 관리자 클래스
게임에서 여러 사운드를 효율적으로 관리하려면, 사운드 관리자 클래스를 만드는 것이 좋다.

```csharp
using Microsoft.Xna.Framework.Audio;
using Microsoft.Xna.Framework.Content;
using System.Collections.Generic;

/// <summary>
/// 게임의 모든 사운드 이펙트를 관리하는 클래스이다.
/// 사운드를 로드하고, 재생하고, 정리하는 기능을 담당한다.
/// </summary>
public class SoundManager
{
    private ContentManager contentManager;
    private Dictionary<string, SoundEffect> soundEffects;
    private List<SoundEffectInstance> activeInstances;
    private float masterVolume;
    
    public float MasterVolume
    {
        get => masterVolume;
        set
        {
            masterVolume = value;
            // 모든 활성 사운드의 볼륨을 업데이트
            foreach (var instance in activeInstances)
            {
                instance.Volume = value;
            }
        }
    }
    
    public SoundManager(ContentManager contentManager)
    {
        this.contentManager = contentManager;
        soundEffects = new Dictionary<string, SoundEffect>();
        activeInstances = new List<SoundEffectInstance>();
        masterVolume = 1.0f;
    }
    
    /// <summary>
    /// 사운드 이펙트를 로드한다.
    /// 같은 이름의 사운드가 이미 로드되어 있으면 다시 로드하지 않는다.
    /// </summary>
    public void LoadSound(string name, string path)
    {
        if (soundEffects.ContainsKey(name))
            return;
        
        SoundEffect sound = contentManager.Load<SoundEffect>(path);
        soundEffects[name] = sound;
    }
    
    /// <summary>
    /// 사운드를 재생한다.
    /// </summary>
    public void PlaySound(string name, float volume = 1.0f, float pitch = 0.0f, float pan = 0.0f)
    {
        if (!soundEffects.ContainsKey(name))
        {
            System.Diagnostics.Debug.WriteLine($"사운드 '{name}'을(를) 찾을 수 없습니다.");
            return;
        }
        
        SoundEffectInstance instance = soundEffects[name].CreateInstance();
        instance.Volume = masterVolume * volume;
        instance.Pitch = pitch;
        instance.Pan = pan;
        instance.Play();
        
        activeInstances.Add(instance);
    }
    
    /// <summary>
    /// 모든 재생 중인 사운드를 정지한다.
    /// </summary>
    public void StopAll()
    {
        foreach (var instance in activeInstances)
        {
            if (instance.State == SoundState.Playing)
            {
                instance.Stop();
            }
        }
    }
    
    /// <summary>
    /// 특정 사운드를 정지한다.
    /// </summary>
    public void StopSound(string name)
    {
        // 이 메서드는 모든 인스턴스를 추적해야 하므로
        // 더 복잡한 구현이 필요하다.
        // 여기서는 간단한 예제로 구현한다.
    }
    
    /// <summary>
    /// 종료된 사운드 인스턴스를 정리한다.
    /// 이 메서드는 매 프레임마다 호출해야 한다.
    /// </summary>
    public void Update()
    {
        // 종료된 사운드 인스턴스 제거
        activeInstances.RemoveAll(instance => instance.State == SoundState.Stopped);
    }
    
    /// <summary>
    /// 모든 리소스를 해제한다.
    /// </summary>
    public void Dispose()
    {
        StopAll();
        foreach (var sound in soundEffects.Values)
        {
            sound.Dispose();
        }
        soundEffects.Clear();
        activeInstances.Clear();
    }
}
```

이 SoundManager 클래스의 설계 목적은 게임 전체에서 사운드를 일관되게 관리하는 것이다. 여러 사운드를 Dictionary에 저장하여 이름으로 쉽게 접근할 수 있고, MasterVolume을 통해 전체 음량을 제어할 수 있다. activeInstances 리스트는 현재 재생 중인 모든 사운드를 추적하며, Update 메서드에서 종료된 사운드를 자동으로 정리한다.

다음은 SoundManager를 Game 클래스에서 사용하는 방법이다.

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Graphics;
using Microsoft.Xna.Framework.Input;

public class MyGame : Game
{
    private GraphicsDeviceManager graphics;
    private SpriteBatch spriteBatch;
    private SoundManager soundManager;
    
    public MyGame()
    {
        graphics = new GraphicsDeviceManager(this);
        Content.RootDirectory = "Content";
    }
    
    protected override void LoadContent()
    {
        spriteBatch = new SpriteBatch(GraphicsDevice);
        
        soundManager = new SoundManager(Content);
        soundManager.LoadSound("jump", "Sound/jump");
        soundManager.LoadSound("hit", "Sound/hit");
        soundManager.LoadSound("victory", "Sound/victory");
    }
    
    protected override void Update(GameTime gameTime)
    {
        KeyboardState keyboardState = Keyboard.GetState();
        
        if (keyboardState.IsKeyDown(Keys.J))
            soundManager.PlaySound("jump");
        
        if (keyboardState.IsKeyDown(Keys.H))
            soundManager.PlaySound("hit", volume: 0.7f);
        
        if (keyboardState.IsKeyDown(Keys.V))
            soundManager.PlaySound("victory");
        
        // 전체 음량 조절 (Q, W 키)
        if (keyboardState.IsKeyDown(Keys.Q))
            soundManager.MasterVolume = System.Math.Max(0, soundManager.MasterVolume - 0.01f);
        
        if (keyboardState.IsKeyDown(Keys.W))
            soundManager.MasterVolume = System.Math.Min(1, soundManager.MasterVolume + 0.01f);
        
        // 매 프레임마다 사운드 상태 업데이트
        soundManager.Update();
        
        base.Update(gameTime);
    }
    
    protected override void Draw(GameTime gameTime)
    {
        GraphicsDevice.Clear(Color.Black);
        
        spriteBatch.Begin();
        spriteBatch.DrawString(null, "J: Jump, H: Hit, V: Victory", new Vector2(10, 10), Color.White);
        spriteBatch.DrawString(null, $"Master Volume: {soundManager.MasterVolume:F2}", new Vector2(10, 40), Color.White);
        spriteBatch.End();
        
        base.Draw(gameTime);
    }
}
```

## 11.2 배경 음악 (Song)
게임의 배경 음악은 SoundEffect와는 다르게 처리된다. 배경 음악은 보통 길고, 반복해서 재생되며, 게임이 진행되는 동안 계속 울려야 한다. MonoGame에서는 Song 클래스를 사용하여 이를 관리한다.

### Song 로드와 재생
Song은 SoundEffect와 유사하게 Content.Load를 통해 로드하지만, 내부적으로 다르게 처리된다. MediaPlayer를 사용하여 재생한다.

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Media;
using Microsoft.Xna.Framework.Content;

public class MusicManager
{
    private ContentManager contentManager;
    private Dictionary<string, Song> songs;
    private Song currentSong;
    private float musicVolume;
    
    public float MusicVolume
    {
        get => musicVolume;
        set
        {
            musicVolume = System.Math.Clamp(value, 0.0f, 1.0f);
            MediaPlayer.Volume = musicVolume;
        }
    }
    
    public bool IsPlaying => MediaPlayer.State == MediaState.Playing;
    
    public MusicManager(ContentManager contentManager)
    {
        this.contentManager = contentManager;
        songs = new Dictionary<string, Song>();
        musicVolume = 1.0f;
        
        // MediaPlayer 초기 설정
        MediaPlayer.Volume = musicVolume;
    }
    
    /// <summary>
    /// 배경 음악을 로드한다.
    /// </summary>
    public void LoadMusic(string name, string path)
    {
        if (songs.ContainsKey(name))
            return;
        
        Song song = contentManager.Load<Song>(path);
        songs[name] = song;
    }
    
    /// <summary>
    /// 음악을 재생한다.
    /// </summary>
    public void PlayMusic(string name, bool isLooping = true)
    {
        if (!songs.ContainsKey(name))
        {
            System.Diagnostics.Debug.WriteLine($"음악 '{name}'을(를) 찾을 수 없습니다.");
            return;
        }
        
        currentSong = songs[name];
        MediaPlayer.IsRepeating = isLooping;
        MediaPlayer.Play(currentSong);
    }
    
    /// <summary>
    /// 현재 재생 중인 음악을 일시 중지한다.
    /// </summary>
    public void Pause()
    {
        MediaPlayer.Pause();
    }
    
    /// <summary>
    /// 일시 중지된 음악을 재개한다.
    /// </summary>
    public void Resume()
    {
        MediaPlayer.Resume();
    }
    
    /// <summary>
    /// 현재 재생 중인 음악을 정지한다.
    /// </summary>
    public void Stop()
    {
        MediaPlayer.Stop();
    }
    
    /// <summary>
    /// 한 음악에서 다른 음악으로 페이드 전환한다.
    /// </summary>
    public void FadeOut()
    {
        MusicVolume -= 0.01f;
        if (MusicVolume <= 0)
        {
            Stop();
            MusicVolume = 1.0f;
        }
    }
}
```

다음은 MusicManager를 Game 클래스에서 사용하는 예제이다.

```csharp

using Microsoft.Xna.Framework.Input;

            musicManager.Stop();
        
        base.Update(gameTime);
    }
}
```

## 11.3 볼륨과 반복 재생

### 사운드 이펙트의 볼륨 제어

앞에서 이미 SoundManager에서 masterVolume을 구현했지만, 개별 사운드의 볼륨도 제어할 수 있다.

```csharp
// 개별 사운드 재생 시 볼륨 지정
soundManager.PlaySound("jump", volume: 0.5f);
soundManager.PlaySound("hit", volume: 0.8f);

// 마스터 볼륨 조절
soundManager.MasterVolume = 0.7f; // 70% 볼륨
```

### 배경 음악의 반복 재생

배경 음악은 보통 반복해서 재생되어야 한다. MediaPlayer의 IsRepeating 속성으로 이를 제어한다.

```csharp
// 반복 재생 설정
MediaPlayer.IsRepeating = true; // 무한 반복
musicManager.PlayMusic("background", isLooping: true);

// 한 번만 재생
MediaPlayer.IsRepeating = false;
musicManager.PlayMusic("intro", isLooping: false);
```

### 고급: 사운드 페이드 효과

게임에서 자주 사용되는 효과 중 하나가 바로 페이드이다. 음악이 천천히 작아졌다가 조용해지거나, 또는 조용하다가 크게 들려지는 효과이다.

```csharp
using Microsoft.Xna.Framework;
using System;

/// <summary>
/// 사운드 페이드 효과를 관리하는 클래스이다.
/// 음악이 일정 시간에 걸쳐 천천히 작아지거나 커지도록 제어한다.
/// </summary>
public class AudioFader
{
    private float targetVolume;
    private float currentVolume;
    private float fadeDuration;
    private float elapsedTime;
    private Action onFadeComplete;
    private bool isActive;
    
    public AudioFader()
    {
        currentVolume = 1.0f;
        targetVolume = 1.0f;
        fadeDuration = 0;
        elapsedTime = 0;
        isActive = false;
    }
    
    /// <summary>
    /// 페이드 아웃을 시작한다.
    /// duration: 완전히 조용해질 때까지의 시간(초)
    /// </summary>
    public void FadeOut(float duration, Action onComplete = null)
    {
        FadeTo(0.0f, duration, onComplete);
    }
    
    /// <summary>
    /// 페이드 인을 시작한다.
    /// duration: 최대 볼륨에 도달할 때까지의 시간(초)
    /// </summary>
    public void FadeIn(float duration, Action onComplete = null)
    {
        FadeTo(1.0f, duration, onComplete);
    }
    
    /// <summary>
    /// 특정 볼륨으로 페이드한다.
    /// </summary>
    public void FadeTo(float targetVol, float duration, Action onComplete = null)
    {
        targetVolume = Math.Clamp(targetVol, 0.0f, 1.0f);
        fadeDuration = duration;
        elapsedTime = 0;
        isActive = true;
        onFadeComplete = onComplete;
    }
    
    /// <summary>
    /// 매 프레임마다 호출하여 페이드를 업데이트한다.
    /// </summary>
    public float Update(GameTime gameTime)
    {
        if (!isActive)
            return currentVolume;
        
        elapsedTime += (float)gameTime.ElapsedGameTime.TotalSeconds;
        
        if (elapsedTime >= fadeDuration)
        {
            currentVolume = targetVolume;
            isActive = false;
            onFadeComplete?.Invoke();
            return currentVolume;
        }
        
        // 선형 보간으로 볼륨 계산
        float progress = elapsedTime / fadeDuration;
        float startVolume = elapsedTime == 0 ? currentVolume : currentVolume;
        currentVolume = MathHelper.Lerp(currentVolume, targetVolume, progress);
        
        return currentVolume;
    }
    
    public float CurrentVolume => currentVolume;
    public bool IsActive => isActive;
}
```

AudioFader를 MusicManager에 통합하여 사용할 수 있다.

```csharp
public class MusicManager
{
    private AudioFader audioFader;
    private float baseVolume;
    
    public MusicManager(ContentManager contentManager)
    {
        // ... 기존 초기화 코드 ...
        audioFader = new AudioFader();
        baseVolume = 1.0f;
    }
    
    public void FadeOutMusic(float duration)
    {
        audioFader.FadeOut(duration, () =>
        {
            MediaPlayer.Stop();
        });
    }
    
    public void FadeInMusic(float duration)
    {
        audioFader.FadeIn(duration);
    }
    
    public void Update(GameTime gameTime)
    {
        float fadeVolume = audioFader.Update(gameTime);
        MediaPlayer.Volume = baseVolume * fadeVolume;
    }
}
```

### 상황별 사운드 관리 예제
실제 게임에서는 여러 사운드를 동시에 관리해야 한다. 다음은 게임 상태에 따라 음악과 효과음을 관리하는 예제이다.

```csharp
public enum GameState
{
    Menu,
    Playing,
    Paused,
    GameOver
}

public class CompleteAudioManager
{
    private SoundManager soundManager;
    private MusicManager musicManager;
    private GameState currentState;
    
    public CompleteAudioManager(ContentManager contentManager)
    {
        soundManager = new SoundManager(contentManager);
        musicManager = new MusicManager(contentManager);
        
        // 사운드 이펙트 로드
        soundManager.LoadSound("jump", "Sound/jump");
        soundManager.LoadSound("hit", "Sound/hit");
        soundManager.LoadSound("powerup", "Sound/powerup");
        
        // 배경 음악 로드
        musicManager.LoadMusic("menu", "Music/menu");
        musicManager.LoadMusic("gameplay", "Music/gameplay");
        musicManager.LoadMusic("boss", "Music/boss_battle");
        musicManager.LoadMusic("gameover", "Music/gameover");
    }
    
    /// <summary>
    /// 게임 상태를 변경하고 해당하는 음악을 재생한다.
    /// </summary>
    public void ChangeGameState(GameState newState)
    {
        if (currentState == newState)
            return;
        
        currentState = newState;
        
        switch (newState)
        {
            case GameState.Menu:
                musicManager.PlayMusic("menu");
                soundManager.MasterVolume = 1.0f;
                break;
            
            case GameState.Playing:
                musicManager.PlayMusic("gameplay");
                break;
            
            case GameState.Paused:
                musicManager.Pause();
                break;
            
            case GameState.GameOver:
                musicManager.PlayMusic("gameover", isLooping: false);
                break;
        }
    }
    
    public void PlayGameSound(string soundName)
    {
        if (currentState == GameState.Playing || currentState == GameState.Paused)
        {
            soundManager.PlaySound(soundName);
        }
    }
    
    public void Update(GameTime gameTime)
    {
        soundManager.Update();
        musicManager.Update(gameTime);
    }
}
```

## 과제

### 과제 1: 사운드 메뉴 만들기
**목표:** SoundManager를 사용하여 여러 사운드를 로드하고, 플레이어가 키보드 입력으로 각각의 사운드를 재생하는 프로그램을 만들자.

**요구사항:**
- 최소 5개 이상의 서로 다른 SoundEffect 파일을 로드한다.
- 각각의 사운드를 1~5 숫자 키로 재생할 수 있게 한다.
- 화면에 현재 마스터 볼륨을 표시한다.
- Q 키를 누르면 마스터 볼륨이 10% 감소하고, W 키를 누르면 10% 증가한다.
- 각 사운드 이름과 해당하는 키를 화면에 표시한다.
- 마스터 볼륨이 0 이상 1 이하로 유지되도록 제한한다.

### 과제 2: 음악 플레이어 구현
**목표:** MusicManager를 사용하여 여러 곡을 관리하고, 곡 재생, 일시 중지, 정지, 반복 재생을 제어하는 간단한 음악 플레이어를 만들자.

**요구사항:**
- 최소 3곡 이상의 배경 음악 파일을 로드한다.
- P 키로 재생을 시작한다.
- 스페이스 바로 일시 중지/재개를 한다.
- S 키로 재생을 정지한다.
- 숫자 키 1, 2, 3으로 각각 다른 곡을 재생한다.
- 화면에 현재 재생 중인 곡의 이름을 표시한다.
- 음악의 반복 재생 여부를 토글할 수 있는 기능을 추가한다.
- 음악 볼륨을 화면에 표시한다.

### 과제 3: 게임 상태별 오디오 시스템
**목표:** CompleteAudioManager를 확장하여 게임 상태에 따라 자동으로 음악과 효과음이 관리되는 시스템을 만들자.

**요구사항:**
- 게임 상태를 Menu, Playing, Boss, GameOver 4가지로 정의한다.
- 각 상태마다 다른 배경 음악이 재생된다.
- 상태 전환 시 이전 음악이 서서히 사라지고 새로운 음악이 들려오는 페이드 효과를 구현한다. (AudioFader 클래스 사용)
- Playing 상태에서만 효과음(점프, 적 처치 등)을 재생할 수 있다.
- Boss 상태에서는 효과음의 볼륨이 일반 Playing 상태보다 10% 더 작다.
- 숫자 키 1~4로 각 게임 상태를 전환할 수 있다.
- 화면에 현재 게임 상태와 재생 중인 음악, 페이드 진행률을 표시한다.
- 효과음을 재생할 수 있는 여러 개의 테스트 키를 준비한다. (예: J=점프, H=적 처치, P=파워업)

**선택 사항:**
- AudioFader에 곡선 보간(easing)을 추가하여 더 부드러운 페이드 효과를 구현한다.
- 여러 곡을 크로스페이드하여 매끄럽게 전환하는 기능을 추가한다.
- 음악 볼륨과 효과음 볼륨을 독립적으로 제어할 수 있게 한다.
