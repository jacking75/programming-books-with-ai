# 7장. 게임패드 지원
게임패드는 현대 게임에서 매우 중요한 입력 장치다. MonoGame은 Xbox 형식의 게임패드를 기본적으로 지원하므로, 게임을 더욱 다양한 플랫폼에서 플레이할 수 있게 한다. 이 장에서는 게임패드의 연결 상태를 확인하고, 버튼과 아날로그 스틱 입력을 처리하며, 진동 피드백을 구현하는 방법을 배운다.
  

## 7.1 게임패드 연결 확인

### 게임패드의 기본 개념
MonoGame에서 게임패드를 사용하려면 먼저 게임패드가 연결되어 있는지 확인해야 한다. 게임패드는 최대 4개까지 동시에 연결될 수 있으며, 각각 PlayerIndex.One, PlayerIndex.Two, PlayerIndex.Three, PlayerIndex.Four로 구분된다.

게임패드의 상태는 GamePadState 객체로 표현되며, 이를 통해 모든 입력 정보에 접근할 수 있다. 먼저 간단하게 게임패드 연결 상태를 확인하는 코드를 살펴보자.

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Input;

namespace GamepadExample
{
    public class Game1 : Game
    {
        private GraphicsDeviceManager _graphics;
        private SpriteBatch _spriteBatch;
        private GamePadState _currentGamePadState;
        private GamePadState _previousGamePadState;

        public Game1()
        {
            _graphics = new GraphicsDeviceManager(this);
            Content.RootDirectory = "Content";
            IsMouseVisible = true;
        }

        protected override void Initialize()
        {
            base.Initialize();
        }

        protected override void LoadContent()
        {
            _spriteBatch = new SpriteBatch(GraphicsDevice);
        }

        protected override void Update(GameTime gameTime)
        {
            _previousGamePadState = _currentGamePadState;
            _currentGamePadState = GamePad.GetState(PlayerIndex.One);

            // 게임패드가 연결되어 있는지 확인
            if (_currentGamePadState.IsConnected)
            {
                // 게임패드 입력 처리
                if (_currentGamePadState.Buttons.A == ButtonState.Pressed)
                {
                    // A 버튼이 눌렸을 때의 동작
                }
            }

            base.Update(gameTime);
        }

        protected override void Draw(GameTime gameTime)
        {
            GraphicsDevice.Clear(Color.CornflowerBlue);

            _spriteBatch.Begin();
            
            if (_currentGamePadState.IsConnected)
            {
                // "게임패드 연결됨" 표시
            }
            else
            {
                // "게임패드 미연결" 표시
            }

            _spriteBatch.End();

            base.Draw(gameTime);
        }
    }
}
```

이 코드에서 GamePad.GetState(PlayerIndex.One)은 첫 번째 플레이어의 게임패드 상태를 가져온다. IsConnected 속성으로 게임패드가 실제로 연결되어 있는지 확인할 수 있다.

### 게임패드 연결 상태 모니터링
게임패드가 연결되거나 분리될 때를 감지하려면 이전 상태와 현재 상태를 비교해야 한다. 다음은 연결 상태 변화를 감지하는 예제다.

```csharp
private bool _wasGamePadConnected = false;

protected override void Update(GameTime gameTime)
{
    _previousGamePadState = _currentGamePadState;
    _currentGamePadState = GamePad.GetState(PlayerIndex.One);

    // 게임패드가 새로 연결된 경우
    if (_currentGamePadState.IsConnected && !_wasGamePadConnected)
    {
        // 게임패드 연결 이벤트 처리
        OnGamePadConnected();
    }

    // 게임패드가 분리된 경우
    if (!_currentGamePadState.IsConnected && _wasGamePadConnected)
    {
        // 게임패드 분리 이벤트 처리
        OnGamePadDisconnected();
    }

    _wasGamePadConnected = _currentGamePadState.IsConnected;

    base.Update(gameTime);
}

private void OnGamePadConnected()
{
    // 게임패드 연결 시 처리할 로직
}

private void OnGamePadDisconnected()
{
    // 게임패드 분리 시 처리할 로직
}
```

이렇게 하면 사용자가 게임을 플레이 중에 게임패드를 연결하거나 분리했을 때를 감지할 수 있다.

## 7.2 버튼과 스틱 입력

### 게임패드 버튼 입력
Xbox 형식의 게임패드는 A, B, X, Y 버튼과 LB, RB 버튼, 그리고 시작 버튼과 뒤로 가기 버튼을 가지고 있다. 각 버튼의 상태는 ButtonState.Pressed 또는 ButtonState.Released로 표현된다.

```csharp
protected override void Update(GameTime gameTime)
{
    _previousGamePadState = _currentGamePadState;
    _currentGamePadState = GamePad.GetState(PlayerIndex.One);

    if (_currentGamePadState.IsConnected)
    {
        // A 버튼 입력
        if (_currentGamePadState.Buttons.A == ButtonState.Pressed)
        {
            // A 버튼 동작
        }

        // B 버튼 입력
        if (_currentGamePadState.Buttons.B == ButtonState.Pressed)
        {
            // B 버튼 동작
        }

        // X 버튼 입력
        if (_currentGamePadState.Buttons.X == ButtonState.Pressed)
        {
            // X 버튼 동작
        }

        // Y 버튼 입력
        if (_currentGamePadState.Buttons.Y == ButtonState.Pressed)
        {
            // Y 버튼 동작
        }

        // LB 버튼 입력
        if (_currentGamePadState.Buttons.LeftShoulder == ButtonState.Pressed)
        {
            // LB 버튼 동작
        }

        // RB 버튼 입력
        if (_currentGamePadState.Buttons.RightShoulder == ButtonState.Pressed)
        {
            // RB 버튼 동작
        }
    }

    base.Update(gameTime);
}
```

### 버튼 누름 상태 변화 감지
키보드 입력 처리와 마찬가지로, 게임패드 버튼이 처음 눌렸는지 또는 눌렸다가 해제되었는지를 감지하는 것이 중요하다. 이를 위해 이전 상태와 현재 상태를 비교한다.

```csharp
// 버튼이 처음 눌린 순간을 감지
private bool IsButtonPressed(ButtonState current, ButtonState previous)
{
    return current == ButtonState.Pressed && previous == ButtonState.Released;
}

// 버튼이 해제된 순간을 감지
private bool IsButtonReleased(ButtonState current, ButtonState previous)
{
    return current == ButtonState.Released && previous == ButtonState.Pressed;
}

protected override void Update(GameTime gameTime)
{
    _previousGamePadState = _currentGamePadState;
    _currentGamePadState = GamePad.GetState(PlayerIndex.One);

    if (_currentGamePadState.IsConnected)
    {
        // A 버튼이 처음 눌린 순간
        if (IsButtonPressed(_currentGamePadState.Buttons.A, _previousGamePadState.Buttons.A))
        {
            // 단 한 번만 실행되는 로직
        }

        // A 버튼이 해제된 순간
        if (IsButtonReleased(_currentGamePadState.Buttons.A, _previousGamePadState.Buttons.A))
        {
            // 버튼이 떨어진 순간의 로직
        }
    }

    base.Update(gameTime);
}
```

### 아날로그 스틱 입력
게임패드의 왼쪽과 오른쪽 아날로그 스틱은 X축과 Y축 방향의 연속된 값을 제공한다. 이 값은 -1.0f부터 1.0f 사이의 float 값이다.

```csharp
protected override void Update(GameTime gameTime)
{
    _previousGamePadState = _currentGamePadState;
    _currentGamePadState = GamePad.GetState(PlayerIndex.One);

    if (_currentGamePadState.IsConnected)
    {
        // 왼쪽 스틱 입력
        Vector2 leftStick = _currentGamePadState.ThumbSticks.Left;
        float leftStickX = leftStick.X;  // -1.0f ~ 1.0f
        float leftStickY = leftStick.Y;  // -1.0f ~ 1.0f

        // 오른쪽 스틱 입력
        Vector2 rightStick = _currentGamePadState.ThumbSticks.Right;
        float rightStickX = rightStick.X;
        float rightStickY = rightStick.Y;

        // 캐릭터 이동 (왼쪽 스틱 사용)
        Vector2 movement = leftStick * speed;
    }

    base.Update(gameTime);
}
```

### 트리거 입력
게임패드의 LT(왼쪽 트리거)와 RT(오른쪽 트리거)는 0.0f부터 1.0f까지의 값을 제공한다. 이는 아날로그 버튼으로, 얼마나 많이 눌렸는지를 나타낸다.

```csharp
protected override void Update(GameTime gameTime)
{
    _previousGamePadState = _currentGamePadState;
    _currentGamePadState = GamePad.GetState(PlayerIndex.One);

    if (_currentGamePadState.IsConnected)
    {
        // 왼쪽 트리거 (0.0f ~ 1.0f)
        float leftTrigger = _currentGamePadState.Triggers.Left;

        // 오른쪽 트리거 (0.0f ~ 1.0f)
        float rightTrigger = _currentGamePadState.Triggers.Right;

        // 트리거가 어느 정도 눌렸는지 확인
        if (leftTrigger > 0.5f)
        {
            // 왼쪽 트리거가 반 이상 눌린 상태
        }

        if (rightTrigger > 0.5f)
        {
            // 오른쪽 트리거가 반 이상 눌린 상태
        }
    }

    base.Update(gameTime);
}
```

### 디지털 방향 패드 입력
게임패드의 방향 패드(D-Pad)는 위, 아래, 왼쪽, 오른쪽 네 가지 방향을 나타낸다.

protected override void Update(GameTime gameTime)

## 7.3 진동 피드백

### 진동 피드백의 이해
많은 게임패드는 진동 기능을 지원한다. 이는 게임에 촉각적 피드백을 제공하여 게임 경험을 더욱 실감 나게 만든다. MonoGame에서 진동은 저주파 모터와 고주파 모터의 강도를 0.0f부터 1.0f 사이의 값으로 제어한다.

```csharp
// 기본 진동
GamePad.SetVibration(PlayerIndex.One, 0.5f, 0.5f);

// 0.5f의 강도로 저주파와 고주파 모두 진동
// 첫 번째 매개변수: 저주파 모터 강도
// 두 번째 매개변수: 고주파 모터 강도
```

### 진동 피드백 구현
실제 게임에서는 일정 시간 동안만 진동하도록 구현하는 것이 일반적이다. 다음은 진동 시간을 관리하는 예제다.

```csharp
private float _vibrationDuration = 0.0f;
private const float DEFAULT_VIBRATION_DURATION = 0.3f;

protected override void Update(GameTime gameTime)
{
    _previousGamePadState = _currentGamePadState;
    _currentGamePadState = GamePad.GetState(PlayerIndex.One);

    // 진동 시간 감소
    if (_vibrationDuration > 0)
    {
        _vibrationDuration -= (float)gameTime.ElapsedGameTime.TotalSeconds;
        
        // 진동 계속 적용
        GamePad.SetVibration(PlayerIndex.One, 0.5f, 0.5f);
    }
    else
    {
        // 진동 멈춤
        GamePad.SetVibration(PlayerIndex.One, 0.0f, 0.0f);
    }

    if (_currentGamePadState.IsConnected)
    {
        // A 버튼이 눌렸을 때 진동 시작
        if (_currentGamePadState.Buttons.A == ButtonState.Pressed)
        {
            StartVibration(0.5f, 0.5f, 0.3f);
        }
    }

    base.Update(gameTime);
}

private void StartVibration(float leftMotor, float rightMotor, float duration)
{
    GamePad.SetVibration(PlayerIndex.One, leftMotor, rightMotor);
    _vibrationDuration = duration;
}
```

### 강도와 시간에 따른 다양한 진동
게임의 상황에 따라 다양한 진동 효과를 만들 수 있다. 예를 들어, 약한 진동, 강한 진동, 펄스 형태의 진동 등을 만들 수 있다.

```csharp
// 약한 진동 (살짝 찌르는 느낌)
private void WeakVibration()
{
    StartVibration(0.2f, 0.2f, 0.1f);
}

// 중간 진동 (충격을 받은 느낌)
private void MediumVibration()
{
    StartVibration(0.5f, 0.5f, 0.3f);
}

// 강한 진동 (큰 충격을 받은 느낌)
private void StrongVibration()
{
    StartVibration(1.0f, 1.0f, 0.5f);
}

// 비대칭 진동 (독특한 느낌)
private void AsymmetricVibration()
{
    StartVibration(0.3f, 0.8f, 0.4f);  // 저주파는 약하고 고주파는 강함
}
```

### 게임 상황별 진동 효과
충돌, 폭발, 공격 등 다양한 게임 상황에 진동 피드백을 적용할 수 있다.

```csharp
public class GamepadVibrationManager
{
    private PlayerIndex _playerIndex;
    private float _vibrationDuration = 0.0f;

    public GamepadVibrationManager(PlayerIndex playerIndex)
    {
        _playerIndex = playerIndex;
    }

    public void Update(GameTime gameTime)
    {
        if (_vibrationDuration > 0)
        {
            _vibrationDuration -= (float)gameTime.ElapsedGameTime.TotalSeconds;
        }
        else
        {
            GamePad.SetVibration(_playerIndex, 0.0f, 0.0f);
        }
    }

    // 총에 맞았을 때의 진동
    public void OnBulletHit()
    {
        // 높은 주파수에 강한 진동으로 충격 표현
        StartVibration(0.2f, 0.8f, 0.15f);
    }

    // 폭발이 일어났을 때의 진동
    public void OnExplosion()
    {
        // 저주파와 고주파 모두 강하게 크레센도 효과
        StartVibration(1.0f, 1.0f, 0.5f);
    }

    // 점프했을 때의 진동
    public void OnJump()
    {
        // 짧고 약한 펄스
        StartVibration(0.3f, 0.3f, 0.1f);
    }

    // 착지했을 때의 진동
    public void OnLanding()
    {
        // 착지의 충격 표현
        StartVibration(0.6f, 0.4f, 0.2f);
    }

    private void StartVibration(float leftMotor, float rightMotor, float duration)
    {
        GamePad.SetVibration(_playerIndex, leftMotor, rightMotor);
        _vibrationDuration = duration;
    }
}
```

### 진동 피드백 사용 시 주의사항
진동 피드백을 사용할 때는 다음 몇 가지를 주의해야 한다. 첫째, 모든 게임패드가 진동을 지원하지는 않으므로, 진동이 항상 작동하지 않을 수 있다. 둘째, 과도한 진동은 오히려 게임 경험을 해칠 수 있으므로 적절한 강도와 시간을 설정해야 한다. 셋째, 게임을 종료할 때 진동을 명시적으로 끄는 것이 좋다.

```csharp
protected override void UnloadContent()
{
    // 게임 종료 시 모든 진동 멈춤
    GamePad.SetVibration(PlayerIndex.One, 0.0f, 0.0f);
    GamePad.SetVibration(PlayerIndex.Two, 0.0f, 0.0f);
    GamePad.SetVibration(PlayerIndex.Three, 0.0f, 0.0f);
    GamePad.SetVibration(PlayerIndex.Four, 0.0f, 0.0f);

    base.UnloadContent();
}
```

## 종합 예제: 게임패드로 캐릭터 제어
지금까지 배운 내용을 종합하여 게임패드로 캐릭터를 제어하는 예제를 만들어보자. 이 예제에서는 왼쪽 스틱으로 이동, A 버튼으로 점프, 방향 패드로 회전을 제어한다.

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Input;
using Microsoft.Xna.Framework.Graphics;

namespace GamepadControlExample
{
    public class Player
    {
        public Vector2 Position { get; set; }
        public Vector2 Velocity { get; set; }
        public float Rotation { get; set; }
        public Texture2D Texture { get; set; }

        private const float MOVE_SPEED = 300.0f;
        private const float JUMP_SPEED = 400.0f;
        private const float ROTATION_SPEED = 3.0f;
        private bool _isJumping = false;

        public Player()
        {
            Position = Vector2.Zero;
            Velocity = Vector2.Zero;
            Rotation = 0.0f;
        }

        public void Update(GameTime gameTime, GamePadState currentState, 
                          GamePadState previousState)
        {
            if (!currentState.IsConnected)
                return;

            float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;

            // 왼쪽 스틱으로 이동
            Vector2 movement = currentState.ThumbSticks.Left * MOVE_SPEED;
            Velocity = new Vector2(movement.X * deltaTime, Velocity.Y);
            Position += new Vector2(Velocity.X, 0);

            // A 버튼으로 점프
            if (currentState.Buttons.A == ButtonState.Pressed && 
                previousState.Buttons.A == ButtonState.Released && 
                !_isJumping)
            {
                Velocity = new Vector2(Velocity.X, -JUMP_SPEED * deltaTime);
                _isJumping = true;
            }

            // 중력 적용
            Velocity = new Vector2(Velocity.X, Velocity.Y + 500.0f * deltaTime);
            Position += new Vector2(0, Velocity.Y);

            // 땅 충돌 감지 (y > 400이라고 가정)
            if (Position.Y > 400)
            {
                Position = new Vector2(Position.X, 400);
                Velocity = new Vector2(Velocity.X, 0);
                _isJumping = false;
            }

            // 방향 패드로 회전
            if (currentState.DPad.Left == ButtonState.Pressed)
            {
                Rotation -= ROTATION_SPEED * deltaTime;
            }
            if (currentState.DPad.Right == ButtonState.Pressed)
            {
                Rotation += ROTATION_SPEED * deltaTime;
            }
        }

        public void Draw(SpriteBatch spriteBatch)
        {
            spriteBatch.Draw(Texture, Position, null, Color.White, Rotation,
                           new Vector2(Texture.Width / 2, Texture.Height / 2),
                           1.0f, SpriteEffects.None, 0.0f);
        }
    }

    public class Game1 : Game
    {
        private GraphicsDeviceManager _graphics;
        private SpriteBatch _spriteBatch;
        private Player _player;
        private GamePadState _currentGamePadState;
        private GamePadState _previousGamePadState;
        private float _vibrationTimer = 0.0f;

        public Game1()
        {
            _graphics = new GraphicsDeviceManager(this);
            Content.RootDirectory = "Content";
            IsMouseVisible = true;
        }

        protected override void Initialize()
        {
            _player = new Player();
            base.Initialize();
        }

        protected override void LoadContent()
        {
            _spriteBatch = new SpriteBatch(GraphicsDevice);

            // 플레이어 텍스처 로드 (1x1 흰색 픽셀 생성)
            _player.Texture = new Texture2D(GraphicsDevice, 32, 32);
            Color[] colorData = new Color[32 * 32];
            for (int i = 0; i < colorData.Length; i++)
            {
                colorData[i] = Color.White;
            }
            _player.Texture.SetData(colorData);
        }

        protected override void Update(GameTime gameTime)
        {
            _previousGamePadState = _currentGamePadState;
            _currentGamePadState = GamePad.GetState(PlayerIndex.One);

            if (_currentGamePadState.IsConnected)
            {
                _player.Update(gameTime, _currentGamePadState, _previousGamePadState);

                // B 버튼으로 진동
                if (_currentGamePadState.Buttons.B == ButtonState.Pressed &&
                    _previousGamePadState.Buttons.B == ButtonState.Released)
                {
                    GamePad.SetVibration(PlayerIndex.One, 0.7f, 0.7f);
                    _vibrationTimer = 0.2f;
                }
            }

            // 진동 타이머 감소
            if (_vibrationTimer > 0)
            {
                _vibrationTimer -= (float)gameTime.ElapsedGameTime.TotalSeconds;
            }
            else
            {
                GamePad.SetVibration(PlayerIndex.One, 0.0f, 0.0f);
            }

            base.Update(gameTime);
        }

        protected override void Draw(GameTime gameTime)
        {
            GraphicsDevice.Clear(Color.CornflowerBlue);

            _spriteBatch.Begin();
            _player.Draw(_spriteBatch);
            _spriteBatch.End();

            base.Draw(gameTime);
        }
    }
}
```

이 예제는 50줄 이상이므로 상세한 설명이 필요하다. Player 클래스는 플레이어 캐릭터의 위치, 속도, 회전을 관리한다. Update 메서드에서 왼쪽 스틱의 입력값을 이용하여 좌우 이동을 처리하고, A 버튼을 감지하여 점프를 구현한다. 여기서 중요한 점은 A 버튼이 "처음 눌린" 순간만 점프가 발생하도록 previousState와 currentState를 비교하는 것이다. 중력을 계속 적용하여 현실적인 물리 효과를 구현했다. 방향 패드의 좌우 입력으로 캐릭터의 회전각을 조절한다. Game1 클래스의 Update 메서드에서는 B 버튼을 감지하여 진동을 발생시키고, 일정 시간 후 자동으로 진동을 멈춘다.

## 과제

### 과제 1: 게임패드 상태 모니터링
게임패드가 연결되면 "게임패드 연결됨"을, 분리되면 "게임패드 분리됨"을 화면에 표시하는 프로그램을 작성하자. SpriteFont를 사용하여 텍스트를 그려야 한다.

**요구사항:**
- PlayerIndex.One 게임패드의 연결 상태를 지속적으로 모니터링한다
- 연결 상태가 변했을 때 메시지를 표시한다
- 현재 게임패드 연결 상태를 항상 화면에 표시한다

### 과제 2: 게임패드 입력 시각화
게임패드의 모든 버튼과 스틱의 입력 상태를 화면에 시각적으로 표시하는 프로그램을 작성하자. 예를 들어, A 버튼이 눌리면 그 버튼을 강조 표시하고, 아날로그 스틱의 위치를 화면에 표시해야 한다.

**요구사항:**
- A, B, X, Y 버튼의 상태를 사각형이나 원으로 표시한다
- 방향 패드의 상태를 표시한다
- 왼쪽과 오른쪽 아날로그 스틱의 위치를 좌표로 표시한다
- 왼쪽과 오른쪽 트리거의 값을 바 형태로 표시한다

### 과제 3: 미니 게임 - 타겟 맞추기
게임패드 진동을 활용한 간단한 미니 게임을 만들자. 화면에 타겟이 나타나고, 플레이어가 타겟의 위치에 정확하게 맞추면 진동 피드백을 주고 점수를 얻는다.

**요구사항:**
- 화면의 무작위 위치에 타겟(원)을 생성한다
- 오른쪽 아날로그 스틱으로 조준점을 이동한다
- A 버튼으로 발사한다
- 조준점이 타겟을 맞히면 성공적인 진동을 발생시킨다
- 점수와 남은 타겟 개수를 표시한다
- 10개 타겟을 모두 맞히면 게임이 종료된다

**힌트:**
- 조준점은 오른쪽 스틱 입력에 따라 이동한다
- 타겟과 조준점 사이의 거리를 계산하여 충돌을 감지한다
- 성공하면 강한 진동을 발생시킨다
- 실패하면 약한 진동을 발생시킨다
