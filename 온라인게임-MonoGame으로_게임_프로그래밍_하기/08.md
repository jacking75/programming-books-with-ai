# 8장. 게임 오브젝트 설계
게임을 구성하는 모든 요소들, 즉 플레이어, 적, 아이템, 효과 등을 효율적으로 관리하려면 공통된 구조를 가진 기본 클래스가 필요하다. 이 장에서는 게임의 모든 오브젝트를 대표할 수 있는 GameObject 클래스를 설계하고, 위치와 속도, 방향과 같은 필수 속성을 구현하며, 오브젝트가 매 프레임마다 어떻게 업데이트되고 그려지는지를 학습한다.

## 8.1 GameObject 클래스 만들기

### GameObject의 필요성
게임이 복잡해질수록 화면에 표시되는 오브젝트의 종류와 개수가 많아진다. 플레이어, 적, 총알, 아이템, 배경 요소 등 수십 개 이상의 오브젝트가 동시에 존재할 수 있다. 각각을 독립적인 클래스로 만든다면 코드의 중복이 심해지고 관리가 어려워진다. 따라서 모든 게임 오브젝트의 공통 기능을 담당하는 기본 클래스인 GameObject를 만들어야 한다.

### 기본 GameObject 클래스
가장 간단한 GameObject 클래스부터 시작해보자.

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Graphics;

namespace GameObjectExample
{
    public class GameObject
    {
        public Vector2 Position { get; set; }
        public Texture2D Texture { get; set; }
        public bool IsActive { get; set; }

        public GameObject()
        {
            Position = Vector2.Zero;
            Texture = null;
            IsActive = true;
        }

        public virtual void Update(GameTime gameTime)
        {
            // 파생 클래스에서 구현
        }

        public virtual void Draw(SpriteBatch spriteBatch)
        {
            if (Texture != null && IsActive)
            {
                spriteBatch.Draw(Texture, Position, Color.White);
            }
        }
    }
}
```

이 기본 GameObject 클래스는 위치(Position), 텍스처(Texture), 활성 상태(IsActive)를 가지고 있다. virtual 키워드를 사용한 Update와 Draw 메서드는 파생 클래스에서 재정의할 수 있도록 만들었다. 이를 통해 서로 다른 오브젝트들이 각자의 업데이트와 렌더링 로직을 구현할 수 있다.

### 파생 클래스 예제
GameObject를 상속하는 구체적인 오브젝트 클래스들을 만들어보자.

```csharp
public class Player : GameObject
{
    public float Speed { get; set; } = 200.0f;

    public override void Update(GameTime gameTime)
    {
        // 플레이어 업데이트 로직
        base.Update(gameTime);
    }

    public override void Draw(SpriteBatch spriteBatch)
    {
        base.Draw(spriteBatch);
    }
}

public class Enemy : GameObject
{
    public float Speed { get; set; } = 100.0f;
    public int Health { get; set; } = 100;

    public override void Update(GameTime gameTime)
    {
        // 적 업데이트 로직
        base.Update(gameTime);
    }

    public override void Draw(SpriteBatch spriteBatch)
    {
        base.Draw(spriteBatch);
    }
}

public class Projectile : GameObject
{
    public float Speed { get; set; } = 500.0f;
    public Vector2 Direction { get; set; }

    public override void Update(GameTime gameTime)
    {
        // 발사체 업데이트 로직
        base.Update(gameTime);
    }

    public override void Draw(SpriteBatch spriteBatch)
    {
        base.Draw(spriteBatch);
    }
}
```

이제 플레이어, 적, 발사체 등 서로 다른 종류의 오브젝트들을 GameObject를 상속하여 만들 수 있다. 각 클래스는 자신만의 고유한 속성과 동작을 추가할 수 있으면서도 기본적인 게임 오브젝트의 인터페이스를 공유한다.

### 오브젝트 풀 관리
게임에서는 여러 개의 오브젝트를 함께 관리해야 한다. 이를 위해 오브젝트들을 리스트에 저장하고 매 프레임마다 순회하며 업데이트하고 그린다.

```csharp
public class Game1 : Game
{
    private GraphicsDeviceManager _graphics;
    private SpriteBatch _spriteBatch;
    private List<GameObject> _gameObjects;

    public Game1()
    {
        _graphics = new GraphicsDeviceManager(this);
        Content.RootDirectory = "Content";
        IsMouseVisible = true;
    }

    protected override void Initialize()
    {
        _gameObjects = new List<GameObject>();
        base.Initialize();
    }

    protected override void LoadContent()
    {
        _spriteBatch = new SpriteBatch(GraphicsDevice);

        // 플레이어 생성
        Player player = new Player();
        player.Position = new Vector2(400, 300);
        _gameObjects.Add(player);

        // 적들 생성
        for (int i = 0; i < 5; i++)
        {
            Enemy enemy = new Enemy();
            enemy.Position = new Vector2(100 + i * 150, 100);
            _gameObjects.Add(enemy);
        }
    }

    protected override void Update(GameTime gameTime)
    {
        // 모든 오브젝트 업데이트
        foreach (GameObject obj in _gameObjects)
        {
            if (obj.IsActive)
            {
                obj.Update(gameTime);
            }
        }

        // 비활성 오브젝트 제거
        _gameObjects.RemoveAll(obj => !obj.IsActive);

        base.Update(gameTime);
    }

    protected override void Draw(GameTime gameTime)
    {
        GraphicsDevice.Clear(Color.CornflowerBlue);

        _spriteBatch.Begin();

        // 모든 오브젝트 그리기
        foreach (GameObject obj in _gameObjects)
        {
            obj.Draw(_spriteBatch);
        }

        _spriteBatch.End();

        base.Draw(gameTime);
    }
}
```

## 8.2 위치, 속도, 방향

### 위치와 속도의 개념
게임에서 오브젝트의 움직임을 표현하려면 위치(Position)와 속도(Velocity)가 필요하다. 위치는 오브젝트가 현재 어디에 있는지를 나타내고, 속도는 오브젝트가 매초 얼마나 이동하는지를 나타낸다. 속도에 시간을 곱하면 실제 이동 거리를 계산할 수 있다.

```csharp
public class GameObject
{
    public Vector2 Position { get; set; }
    public Vector2 Velocity { get; set; }
    public Texture2D Texture { get; set; }
    public bool IsActive { get; set; }

    public GameObject()
    {
        Position = Vector2.Zero;
        Velocity = Vector2.Zero;
        Texture = null;
        IsActive = true;
    }

    public virtual void Update(GameTime gameTime)
    {
        // 속도에 따른 위치 업데이트
        float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;
        Position += Velocity * deltaTime;
    }

    public virtual void Draw(SpriteBatch spriteBatch)
    {
        if (Texture != null && IsActive)
        {
            spriteBatch.Draw(Texture, Position, Color.White);
        }
    }
}
```

여기서 중요한 점은 시간 차이(deltaTime)를 사용한다는 것이다. 게임이 다양한 프레임레이트에서 실행될 수 있으므로, 프레임의 개수가 아닌 실제 경과 시간을 기준으로 이동 거리를 계산해야 한다.

### 방향 속도
많은 게임에서는 속도를 크기(Speed)와 방향(Direction)으로 분리하여 표현하는 것이 더 편하다. 이는 특히 발사체나 회전하는 오브젝트를 다룰 때 유용하다.

```csharp
public class GameObject
{
    public Vector2 Position { get; set; }
    public float Speed { get; set; }
    public float Rotation { get; set; }  // 라디안 단위
    public Texture2D Texture { get; set; }
    public bool IsActive { get; set; }

    public GameObject()
    {
        Position = Vector2.Zero;
        Speed = 0.0f;
        Rotation = 0.0f;
        Texture = null;
        IsActive = true;
    }

    // 방향 벡터를 속도에서 추출
    public Vector2 GetDirection()
    {
        return new Vector2((float)Math.Cos(Rotation), (float)Math.Sin(Rotation));
    }

    // 방향에 따른 속도 계산
    public Vector2 GetVelocity()
    {
        return GetDirection() * Speed;
    }

    public virtual void Update(GameTime gameTime)
    {
        float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;
        Position += GetVelocity() * deltaTime;
    }

    public virtual void Draw(SpriteBatch spriteBatch)
    {
        if (Texture != null && IsActive)
        {
            Vector2 origin = new Vector2(Texture.Width / 2, Texture.Height / 2);
            spriteBatch.Draw(Texture, Position, null, Color.White, Rotation,
                           origin, 1.0f, SpriteEffects.None, 0.0f);
        }
    }
}
```

### 속도 벡터 방식
벡터로 직접 속도를 저장하는 방식도 있다. 이는 2D 게임에서 가장 일반적인 방식이다.

```csharp
public class GameObject
{
    public Vector2 Position { get; set; }
    public Vector2 Velocity { get; set; }
    public Vector2 Acceleration { get; set; }
    public float Rotation { get; set; }
    public Texture2D Texture { get; set; }
    public bool IsActive { get; set; }

    public GameObject()
    {
        Position = Vector2.Zero;
        Velocity = Vector2.Zero;
        Acceleration = Vector2.Zero;
        Rotation = 0.0f;
        Texture = null;
        IsActive = true;
    }

    public virtual void Update(GameTime gameTime)
    {
        float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;

        // 가속도에 따른 속도 변경
        Velocity += Acceleration * deltaTime;

        // 속도에 따른 위치 변경
        Position += Velocity * deltaTime;

        // 속도 방향에 따른 회전 계산
        if (Velocity.Length() > 0)
        {
            Rotation = (float)Math.Atan2(Velocity.Y, Velocity.X);
        }
    }

    public virtual void Draw(SpriteBatch spriteBatch)
    {
        if (Texture != null && IsActive)
        {
            Vector2 origin = new Vector2(Texture.Width / 2, Texture.Height / 2);
            spriteBatch.Draw(Texture, Position, null, Color.White, Rotation,
                           origin, 1.0f, SpriteEffects.None, 0.0f);
        }
    }
}
```

이 방식은 가속도(Acceleration)를 추가로 지원한다. 예를 들어 중력이나 마찰력을 구현할 때 매우 유용하다.

## 8.3 오브젝트 업데이트와 렌더링

### 업데이트 메서드의 역할
업데이트 메서드는 게임 루프의 Update 단계에서 호출되며, 오브젝트의 상태를 변경하는 역할을 한다. 위치 변경, 애니메이션 처리, 생명시간 감소, 상태 변경 등 모든 로직이 여기서 처리된다.

```csharp
public class Player : GameObject
{
    private float _shootCooldown = 0.0f;
    private const float SHOOT_COOLDOWN = 0.5f;

    public override void Update(GameTime gameTime)
    {
        float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;

        // 위치와 속도 기본 업데이트
        base.Update(gameTime);

        // 키보드 입력 처리
        KeyboardState keyboardState = Keyboard.GetState();
        if (keyboardState.IsKeyDown(Keys.Left))
        {
            Velocity = new Vector2(-200, Velocity.Y);
        }
        else if (keyboardState.IsKeyDown(Keys.Right))
        {
            Velocity = new Vector2(200, Velocity.Y);
        }
        else
        {
            Velocity = new Vector2(0, Velocity.Y);
        }

        // 발사 쿨다운 처리
        if (_shootCooldown > 0)
        {
            _shootCooldown -= deltaTime;
        }

        if (keyboardState.IsKeyDown(Keys.Space) && _shootCooldown <= 0)
        {
            // 총알 발사
            _shootCooldown = SHOOT_COOLDOWN;
        }
    }
}
```

### 렌더링 메서드의 역할
렌더링 메서드는 게임 루프의 Draw 단계에서 호출되며, 오브젝트를 화면에 그리는 역할을 한다. 중요한 점은 렌더링 메서드에서는 오브젝트의 상태를 변경해서는 안 된다는 것이다. 렌더링은 현재 상태를 시각화하는 것일 뿐이다.

```csharp
public class Player : GameObject
{
    private int _currentFrame = 0;
    private int _totalFrames = 4;
    private float _frameTimer = 0.0f;
    private const float FRAME_DURATION = 0.1f;

    public override void Draw(SpriteBatch spriteBatch)
    {
        if (Texture != null && IsActive)
        {
            // 애니메이션 프레임 계산
            Rectangle sourceRectangle = new Rectangle(
                _currentFrame * (Texture.Width / _totalFrames),
                0,
                Texture.Width / _totalFrames,
                Texture.Height
            );

            Vector2 origin = new Vector2(
                sourceRectangle.Width / 2,
                sourceRectangle.Height / 2
            );

            spriteBatch.Draw(Texture, Position, sourceRectangle, Color.White,
                           Rotation, origin, 1.0f, SpriteEffects.None, 0.0f);
        }
    }
}
```

### 정리된 게임 루프
이제 모든 개념을 통합하여 완전한 게임 루프를 구현해보자. 

```csharp
using Microsoft.Xna.Framework;
using Microsoft.Xna.Framework.Graphics;
using Microsoft.Xna.Framework.Input;
using System;
using System.Collections.Generic;

namespace CompleteGameObjectExample
{
    // 기본 GameObject 클래스
    public abstract class GameObject
    {
        public Vector2 Position { get; set; }
        public Vector2 Velocity { get; set; }
        public Vector2 Acceleration { get; set; }
        public float Rotation { get; set; }
        public Texture2D Texture { get; set; }
        public bool IsActive { get; set; }

        public GameObject()
        {
            Position = Vector2.Zero;
            Velocity = Vector2.Zero;
            Acceleration = Vector2.Zero;
            Rotation = 0.0f;
            Texture = null;
            IsActive = true;
        }

        public virtual void Update(GameTime gameTime)
        {
            float deltaTime = (float)gameTime.ElapsedGameTime.TotalSeconds;
            Velocity += Acceleration * deltaTime;
            Position += Velocity * deltaTime;
        }

        public virtual void Draw(SpriteBatch spriteBatch)
        {
            if (Texture != null && IsActive)
            {
                Vector2 origin = new Vector2(Texture.Width / 2, Texture.Height / 2);
                spriteBatch.Draw(Texture, Position, null, Color.White, Rotation,
                               origin, 1.0f, SpriteEffects.None, 0.0f);
            }
        }
    }

    // 플레이어 클래스
    public class Player : GameObject
    {
        private const float MAX_SPEED = 300.0f;
        private const float FRICTION = 0.9f;

        public override void Update(GameTime gameTime)
        {
            base.Update(gameTime);

            // 입력 처리
            KeyboardState keyboard = Keyboard.GetState();
            Vector2 inputDirection = Vector2.Zero;

            if (keyboard.IsKeyDown(Keys.Left) || keyboard.IsKeyDown(Keys.A))
                inputDirection.X = -1;
            if (keyboard.IsKeyDown(Keys.Right) || keyboard.IsKeyDown(Keys.D))
                inputDirection.X = 1;
            if (keyboard.IsKeyDown(Keys.Up) || keyboard.IsKeyDown(Keys.W))
                inputDirection.Y = -1;
            if (keyboard.IsKeyDown(Keys.Down) || keyboard.IsKeyDown(Keys.S))
                inputDirection.Y = 1;

            // 입력에 따른 가속
            if (inputDirection != Vector2.Zero)
            {
                inputDirection.Normalize();
                Velocity = inputDirection * MAX_SPEED;
            }
            else
            {
                // 입력이 없으면 마찰 적용
                Velocity *= FRICTION;
            }

            // 화면 경계 체크
            if (Position.X < 0) Position.X = 0;
            if (Position.X > 800) Position.X = 800;
            if (Position.Y < 0) Position.Y = 0;
            if (Position.Y > 600) Position.Y = 600;

            // 속도 방향에 따른 회전
            if (Velocity.Length() > 0)
            {
                Rotation = (float)Math.Atan2(Velocity.Y, Velocity.X);
            }
        }
    }

    // 적 클래스
    public class Enemy : GameObject
    {
        private float _moveTimer = 0.0f;
        private const float MOVE_CHANGE_TIME = 2.0f;
        private const float ENEMY_SPEED = 100.0f;

        public override void Update(GameTime gameTime)
        {
            base.Update(gameTime);

            _moveTimer += (float)gameTime.ElapsedGameTime.TotalSeconds;

            // 주기적으로 방향 변경
            if (_moveTimer > MOVE_CHANGE_TIME)
            {
                Random random = new Random();
                float angle = (float)(random.NextDouble() * Math.PI * 2);
                Velocity = new Vector2(
                    (float)Math.Cos(angle) * ENEMY_SPEED,
                    (float)Math.Sin(angle) * ENEMY_SPEED
                );
                _moveTimer = 0;

                // 속도 방향에 따른 회전
                Rotation = angle;
            }

            // 화면 경계 체크
            if (Position.X < 0 || Position.X > 800 ||
                Position.Y < 0 || Position.Y > 600)
            {
                IsActive = false;
            }
        }
    }

    // 발사체 클래스
    public class Projectile : GameObject
    {
        private float _lifetime = 5.0f;

        public override void Update(GameTime gameTime)
        {
            base.Update(gameTime);

            _lifetime -= (float)gameTime.ElapsedGameTime.TotalSeconds;
            if (_lifetime <= 0)
            {
                IsActive = false;
            }

            // 화면 밖으로 나가면 비활성화
            if (Position.X < -50 || Position.X > 850 ||
                Position.Y < -50 || Position.Y > 650)
            {
                IsActive = false;
            }
        }
    }

    // 게임 클래스
    public class Game1 : Game
    {
        private GraphicsDeviceManager _graphics;
        private SpriteBatch _spriteBatch;
        private List<GameObject> _gameObjects;
        private Player _player;
        private Texture2D _playerTexture;
        private Texture2D _enemyTexture;
        private float _spawnTimer = 0.0f;
        private const float SPAWN_INTERVAL = 3.0f;

        public Game1()
        {
            _graphics = new GraphicsDeviceManager(this);
            Content.RootDirectory = "Content";
            IsMouseVisible = true;
        }

        protected override void Initialize()
        {
            _gameObjects = new List<GameObject>();
            base.Initialize();
        }

        protected override void LoadContent()
        {
            _spriteBatch = new SpriteBatch(GraphicsDevice);

            // 텍스처 생성 (32x32 원형)
            _playerTexture = CreateCircleTexture(32, Color.Blue);
            _enemyTexture = CreateCircleTexture(32, Color.Red);

            // 플레이어 생성
            _player = new Player();
            _player.Texture = _playerTexture;
            _player.Position = new Vector2(400, 300);
            _gameObjects.Add(_player);
        }

        private Texture2D CreateCircleTexture(int size, Color color)
        {
            Texture2D texture = new Texture2D(GraphicsDevice, size, size);
            Color[] colorData = new Color[size * size];

            float radius = size / 2.0f;
            Vector2 center = new Vector2(radius, radius);

            for (int y = 0; y < size; y++)
            {
                for (int x = 0; x < size; x++)
                {
                    Vector2 position = new Vector2(x, y);
                    if (Vector2.Distance(position, center) <= radius)
                    {
                        colorData[y * size + x] = color;
                    }
                    else
                    {
                        colorData[y * size + x] = Color.Transparent;
                    }
                }
            }

            texture.SetData(colorData);
            return texture;
        }

        protected override void Update(GameTime gameTime)
        {
            // 적 생성
            _spawnTimer += (float)gameTime.ElapsedGameTime.TotalSeconds;
            if (_spawnTimer > SPAWN_INTERVAL)
            {
                Enemy enemy = new Enemy();
                enemy.Texture = _enemyTexture;
                enemy.Position = new Vector2(
                    (float)(new Random().NextDouble() * 800),
                    -50
                );
                Vector2 direction = Vector2.Normalize(_player.Position - enemy.Position);
                enemy.Velocity = direction * 100.0f;
                _gameObjects.Add(enemy);
                _spawnTimer = 0;
            }

            // 모든 오브젝트 업데이트
            foreach (GameObject obj in _gameObjects)
            {
                if (obj.IsActive)
                {
                    obj.Update(gameTime);
                }
            }

            // 비활성 오브젝트 제거
            _gameObjects.RemoveAll(obj => !obj.IsActive);

            base.Update(gameTime);
        }

        protected override void Draw(GameTime gameTime)
        {
            GraphicsDevice.Clear(Color.CornflowerBlue);

            _spriteBatch.Begin();

            // 모든 오브젝트 그리기
            foreach (GameObject obj in _gameObjects)
            {
                obj.Draw(_spriteBatch);
            }

            _spriteBatch.End();

            base.Draw(gameTime);
        }
    }
}
```

이 코드는 완전한 게임 오브젝트 시스템을 보여준다. GameObject는 모든 게임 오브젝트의 기본이 되는 추상 클래스다. 위치, 속도, 가속도, 회전을 가지고 있으며, Update와 Draw 메서드를 가상 메서드로 정의하여 파생 클래스에서 재정의할 수 있게 했다.

Player 클래스는 키보드 입력을 처리하고 최대 속도를 제한한다. 입력이 없을 때는 마찰력을 적용하여 점진적으로 속도를 줄인다. 속도 방향에 따라 회전각을 자동으로 계산한다.

Enemy 클래스는 일정 시간 간격으로 방향을 바꾸며 이동한다. 화면을 벗어나면 자동으로 비활성화된다.

Projectile 클래스는 생명시간을 가지고 있으며, 시간이 다하거나 화면을 벗어나면 비활성화된다.

Game1 클래스는 게임 루프를 관리하고, 일정 시간마다 새로운 적을 생성한다. 모든 게임 오브젝트를 리스트에 저장하고 매 프레임마다 업데이트하고 그린다. CreateCircleTexture 메서드는 프로그래밍 방식으로 원형 텍스처를 생성한다.

## 과제

### 과제 1: 기본 게임 오브젝트 확장
기본 GameObject 클래스에 다음 기능을 추가하자.

- Scale(크기 조절) 속성 추가
- Color(색상) 속성 추가
- GetBounds() 메서드로 오브젝트의 충돌 영역 반환
- IsAlive 속성을 IsActive로 변경하고, 생명 시간(LifeTime) 추가

**요구사항:**
- GameObject 클래스를 수정한다
- Draw 메서드에서 Scale과 Color를 적용한다
- Update 메서드에서 LifeTime을 감소시키고 0이 되면 IsActive를 false로 설정한다

### 과제 2: 플레이어 확장 기능
Player 클래스에 다음 기능들을 추가하자.

- 현재 방향을 나타내는 Direction 속성
- 왼쪽과 오른쪽으로 회전하는 기능 (Q, E 키)
- 현재 속력을 백분율로 표시하는 Speed 속성
- 최대 체력(MaxHealth)과 현재 체력(Health) 속성

**요구사항:**
- 플레이어가 원활하게 회전한다
- 회전 속도는 일정해야 한다
- 키를 누르면 회전 각도를 표시한다

### 과제 3: 오브젝트 관리 시스템
다음 기능들을 가진 GameObjectManager 클래스를 만들자.

```
GameObjectManager 클래스:
- 게임 오브젝트들을 관리한다
- AddObject() 메서드로 오브젝트 추가
- RemoveObject() 메서드로 오브젝트 제거
- UpdateAll() 메서드로 모든 오브젝트 업데이트
- DrawAll() 메서드로 모든 오브젝트 그리기
- GetActiveObjectCount() 메서드로 활성 오브젝트 개수 반환
- GetObjectsAtPosition() 메서드로 특정 위치의 오브젝트 반환
```

**요구사항:**
- 제네릭을 사용하여 다양한 타입의 오브젝트를 관리한다
- 내부적으로 List를 사용한다
- 매 업데이트마다 비활성 오브젝트를 자동으로 제거한다
- 모든 메서드가 안전하게 동작해야 한다

### 과제 4: 추적하는 적 구현
Enemy 클래스를 다음과 같이 개선하자.

- 플레이어를 추적하는 TrackingEnemy 클래스 만들기
- 플레이어까지의 거리를 기준으로 추적 여부 결정
- 거리에 따라 속도 조절
- 일정 거리 이내에 있을 때만 추적

**요구사항:**
- TrackingEnemy는 Enemy를 상속한다
- 플레이어 참조를 생성자에서 받는다
- 추적 범위(TrackingRange) 속성을 가진다
- 추적할 때 항상 플레이어를 향한다

### 과제 5: 간단한 게임 구현
위의 모든 개념을 사용하여 다음 게임을 구현하자.

- 플레이어가 화면 중앙에서 시작한다
- 일정 시간마다 적들이 화면 상단에서 나타난다
- 적들은 플레이어를 추적한다
- 플레이어가 적과 충돌하면 체력이 감소한다
- 체력이 0이 되면 게임이 종료된다
- 적을 일정 개수 처치하면 레벨이 증가하고 난이도가 올라간다

**요구사항:**
- 게임 상태 관리 (실행 중, 게임 오버)
- 화면에 현재 체력, 점수, 레벨을 표시한다
- 적의 개수와 속도가 레벨에 따라 증가한다
- ESC 키로 게임을 종료할 수 있다
