# 3. C++ thread_local

## 3.1 기본 문법과 사용법
C++11에서 도입된 `thread_local` 키워드는 Win32 API의 TLS를 훨씬 더 간편하고 안전하게 사용할 수 있게 해줍니다. 컴파일러가 자동으로 각 스레드별 독립적인 변수 인스턴스를 생성하고 관리합니다.

### 기본 문법

```cpp
thread_local int threadLocalVar = 0;
thread_local std::string threadLocalString;
thread_local MyClass* pMyObject;

int main() {
    // 각 스레드에서 이들 변수는 독립적입니다
    threadLocalVar = 42;  // 현재 스레드만 영향
}
```

### Win32 API TLS와의 비교

| 항목 | Win32 API TLS | C++ thread_local |
|------|---------------|-----------------|
| 문법 | `TlsAlloc(), TlsSetValue(), TlsGetValue()` | `thread_local int x;` |
| 초기화 | 수동 (NULL) | 자동 (기본값) |
| 메모리 정리 | 수동 | 자동 |
| 타입 안전성 | 없음 (void*) | 완벽함 |
| 슬롯 수 제한 | 최대 1088개 | 제한 없음 |
| 스코프 | 프로세스 전역 | 파일 스코프, 함수 스코프, 전역 |
| 성능 | 약간 느림 | 약간 더 빠름 |

### 사용 위치에 따른 분류

#### 1. 전역 네임스페이스에서의 사용


### 예제 1: 기본 사용법

```cpp
#include <iostream>
#include <thread>
#include <vector>
#include <windows.h>

// 스레드별 독립 변수
thread_local int threadLocalCounter = 0;
thread_local std::string threadLocalName = "";

void ThreadFunction(int threadNum) {
    // 각 스레드가 자신의 복사본을 가짐
    threadLocalCounter = threadNum * 100;
    threadLocalName = "Thread_" + std::to_string(threadNum);
    
    printf("[%s] Counter = %d\n", threadLocalName.c_str(), threadLocalCounter);
    
    // 값 수정
    threadLocalCounter += 50;
    printf("[%s] Counter after modification = %d\n", 
           threadLocalName.c_str(), threadLocalCounter);
    
    Sleep(500);
    
    // 다른 스레드의 값은 영향을 받지 않음
    printf("[%s] Final Counter = %d\n", 
           threadLocalName.c_str(), threadLocalCounter);
}

int main() {
    std::vector<std::thread> threads;
    
    // 3개의 스레드 생성
    for (int i = 0; i < 3; i++) {
        threads.emplace_back(ThreadFunction, i);
    }
    
    // 모든 스레드 완료 대기
    for (auto& t : threads) {
        t.join();
    }
    
    // 메인 스레드의 값은 여전히 초기값
    printf("[Main] threadLocalCounter = %d\n", threadLocalCounter);
    printf("[Main] threadLocalName = %s\n", 
           threadLocalName.empty() ? "(empty)" : threadLocalName.c_str());
    
    return 0;
}
```

**출력:**
```

[Thread_0] Counter after modification = 50
[Thread_1] Counter = 100
[Thread_1] Counter after modification = 150
[Thread_2] Counter = 200
[Thread_2] Counter after modification = 250
[Thread_0] Final Counter = 50
[Thread_1] Final Counter = 150
[Thread_2] Final Counter = 250
```

### 예제 2: 함수 내 정적 thread_local

```cpp
#include <iostream>
#include <thread>
#include <chrono>

class ThreadIdManager {
public:
    // 각 스레드의 고유 ID를 반환하고 캐시
    static int GetThreadId() {
        static thread_local int threadId = -1;
        
        if (threadId == -1) {
            // 첫 호출 시에만 실행
            threadId = GetCurrentThreadId();
            std::cout << "Thread ID initialized: " << threadId << "\n";
        }
        
        return threadId;
    }
    
    // 각 스레드의 실행 횟수 추적
    static int GetCallCount() {
        static thread_local int callCount = 0;
        return ++callCount;
    }
};

void Worker(int num) {
    std::cout << "Worker " << num << " started (ID: " 
              << ThreadIdManager::GetThreadId() << ")\n";
    
    for (int i = 0; i < 3; i++) {
        int count = ThreadIdManager::GetCallCount();
        std::cout << "Worker " << num << " call count: " << count << "\n";
        std::this_thread::sleep_for(std::chrono::milliseconds(100));
    }
}

int main() {
    std::thread t1(Worker, 1);
    std::thread t2(Worker, 2);
    
    t1.join();
    t2.join();
    
    // 메인 스레드에서도 호출 가능
    std::cout << "Main thread call count: " 
              << ThreadIdManager::GetCallCount() << "\n";
    
    return 0;
}
```
  

## 3.2 초기화와 생명주기

### 초기화 시점
C++의 `thread_local`은 다양한 초기화 패턴을 제공합니다.

```cpp
// 1. 기본값으로 초기화 (암시적)
thread_local int simpleInt;  // = 0

// 2. 명시적 초기화
thread_local int explicitInt = 42;

// 3. 복잡한 초기화 (클래스 생성자)
thread_local std::vector<int> threadVector;  // 빈 벡터로 초기화

// 4. 초기화 함수 사용
thread_local std::unique_ptr<class MyResource> resource;

void InitializeThreadResource() {
    if (!resource) {
        resource = std::make_unique<MyResource>();
    }
}
```

### Lazy Initialization 패턴
자원을 필요할 때만 할당하는 방식입니다.

```cpp
#include <iostream>
#include <thread>
#include <memory>

class ExpensiveResource {
private:
    int resourceId;
    
public:
    ExpensiveResource() : resourceId(0) {
        std::cout << "ExpensiveResource created for thread " 
                  << GetCurrentThreadId() << "\n";
    }
    
    ~ExpensiveResource() {
        std::cout << "ExpensiveResource destroyed for thread " 
                  << GetCurrentThreadId() << "\n";
    }
    
    void DoWork() {
        std::cout << "Working with resource in thread " 
                  << GetCurrentThreadId() << "\n";
    }
};

// Lazy initialization helper
template<typename T>
class ThreadLocalLazy {
private:
    static thread_local std::unique_ptr<T> instance;
    
public:
    static T& Get() {
        if (!instance) {
            instance = std::make_unique<T>();
        }
        return *instance;
    }
};

// 명시적 인스턴스 선언
template<typename T>
thread_local std::unique_ptr<T> ThreadLocalLazy<T>::instance = nullptr;

void WorkerThread(int id) {
    std::cout << "Worker " << id << " starting\n";
    
    // 첫 접근 시만 생성
    auto& resource = ThreadLocalLazy<ExpensiveResource>::Get();
    resource.DoWork();
    
    // 두 번째 접근은 캐시된 인스턴스 사용
    auto& resource2 = ThreadLocalLazy<ExpensiveResource>::Get();
    resource2.DoWork();
    
    std::cout << "Worker " << id << " ending\n";
}

int main() {
    std::cout << "Main thread: " << GetCurrentThreadId() << "\n";
    
    std::thread t1(WorkerThread, 1);
    std::thread t2(WorkerThread, 2);
    
    t1.join();
    t2.join();
    
    std::cout << "All threads completed\n";
    return 0;
}
```

**출력:**
```

Worker 1 starting
ExpensiveResource created for thread 67890
Worker 1 ending
Worker 2 starting
ExpensiveResource created for thread 67891
Worker 2 ending
ExpensiveResource destroyed for thread 67890
ExpensiveResource destroyed for thread 67891
All threads completed
```

### 생명주기 추적

```cpp
#include <iostream>
#include <thread>
#include <mutex>

class LifecycleTracker {
private:
    int id;
    static std::mutex outputMutex;
    
public:
    LifecycleTracker(int id = -1) : id(id) {
        std::lock_guard<std::mutex> lock(outputMutex);
        std::cout << "[생성] LifecycleTracker(" << id << ") "
                  << "in thread " << GetCurrentThreadId() << "\n";
    }
    
    ~LifecycleTracker() {
        std::lock_guard<std::mutex> lock(outputMutex);
        std::cout << "[소멸] LifecycleTracker(" << id << ") "
                  << "in thread " << GetCurrentThreadId() << "\n";
    }
    
    void Print() {
        std::lock_guard<std::mutex> lock(outputMutex);
        std::cout << "[사용] LifecycleTracker(" << id << ")\n";
    }
};

std::mutex LifecycleTracker::outputMutex;

// thread_local 변수
thread_local LifecycleTracker g_tracker(0);

void WorkerThread(int num) {
    std::cout << "Worker " << num << " started\n";
    
    // thread_local 변수는 이미 생성됨
    g_tracker.Print();
    
    // 새로운 thread_local 변수 생성 가능
    static thread_local LifecycleTracker s_local(num);
    s_local.Print();
    
    std::cout << "Worker " << num << " ending\n";
}

int main() {
    std::cout << "=== Main thread start ===\n";
    g_tracker.Print();
    
    {
        std::thread t1(WorkerThread, 1);
        std::thread t2(WorkerThread, 2);
        
        t1.join();
        t2.join();
    }
    
    std::cout << "=== Main thread end ===\n";
    g_tracker.Print();
    
    return 0;
}
```
  

## 3.3 클래스 멤버와의 활용

### 정적 멤버로의 사용

```cpp
#include <iostream>
#include <thread>
#include <sstream>

class FileHandler {
private:
    // 각 스레드별 독립적인 버퍼
    static thread_local std::stringstream s_writeBuffer;
    static thread_local int s_bytesWritten;
    static thread_local std::string s_lastError;
    
public:
    static void Write(const std::string& data) {
        s_writeBuffer << data;
        s_bytesWritten += data.length();
    }
    
    static void Flush() {
        std::string content = s_writeBuffer.str();
        printf("[Thread %d] Flushing %zu bytes: %s\n", 
               GetCurrentThreadId(), content.length(), content.c_str());
        s_writeBuffer.str("");
        s_writeBuffer.clear();
    }
    
    static void SetError(const std::string& error) {
        s_lastError = error;
    }
    
    static std::string GetError() {
        return s_lastError;
    }
    
    static int GetBytesWritten() {
        return s_bytesWritten;
    }
};

// 멤버 초기화
thread_local std::stringstream FileHandler::s_writeBuffer;
thread_local int FileHandler::s_bytesWritten = 0;
thread_local std::string FileHandler::s_lastError = "";

void WriterThread(int id) {
    for (int i = 0; i < 3; i++) {
        std::string msg = "Thread " + std::to_string(id) + 
                         " message " + std::to_string(i) + "\n";
        FileHandler::Write(msg);
    }
    
    printf("[Thread %d] Total bytes: %d\n", 
           GetCurrentThreadId(), FileHandler::GetBytesWritten());
    
    FileHandler::Flush();
}

int main() {
    std::thread t1(WriterThread, 1);
    std::thread t2(WriterThread, 2);
    
    t1.join();
    t2.join();
    
    return 0;
}
```

### 고급 패턴: 스레드풀과의 결합

```cpp
#include <iostream>
#include <thread>
#include <queue>
#include <mutex>
#include <condition_variable>

class RequestContext {
public:
    int requestId;
    std::string data;
    
    RequestContext(int id, const std::string& d) 
        : requestId(id), data(d) {}
};

class RequestProcessor {
private:
    // 각 워커 스레드별 컨텍스트
    static thread_local RequestContext* s_currentContext;
    static thread_local int s_processedCount;
    
public:
    static void ProcessRequest(const RequestContext& ctx) {
        s_currentContext = new RequestContext(ctx);
        s_processedCount++;
        
        printf("[Thread %d] Processing request %d: %s\n", 
               GetCurrentThreadId(), 
               s_currentContext->requestId,
               s_currentContext->data.c_str());
        
        delete s_currentContext;
        s_currentContext = nullptr;
    }
    
    static int GetProcessedCount() {
        return s_processedCount;
    }
};

// 초기화
thread_local RequestContext* RequestProcessor::s_currentContext = nullptr;
thread_local int RequestProcessor::s_processedCount = 0;

void WorkerThreadFunction() {
    RequestContext req1(1, "First request");
    RequestProcessor::ProcessRequest(req1);
    
    RequestContext req2(2, "Second request");
    RequestProcessor::ProcessRequest(req2);
    
    printf("[Thread %d] Total processed: %d\n", 
           GetCurrentThreadId(), 
           RequestProcessor::GetProcessedCount());
}

int main() {
    std::thread workers[2];
    for (int i = 0; i < 2; i++) {
        workers[i] = std::thread(WorkerThreadFunction);
    }
    
    for (int i = 0; i < 2; i++) {
        workers[i].join();
    }
    
    return 0;
}
```

### thread_local과 소멸자 호출 보장

```cpp
#include <iostream>
#include <thread>
#include <memory>

class Resource {
private:
    int id;
    
public:
    Resource(int id) : id(id) {
        printf("[생성] Resource %d in thread %d\n", 
               id, GetCurrentThreadId());
    }
    
    ~Resource() {
        printf("[소멸] Resource %d in thread %d\n", 
               id, GetCurrentThreadId());
    }
    
    void Use() {
        printf("[사용] Resource %d\n", id);
    }
};

// 전역 thread_local - 스레드 종료 시 자동 소멸
thread_local std::unique_ptr<Resource> g_resource;

void InitializeResourceForThread(int id) {
    g_resource = std::make_unique<Resource>(id);
}

void WorkerThread(int num) {
    printf("Worker %d starting in thread %d\n", num, GetCurrentThreadId());
    
    InitializeResourceForThread(num);
    g_resource->Use();
    
    printf("Worker %d ending\n", num);
    // 이 지점에서 스레드가 종료되면 g_resource의 소멸자가 자동으로 호출됨
}

int main() {
    {
        std::thread t1(WorkerThread, 1);
        std::thread t2(WorkerThread, 2);
        
        t1.join();  // 여기서 Resource 1의 소멸자 호출
        t2.join();  // 여기서 Resource 2의 소멸자 호출
    }
    
    printf("All threads completed\n");
    return 0;
}
```

### Win32 API TLS와의 비교 예제

```cpp
#include <windows.h>
#include <iostream>
#include <thread>

// === Win32 API TLS 방식 ===
DWORD g_tlsIndex = TLS_OUT_OF_INDEXES;

struct TLSData {
    int value;
    char buffer[256];
};

void Win32TLSExample() {
    g_tlsIndex = TlsAlloc();
    
    auto worker = [](int id) {
        TLSData* pData = new TLSData{id * 100, ""};
        sprintf_s(pData->buffer, sizeof(pData->buffer), "Data %d", id);
        
        TlsSetValue(g_tlsIndex, pData);
        
        // 사용
        TLSData* retrieved = (TLSData*)TlsGetValue(g_tlsIndex);
        printf("Win32 TLS - ID: %d, Value: %d\n", id, retrieved->value);
        
        // 수동 정리 필요
        delete retrieved;
    };
    
    std::thread t1(worker, 1);
    std::thread t2(worker, 2);
    
    t1.join();
    t2.join();
    
    TlsFree(g_tlsIndex);
}

// === C++ thread_local 방식 ===
thread_local int g_value = 0;
thread_local std::string g_buffer = "";

void CppThreadLocalExample() {
    auto worker = [](int id) {
        g_value = id * 100;
        g_buffer = std::string("Data ") + std::to_string(id);
        
        // 사용
        printf("C++ thread_local - ID: %d, Value: %d\n", id, g_value);
        // 자동 정리
    };
    
    std::thread t1(worker, 1);
    std::thread t2(worker, 2);
    
    t1.join();
    t2.join();
}

int main() {
    printf("=== Win32 API TLS ===\n");
    Win32TLSExample();
    
    printf("\n=== C++ thread_local ===\n");
    CppThreadLocalExample();
    
    return 0;
}
```

- **자동 초기화**: 기본값 또는 초기화식으로 각 스레드별 자동 초기화
