# 13장. Redis 데이터 타입 활용 (실습)

## 13.1 개요

Redis는 다양한 데이터 타입을 제공하며, 각 타입은 특정 용도에 최적화되어 있다. 이번 장에서는 게임 서버에서 자주 사용되는 5가지 핵심 데이터 타입을 실습을 통해 학습한다. CloudStructures 라이브러리를 사용하여 각 타입을 효율적으로 다루는 방법을 익힌다.

## 13.2 String 타입 - 인증 토큰 저장

### 13.2.1 String 타입 개요

String은 Redis의 가장 기본적인 데이터 타입이다. 텍스트뿐만 아니라 숫자, 직렬화된 객체 등 다양한 데이터를 저장할 수 있다. 게임 서버에서는 주로 다음과 같은 용도로 사용한다.

- 인증 토큰 저장
- 세션 데이터 저장
- 캐시 데이터 저장
- 카운터 (접속자 수, 조회수 등)

### 13.2.2 RedisString 기본 사용법

CloudStructures의 `RedisString<T>` 클래스를 사용하여 String 타입 데이터를 다룬다.

```csharp
// Services/StringExampleService.cs
using CloudStructures;
using CloudStructures.Structures;

namespace GameApiServer.Services;

public interface IStringExampleService
{
    Task<bool> SaveTokenAsync(long userId, string token, TimeSpan expiry);
    Task<string?> GetTokenAsync(long userId);
    Task<bool> DeleteTokenAsync(long userId);
    Task<long> IncrementLoginCountAsync(long userId);
}

public class StringExampleService : IStringExampleService
{
    private readonly RedisConnection _redisConnection;
    private readonly ILogger<StringExampleService> _logger;

    public StringExampleService(IRedisService redisService, ILogger<StringExampleService> logger)
    {
        _redisConnection = redisService.GetConnection();
        _logger = logger;
    }

    public async Task<bool> SaveTokenAsync(long userId, string token, TimeSpan expiry)
    {
        try
        {
            var key = $"auth:token:{userId}";
            var redis = new RedisString<string>(_redisConnection, key, expiry);
            
            var result = await redis.SetAsync(token);
            
            _logger.LogInformation("Token saved for UserId: {UserId}", userId);
            return result;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to save token for UserId: {UserId}", userId);
            return false;
        }
    }

    public async Task<string?> GetTokenAsync(long userId)
    {
        try
        {
            var key = $"auth:token:{userId}";
            var redis = new RedisString<string>(_redisConnection, key, null);
            
            var result = await redis.GetAsync();
            
            if (result.HasValue)
            {
                _logger.LogInformation("Token retrieved for UserId: {UserId}", userId);
                return result.Value;
            }
            
            _logger.LogWarning("Token not found for UserId: {UserId}", userId);
            return null;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get token for UserId: {UserId}", userId);
            return null;
        }
    }

    public async Task<bool> DeleteTokenAsync(long userId)
    {
        try
        {
            var key = $"auth:token:{userId}";
            var redis = new RedisString<string>(_redisConnection, key, null);
            
            var result = await redis.DeleteAsync();
            
            _logger.LogInformation("Token deleted for UserId: {UserId}", userId);
            return result;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to delete token for UserId: {UserId}", userId);
            return false;
        }
    }

    public async Task<long> IncrementLoginCountAsync(long userId)
    {
        try
        {
            var key = $"user:login_count:{userId}";
            var redis = new RedisString<long>(_redisConnection, key, null);
            
            var count = await redis.IncrementAsync(1);
            
            _logger.LogInformation("Login count incremented for UserId: {UserId}, Count: {Count}", 
                userId, count);
            
            return count;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to increment login count for UserId: {UserId}", userId);
            return 0;
        }
    }
}
```

## 13.3 Hash 타입 - 사용자 세션 정보

### 13.3.1 Hash 타입 개요

Hash는 필드-값 쌍의 집합을 저장하는 데이터 타입이다. 하나의 키 안에 여러 필드를 저장할 수 있어 객체의 속성들을 저장하기에 적합하다. 게임 서버에서는 주로 다음과 같은 용도로 사용한다.

- 사용자 세션 정보
- 캐릭터 상태 정보
- 임시 게임 데이터

### 13.3.3 RedisHash 사용 예제

```csharp
// Services/HashExampleService.cs
using CloudStructures.Structures;
using GameApiServer.Models.Redis;

namespace GameApiServer.Services;

public interface IHashExampleService
{
    Task<bool> SaveUserSessionAsync(UserSession session);
    Task<UserSession?> GetUserSessionAsync(long userId);
    Task<bool> UpdateSessionFieldAsync(long userId, string field, string value);
    Task<Dictionary<string, string>?> GetAllSessionFieldsAsync(long userId);
    Task<bool> DeleteUserSessionAsync(long userId);
}

public class HashExampleService : IHashExampleService
{
    private readonly RedisConnection _redisConnection;
    private readonly ILogger<HashExampleService> _logger;

    public HashExampleService(IRedisService redisService, ILogger<HashExampleService> logger)
    {
        _redisConnection = redisService.GetConnection();
        _logger = logger;
    }

    public async Task<bool> SaveUserSessionAsync(UserSession session)
    {
        try
        {
            var key = $"session:{session.UserId}";
            var redis = new RedisHash<string>(_redisConnection, key, TimeSpan.FromHours(24));

            var fields = new Dictionary<string, string>
            {
                { "UserId", session.UserId.ToString() },
                { "Nickname", session.Nickname },
                { "Level", session.Level.ToString() },
                { "Gold", session.Gold.ToString() },
                { "LastLoginAt", session.LastLoginAt.ToString("O") },
                { "DeviceId", session.DeviceId },
                { "AppVersion", session.AppVersion }
            };

            await redis.SetAsync(fields);

            _logger.LogInformation("User session saved for UserId: {UserId}", session.UserId);
            return true;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to save user session for UserId: {UserId}", session.UserId);
            return false;
        }
    }

    public async Task<UserSession?> GetUserSessionAsync(long userId)
    {
        try
        {
            var key = $"session:{userId}";
            var redis = new RedisHash<string>(_redisConnection, key, null);

            var fields = await redis.GetAllAsync();

            if (fields == null || fields.Count == 0)
            {
                _logger.LogWarning("User session not found for UserId: {UserId}", userId);
                return null;
            }

            var session = new UserSession
            {
                UserId = long.Parse(fields["UserId"]),
                Nickname = fields["Nickname"],
                Level = int.Parse(fields["Level"]),
                Gold = long.Parse(fields["Gold"]),
                LastLoginAt = DateTime.Parse(fields["LastLoginAt"]),
                DeviceId = fields["DeviceId"],
                AppVersion = fields["AppVersion"]
            };

            _logger.LogInformation("User session retrieved for UserId: {UserId}", userId);
            return session;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get user session for UserId: {UserId}", userId);
            return null;
        }
    }

    public async Task<bool> UpdateSessionFieldAsync(long userId, string field, string value)
    {
        try
        {
            var key = $"session:{userId}";
            var redis = new RedisHash<string>(_redisConnection, key, null);

            await redis.SetAsync(field, value);

            _logger.LogInformation(
                "Session field updated for UserId: {UserId}, Field: {Field}",
                userId, field
            );
            return true;
        }
        catch (Exception ex)
        {
            _logger.LogError(
                ex,
                "Failed to update session field for UserId: {UserId}, Field: {Field}",
                userId, field
            );
            return false;
        }
    }

    public async Task<Dictionary<string, string>?> GetAllSessionFieldsAsync(long userId)
    {
        try
        {
            var key = $"session:{userId}";
            var redis = new RedisHash<string>(_redisConnection, key, null);

            var fields = await redis.GetAllAsync();

            _logger.LogInformation("All session fields retrieved for UserId: {UserId}", userId);
            return fields;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get session fields for UserId: {UserId}", userId);
            return null;
        }
    }

    public async Task<bool> DeleteUserSessionAsync(long userId)
    {
        try
        {
            var key = $"session:{userId}";
            var redis = new RedisHash<string>(_redisConnection, key, null);

            var result = await redis.DeleteAsync();

            _logger.LogInformation("User session deleted for UserId: {UserId}", userId);
            return result;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to delete user session for UserId: {UserId}", userId);
            return false;
        }
    }
}
```

## 13.4 List 타입 - 최근 로그인 기록

### 13.4.1 List 타입 개요

List는 순서가 있는 문자열 목록을 저장하는 데이터 타입이다. 양쪽 끝에서 데이터를 추가하거나 제거할 수 있어 큐나 스택으로 활용할 수 있다. 게임 서버에서는 주로 다음과 같은 용도로 사용한다.

- 최근 활동 기록
- 채팅 메시지 히스토리
- 알림 목록
- 작업 큐

### 13.4.3 RedisList 사용 예제

```csharp
// Services/ListExampleService.cs
using CloudStructures.Structures;
using GameApiServer.Models.Redis;
using System.Text.Json;

namespace GameApiServer.Services;

public interface IListExampleService
{
    Task<bool> AddLoginHistoryAsync(long userId, LoginHistory history);
    Task<List<LoginHistory>> GetRecentLoginHistoryAsync(long userId, int count = 10);
    Task<long> GetLoginHistoryCountAsync(long userId);
    Task<bool> ClearOldLoginHistoryAsync(long userId, int keepCount = 50);
}

public class ListExampleService : IListExampleService
{
    private readonly RedisConnection _redisConnection;
    private readonly ILogger<ListExampleService> _logger;

    public ListExampleService(IRedisService redisService, ILogger<ListExampleService> logger)
    {
        _redisConnection = redisService.GetConnection();
        _logger = logger;
    }

    public async Task<bool> AddLoginHistoryAsync(long userId, LoginHistory history)
    {
        try
        {
            var key = $"login_history:{userId}";
            var redis = new RedisList<string>(_redisConnection, key, TimeSpan.FromDays(30));

            var json = JsonSerializer.Serialize(history);
            
            // 리스트의 왼쪽(앞)에 추가 (최신 데이터가 앞에 오도록)
            await redis.LeftPushAsync(json);

            _logger.LogInformation("Login history added for UserId: {UserId}", userId);
            return true;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to add login history for UserId: {UserId}", userId);
            return false;
        }
    }

    public async Task<List<LoginHistory>> GetRecentLoginHistoryAsync(long userId, int count = 10)
    {
        try
        {
            var key = $"login_history:{userId}";
            var redis = new RedisList<string>(_redisConnection, key, null);

            // 왼쪽(앞)에서부터 count개 조회
            var items = await redis.RangeAsync(0, count - 1);

            var histories = items
                .Select(json => JsonSerializer.Deserialize<LoginHistory>(json))
                .Where(h => h != null)
                .Select(h => h!)
                .ToList();

            _logger.LogInformation(
                "Retrieved {Count} login histories for UserId: {UserId}",
                histories.Count, userId
            );

            return histories;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get login history for UserId: {UserId}", userId);
            return new List<LoginHistory>();
        }
    }

    public async Task<long> GetLoginHistoryCountAsync(long userId)
    {
        try
        {
            var key = $"login_history:{userId}";
            var redis = new RedisList<string>(_redisConnection, key, null);

            var count = await redis.LengthAsync();

            _logger.LogInformation("Login history count for UserId: {UserId} is {Count}", userId, count);
            return count;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get login history count for UserId: {UserId}", userId);
            return 0;
        }
    }

    public async Task<bool> ClearOldLoginHistoryAsync(long userId, int keepCount = 50)
    {
        try
        {
            var key = $"login_history:{userId}";
            var redis = new RedisList<string>(_redisConnection, key, null);

            // 최신 keepCount개만 유지하고 나머지 삭제
            await redis.TrimAsync(0, keepCount - 1);

            _logger.LogInformation(
                "Old login history cleared for UserId: {UserId}, Kept: {KeepCount}",
                userId, keepCount
            );
            return true;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to clear old login history for UserId: {UserId}", userId);
            return false;
        }
    }
}
```

## 13.5 Set 타입 - 친구 목록

### 13.5.1 Set 타입 개요

Set은 중복되지 않는 문자열의 집합을 저장하는 데이터 타입이다. 순서가 보장되지 않지만 매우 빠른 조회 성능을 제공한다. 게임 서버에서는 주로 다음과 같은 용도로 사용한다.

- 친구 목록
- 블랙리스트
- 태그 목록
- 집합 연산 (교집합, 합집합, 차집합)

### 13.5.2 RedisSet 사용 예제

```csharp
// Services/SetExampleService.cs
using CloudStructures.Structures;

namespace GameApiServer.Services;

public interface ISetExampleService
{
    Task<bool> AddFriendAsync(long userId, long friendId);
    Task<bool> RemoveFriendAsync(long userId, long friendId);
    Task<bool> IsFriendAsync(long userId, long friendId);
    Task<List<long>> GetFriendListAsync(long userId);
    Task<long> GetFriendCountAsync(long userId);
    Task<List<long>> GetMutualFriendsAsync(long userId1, long userId2);
}

public class SetExampleService : ISetExampleService
{
    private readonly RedisConnection _redisConnection;
    private readonly ILogger<SetExampleService> _logger;

    public SetExampleService(IRedisService redisService, ILogger<SetExampleService> logger)
    {
        _redisConnection = redisService.GetConnection();
        _logger = logger;
    }

    public async Task<bool> AddFriendAsync(long userId, long friendId)
    {
        try
        {
            var key = $"friends:{userId}";
            var redis = new RedisSet<long>(_redisConnection, key, null);

            var added = await redis.AddAsync(friendId);

            if (added)
            {
                _logger.LogInformation(
                    "Friend added. UserId: {UserId}, FriendId: {FriendId}",
                    userId, friendId
                );
            }

            return added;
        }
        catch (Exception ex)
        {
            _logger.LogError(
                ex,
                "Failed to add friend. UserId: {UserId}, FriendId: {FriendId}",
                userId, friendId
            );
            return false;
        }
    }

    public async Task<bool> RemoveFriendAsync(long userId, long friendId)
    {
        try
        {
            var key = $"friends:{userId}";
            var redis = new RedisSet<long>(_redisConnection, key, null);

            var removed = await redis.RemoveAsync(friendId);

            if (removed)
            {
                _logger.LogInformation(
                    "Friend removed. UserId: {UserId}, FriendId: {FriendId}",
                    userId, friendId
                );
            }

            return removed;
        }
        catch (Exception ex)
        {
            _logger.LogError(
                ex,
                "Failed to remove friend. UserId: {UserId}, FriendId: {FriendId}",
                userId, friendId
            );
            return false;
        }
    }

    public async Task<bool> IsFriendAsync(long userId, long friendId)
    {
        try
        {
            var key = $"friends:{userId}";
            var redis = new RedisSet<long>(_redisConnection, key, null);

            var isMember = await redis.ContainsAsync(friendId);

            _logger.LogInformation(
                "Friend check. UserId: {UserId}, FriendId: {FriendId}, IsFriend: {IsFriend}",
                userId, friendId, isMember
            );

            return isMember;
        }
        catch (Exception ex)
        {
            _logger.LogError(
                ex,
                "Failed to check friend. UserId: {UserId}, FriendId: {FriendId}",
                userId, friendId
            );
            return false;
        }
    }

    public async Task<List<long>> GetFriendListAsync(long userId)
    {
        try
        {
            var key = $"friends:{userId}";
            var redis = new RedisSet<long>(_redisConnection, key, null);

            var members = await redis.MembersAsync();
            var friendList = members.ToList();

            _logger.LogInformation(
                "Friend list retrieved for UserId: {UserId}, Count: {Count}",
                userId, friendList.Count
            );

            return friendList;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get friend list for UserId: {UserId}", userId);
            return new List<long>();
        }
    }

    public async Task<long> GetFriendCountAsync(long userId)
    {
        try
        {
            var key = $"friends:{userId}";
            var redis = new RedisSet<long>(_redisConnection, key, null);

            var count = await redis.LengthAsync();

            _logger.LogInformation("Friend count for UserId: {UserId} is {Count}", userId, count);
            return count;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get friend count for UserId: {UserId}", userId);
            return 0;
        }
    }

    public async Task<List<long>> GetMutualFriendsAsync(long userId1, long userId2)
    {
        try
        {
            var key1 = $"friends:{userId1}";
            var key2 = $"friends:{userId2}";
            var redis1 = new RedisSet<long>(_redisConnection, key1, null);

            // 교집합 계산
            var mutualFriends = await redis1.IntersectAsync(new[] { key2 });
            var result = mutualFriends.ToList();

            _logger.LogInformation(
                "Mutual friends found. User1: {UserId1}, User2: {UserId2}, Count: {Count}",
                userId1, userId2, result.Count
            );

            return result;
        }
        catch (Exception ex)
        {
            _logger.LogError(
                ex,
                "Failed to get mutual friends. User1: {UserId1}, User2: {UserId2}",
                userId1, userId2
            );
            return new List<long>();
        }
    }
```

### 13.6.1 Sorted Set 타입 개요

Sorted Set은 각 요소에 점수(score)를 부여하여 정렬된 상태로 저장하는 데이터 타입이다. 점수를 기준으로 빠른 조회와 범위 검색이 가능하다. 게임 서버에서는 주로 다음과 같은 용도로 사용한다.

- 랭킹 시스템
- 리더보드
- 우선순위 큐
- 시간 기반 정렬 데이터

### 13.6.3 RedisSortedSet 사용 예제

```csharp
// Services/SortedSetExampleService.cs
using CloudStructures.Structures;
using GameApiServer.Models.Redis;

namespace GameApiServer.Services;

public interface ISortedSetExampleService
{
    Task<bool> UpdateScoreAsync(long userId, long score);
    Task<long?> GetScoreAsync(long userId);
    Task<int?> GetRankAsync(long userId);
    Task<List<RankingEntry>> GetTopRankingsAsync(int count = 10);
    Task<List<RankingEntry>> GetRankingRangeAsync(int startRank, int endRank);
    Task<List<RankingEntry>> GetAroundRankingsAsync(long userId, int range = 5);
}

public class SortedSetExampleService : ISortedSetExampleService
{
    private readonly RedisConnection _redisConnection;
    private readonly ILogger<SortedSetExampleService> _logger;

    public SortedSetExampleService(IRedisService redisService, ILogger<SortedSetExampleService> logger)
    {
        _redisConnection = redisService.GetConnection();
        _logger = logger;
    }

    public async Task<bool> UpdateScoreAsync(long userId, long score)
    {
        try
        {
            var key = "ranking:global";
            var redis = new RedisSortedSet<long>(_redisConnection, key, null);

            // score가 높을수록 상위 랭킹이 되도록 저장
            await redis.AddAsync(userId, score);

            _logger.LogInformation("Score updated. UserId: {UserId}, Score: {Score}", userId, score);
            return true;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to update score for UserId: {UserId}", userId);
            return false;
        }
    }

    public async Task<long?> GetScoreAsync(long userId)
    {
        try
        {
            var key = "ranking:global";
            var redis = new RedisSortedSet<long>(_redisConnection, key, null);

            var score = await redis.ScoreAsync(userId);

            if (score.HasValue)
            {
                _logger.LogInformation("Score retrieved. UserId: {UserId}, Score: {Score}", userId, score.Value);
                return (long)score.Value;
            }

            _logger.LogWarning("Score not found for UserId: {UserId}", userId);
            return null;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get score for UserId: {UserId}", userId);
            return null;
        }
    }

    public async Task<int?> GetRankAsync(long userId)
    {
        try
        {
            var key = "ranking:global";
            var redis = new RedisSortedSet<long>(_redisConnection, key, null);

            // 내림차순 순위 (점수가 높을수록 상위)
            var rank = await redis.RankAsync(userId, order: StackExchange.Redis.Order.Descending);

            if (rank.HasValue)
            {
                // Redis는 0부터 시작하므로 +1
                var actualRank = (int)rank.Value + 1;
                _logger.LogInformation("Rank retrieved. UserId: {UserId}, Rank: {Rank}", userId, actualRank);
                return actualRank;
            }

            _logger.LogWarning("Rank not found for UserId: {UserId}", userId);
            return null;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get rank for UserId: {UserId}", userId);
            return null;
        }
    }

    public async Task<List<RankingEntry>> GetTopRankingsAsync(int count = 10)
    {
        try
        {
            var key = "ranking:global";
            var redis = new RedisSortedSet<long>(_redisConnection, key, null);

            // 상위 count개 조회 (내림차순)
            var entries = await redis.RangeByRankWithScoresAsync(
                0,
                count - 1,
                order: StackExchange.Redis.Order.Descending
            );

            var rankings = entries.Select((entry, index) => new RankingEntry
            {
                UserId = entry.Value,
                Score = (long)entry.Score,
                Rank = index + 1,
                Nickname = $"User{entry.Value}" // 실제로는 DB에서 닉네임 조회 필요
            }).ToList();

            _logger.LogInformation("Top {Count} rankings retrieved", count);
            return rankings;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get top rankings");
            return new List<RankingEntry>();
        }
    }

    public async Task<List<RankingEntry>> GetRankingRangeAsync(int startRank, int endRank)
    {
        try
        {
            var key = "ranking:global";
            var redis = new RedisSortedSet<long>(_redisConnection, key, null);

            // 특정 범위 조회 (Redis는 0부터 시작하므로 -1)
            var entries = await redis.RangeByRankWithScoresAsync(
                startRank - 1,
                endRank - 1,
                order: StackExchange.Redis.Order.Descending
            );

            var rankings = entries.Select((entry, index) => new RankingEntry
            {
                UserId = entry.Value,
                Score = (long)entry.Score,
                Rank = startRank + index,
                Nickname = $"User{entry.Value}"
            }).ToList();

            _logger.LogInformation("Rankings retrieved. Range: {Start}-{End}", startRank, endRank);
            return rankings;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get ranking range");
            return new List<RankingEntry>();
        }
    }

    public async Task<List<RankingEntry>> GetAroundRankingsAsync(long userId, int range = 5)
    {
        try
        {
            var rank = await GetRankAsync(userId);
            if (!rank.HasValue)
            {
                return new List<RankingEntry>();
            }

            var startRank = Math.Max(1, rank.Value - range);
            var endRank = rank.Value + range;

            var rankings = await GetRankingRangeAsync(startRank, endRank);

            _logger.LogInformation(
                "Around rankings retrieved for UserId: {UserId}, Range: ±{Range}",
                userId, range
            );

            return rankings;
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "Failed to get around rankings for UserId: {UserId}", userId);
            return new List<RankingEntry>();
        }
    }
}
```

## 13.10 정리

이번 장에서는 Redis의 5가지 핵심 데이터 타입을 실습을 통해 학습했다. 각 타입의 특징과 게임 서버에서의 활용 방법을 익혔다.

주요 학습 내용은 다음과 같다.

- **String 타입**은 인증 토큰, 카운터 등 단순한 키-값 저장에 적합하다
- **Hash 타입**은 사용자 세션처럼 여러 필드를 가진 객체 데이터 저장에 유용하다
- **List 타입**은 로그인 기록 같은 시간순 데이터나 큐 구현에 효과적이다
- **Set 타입**은 친구 목록처럼 중복 없는 집합 데이터와 집합 연산에 최적화되어 있다
- **Sorted Set 타입**은 랭킹 시스템처럼 정렬이 필요한 데이터에 가장 적합하다

다음 장에서는 이러한 Redis 데이터 타입을 활용한 다양한 캐시 전략을 구현해본다.
