# 18장. 상점 시스템 (실습)

이번 장에서는 게임 내 재화를 사용하여 아이템을 구매하는 상점 시스템을 구현한다. 상점 상품 마스터 데이터를 관리하고, Redis를 활용하여 성능을 최적화하며, 트랜잭션을 통해 재화 차감과 아이템 지급의 원자성을 보장하는 방법을 학습한다.

## 18.1 상점 시스템 개요

상점 시스템은 사용자가 게임 내 재화를 사용하여 아이템이나 다른 재화를 구매하는 메커니즘이다. 일반적으로 골드 상점, 다이아 상점, 이벤트 상점 등 여러 종류의 상점이 존재한다.

상점 시스템의 핵심은 구매 과정의 원자성 보장이다. 재화를 차감했는데 아이템 지급에 실패하거나, 아이템을 지급했는데 재화 차감에 실패하는 상황이 발생해서는 안 된다. 이를 위해 데이터베이스 트랜잭션을 적극 활용한다.

또한 구매 제한이나 일일 구매 횟수 제한 같은 정책을 구현하여 게임 밸런스를 유지해야 한다. 구매 이력을 저장하여 부정 구매를 추적하고, 환불이나 보상 지급 시 참고 자료로 활용할 수 있다.

## 18.2 데이터베이스 테이블 설계

상점 상품 마스터 데이터를 저장할 테이블을 생성한다.

```sql
CREATE TABLE shop_product (
    product_id INT PRIMARY KEY,
    product_name VARCHAR(100) NOT NULL,
    shop_type TINYINT NOT NULL COMMENT '1:골드상점, 2:다이아상점, 3:이벤트상점',
    price_type TINYINT NOT NULL COMMENT '1:골드, 2:다이아, 3:이벤트재화',
    price_amount INT NOT NULL,
    reward_type TINYINT NOT NULL COMMENT '1:아이템, 2:골드, 3:다이아',
    reward_item_id INT DEFAULT NULL,
    reward_amount INT NOT NULL,
    daily_purchase_limit INT DEFAULT 0 COMMENT '0:무제한',
    total_purchase_limit INT DEFAULT 0 COMMENT '0:무제한',
    display_order INT NOT NULL DEFAULT 0,
    is_active BOOLEAN NOT NULL DEFAULT true,
    start_date DATETIME DEFAULT NULL,
    end_date DATETIME DEFAULT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_shop_type (shop_type, is_active, display_order)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

사용자의 구매 이력을 저장할 테이블을 생성한다.

```sql
CREATE TABLE user_purchase_history (
    purchase_id BIGINT PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT NOT NULL,
    product_id INT NOT NULL,
    price_type TINYINT NOT NULL,
    price_amount INT NOT NULL,
    reward_type TINYINT NOT NULL,
    reward_item_id INT DEFAULT NULL,
    reward_amount INT NOT NULL,
    purchased_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_user_id (user_id, purchased_at),
    INDEX idx_product_id (product_id),
    FOREIGN KEY (user_id) REFERENCES users(user_id) ON DELETE CASCADE,
    FOREIGN KEY (product_id) REFERENCES shop_product(product_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

일일 구매 횟수를 추적하기 위한 테이블을 생성한다.

```sql
CREATE TABLE user_daily_purchase (
    user_id BIGINT NOT NULL,
    product_id INT NOT NULL,
    purchase_date DATE NOT NULL,
    purchase_count INT NOT NULL DEFAULT 0,
    PRIMARY KEY (user_id, product_id, purchase_date),
    INDEX idx_user_date (user_id, purchase_date),
    FOREIGN KEY (user_id) REFERENCES users(user_id) ON DELETE CASCADE,
    FOREIGN KEY (product_id) REFERENCES shop_product(product_id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4;
```

## 18.4 상점 마스터 데이터 서비스

상점 상품 마스터 데이터를 로딩하고 관리하는 서비스를 구현한다.

```csharp
// Services/ShopMasterDataService.cs
using GameServer.Models;
using SqlKata.Execution;
using System.Collections.Concurrent;

namespace GameServer.Services
{
    public class ShopMasterDataService
    {
        private readonly QueryFactory _queryFactory;
        private ConcurrentDictionary<int, ShopProduct> _products;
        private ConcurrentDictionary<byte, List<ShopProduct>> _productsByShopType;

        public ShopMasterDataService(QueryFactory queryFactory)
        {
            _queryFactory = queryFactory;
            _products = new ConcurrentDictionary<int, ShopProduct>();
            _productsByShopType = new ConcurrentDictionary<byte, List<ShopProduct>>();
        }

        public async Task LoadShopDataAsync()
        {
            var products = await _queryFactory.Query("shop_product")
                .GetAsync<ShopProduct>();

            _products.Clear();
            _productsByShopType.Clear();

            foreach (var product in products)
            {
                _products.TryAdd(product.ProductId, product);

                if (!_productsByShopType.ContainsKey(product.ShopType))
                {
                    _productsByShopType[product.ShopType] = new List<ShopProduct>();
                }
                _productsByShopType[product.ShopType].Add(product);
            }

            // 각 상점 타입별로 표시 순서대로 정렬한다
            foreach (var shopType in _productsByShopType.Keys)
            {
                _productsByShopType[shopType] = _productsByShopType[shopType]
                    .OrderBy(p => p.DisplayOrder)
                    .ToList();
            }

            Console.WriteLine($"[ShopMasterData] Loaded {_products.Count} products");
        }

        public ShopProduct GetProduct(int productId)
        {
            _products.TryGetValue(productId, out var product);
            return product;
        }

        public List<ShopProduct> GetProductsByShopType(byte shopType)
        {
            _productsByShopType.TryGetValue(shopType, out var products);
            return products ?? new List<ShopProduct>();
        }

        public bool IsValidProduct(int productId)
        {
            return _products.ContainsKey(productId);
        }

        public bool IsProductAvailable(ShopProduct product)
        {
            if (!product.IsActive)
                return false;

            var now = DateTime.Now;

            if (product.StartDate.HasValue && now < product.StartDate.Value)
                return false;

            if (product.EndDate.HasValue && now > product.EndDate.Value)
                return false;

            return true;
        }
    }
}
```

## 18.5 Redis를 이용한 상점 데이터 캐싱

상점 목록을 Redis에 캐싱하여 빠르게 조회할 수 있도록 구현한다.

```csharp
// Services/ShopCacheService.cs
using CloudStructures;
using CloudStructures.Structures;
using GameServer.Models;
using System.Text.Json;

namespace GameServer.Services
{
    public class ShopCacheService
    {
        private readonly RedisConnection _redisConnection;
        private readonly TimeSpan _cacheExpiry = TimeSpan.FromMinutes(10);

        public ShopCacheService(RedisConnection redisConnection)
        {
            _redisConnection = redisConnection;
        }

        private string GetShopListCacheKey(byte shopType)
        {
            return $"shop:list:{shopType}";
        }

        public async Task<List<ShopProduct>> GetShopListAsync(byte shopType)
        {
            var redis = new RedisString<string>(_redisConnection, GetShopListCacheKey(shopType), _cacheExpiry);
            var cachedData = await redis.GetAsync();

            if (cachedData.HasValue)
            {
                return JsonSerializer.Deserialize<List<ShopProduct>>(cachedData.Value);
            }

            return null;
        }

        public async Task SetShopListAsync(byte shopType, List<ShopProduct> products)
        {
            var redis = new RedisString<string>(_redisConnection, GetShopListCacheKey(shopType), _cacheExpiry);
            var json = JsonSerializer.Serialize(products);
            await redis.SetAsync(json, _cacheExpiry);
        }

        public async Task InvalidateShopListAsync(byte shopType)
        {
            var redis = new RedisString<string>(_redisConnection, GetShopListCacheKey(shopType), _cacheExpiry);
            await redis.DeleteAsync();
        }

        public async Task InvalidateAllShopListsAsync()
        {
            foreach (byte shopType in Enum.GetValues(typeof(ShopType)))
            {
                await InvalidateShopListAsync(shopType);
            }
        }
    }
}
```

## 18.6 상점 Repository 구현

상점 관련 데이터를 MySQL에서 조회하고 수정하는 Repository를 구현한다.

```csharp
// Repositories/ShopRepository.cs
using GameServer.Models;
using SqlKata.Execution;
using System.Data;

namespace GameServer.Repositories
{
    public class ShopRepository
    {
        private readonly QueryFactory _queryFactory;

        public ShopRepository(QueryFactory queryFactory)
        {
            _queryFactory = queryFactory;
        }

        public async Task<int> GetUserGoldAsync(long userId)
        {
            var result = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .Select("gold")
                .FirstOrDefaultAsync<dynamic>();

            return result?.gold ?? 0;
        }

        public async Task<int> GetUserDiamondAsync(long userId)
        {
            var result = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .Select("diamond")
                .FirstOrDefaultAsync<dynamic>();

            return result?.diamond ?? 0;
        }

        public async Task<int> GetUserEventCurrencyAsync(long userId)
        {
            var result = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .Select("event_currency")
                .FirstOrDefaultAsync<dynamic>();

            return result?.event_currency ?? 0;
        }

        public async Task<bool> DeductGoldAsync(IDbTransaction transaction, long userId, int amount)
        {
            var affected = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .Where("gold", ">=", amount)
                .DecrementAsync("gold", amount, transaction);

            return affected > 0;
        }

        public async Task<bool> DeductDiamondAsync(IDbTransaction transaction, long userId, int amount)
        {
            var affected = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .Where("diamond", ">=", amount)
                .DecrementAsync("diamond", amount, transaction);

            return affected > 0;
        }

        public async Task<bool> DeductEventCurrencyAsync(IDbTransaction transaction, long userId, int amount)
        {
            var affected = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .Where("event_currency", ">=", amount)
                .DecrementAsync("event_currency", amount, transaction);

            return affected > 0;
        }

        public async Task<bool> AddGoldAsync(IDbTransaction transaction, long userId, int amount)
        {
            var affected = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .IncrementAsync("gold", amount, transaction);

            return affected > 0;
        }

        public async Task<bool> AddDiamondAsync(IDbTransaction transaction, long userId, int amount)
        {
            var affected = await _queryFactory.Query("users")
                .Where("user_id", userId)
                .IncrementAsync("diamond", amount, transaction);

            return affected > 0;
        }

        public async Task<long> CreatePurchaseHistoryAsync(IDbTransaction transaction, UserPurchaseHistory history)
        {
            var purchaseId = await _queryFactory.Query("user_purchase_history")
                .InsertGetIdAsync<long>(new
                {
                    user_id = history.UserId,
                    product_id = history.ProductId,
                    price_type = history.PriceType,
                    price_amount = history.PriceAmount,
                    reward_type = history.RewardType,
                    reward_item_id = history.RewardItemId,
                    reward_amount = history.RewardAmount,
                    purchased_at = DateTime.Now
                }, transaction);

            return purchaseId;
        }

        public async Task<int> GetDailyPurchaseCountAsync(long userId, int productId, DateTime date)
        {
            var result = await _queryFactory.Query("user_daily_purchase")
                .Where("user_id", userId)
                .Where("product_id", productId)
                .Where("purchase_date", date.Date)
                .Select("purchase_count")
                .FirstOrDefaultAsync<dynamic>();

            return result?.purchase_count ?? 0;
        }

        public async Task<int> GetTotalPurchaseCountAsync(long userId, int productId)
        {
            var count = await _queryFactory.Query("user_purchase_history")
                .Where("user_id", userId)
                .Where("product_id", productId)
                .CountAsync<int>();

            return count;
        }

        public async Task IncrementDailyPurchaseCountAsync(IDbTransaction transaction, long userId, int productId, DateTime date, int count)
        {
            // UPSERT 방식으로 구현한다
            var existing = await _queryFactory.Query("user_daily_purchase")
                .Where("user_id", userId)
                .Where("product_id", productId)
                .Where("purchase_date", date.Date)
                .FirstOrDefaultAsync<UserDailyPurchase>(transaction);

            if (existing != null)
            {
                await _queryFactory.Query("user_daily_purchase")
                    .Where("user_id", userId)
                    .Where("product_id", productId)
                    .Where("purchase_date", date.Date)
                    .IncrementAsync("purchase_count", count, transaction);
            }
            else
            {
                await _queryFactory.Query("user_daily_purchase")
                    .InsertAsync(new
                    {
                        user_id = userId,
                        product_id = productId,
                        purchase_date = date.Date,
                        purchase_count = count
                    }, transaction);
            }
        }
    }
}
```

## 18.7 상점 서비스 구현

상점 관련 비즈니스 로직을 처리하는 서비스를 구현한다.

```csharp
// Services/ShopService.cs
using GameServer.DTOs;
using GameServer.Models;
using GameServer.Repositories;
using MySqlConnector;

namespace GameServer.Services
{
    public class ShopService
    {
        private readonly ShopRepository _shopRepository;
        private readonly InventoryRepository _inventoryRepository;
        private readonly ShopMasterDataService _masterDataService;
        private readonly ShopCacheService _cacheService;
        private readonly MasterDataService _itemMasterDataService;
        private readonly MySqlConnection _connection;

        public ShopService(
            ShopRepository shopRepository,
            InventoryRepository inventoryRepository,
            ShopMasterDataService masterDataService,
            ShopCacheService cacheService,
            MasterDataService itemMasterDataService,
            MySqlConnection connection)
        {
            _shopRepository = shopRepository;
            _inventoryRepository = inventoryRepository;
            _masterDataService = masterDataService;
            _cacheService = cacheService;
            _itemMasterDataService = itemMasterDataService;
            _connection = connection;
        }

        public async Task<GetShopListResponse> GetShopListAsync(long userId, byte shopType)
        {
            // 캐시에서 상점 목록을 조회한다
            var products = await _cacheService.GetShopListAsync(shopType);

            if (products == null)
            {
                // 캐시 미스 시 마스터 데이터에서 조회하고 캐싱한다
                products = _masterDataService.GetProductsByShopType(shopType);
                if (products.Count > 0)
                {
                    await _cacheService.SetShopListAsync(shopType, products);
                }
            }

            var productDtos = new List<ShopProductDto>();

            foreach (var product in products)
            {
                // 판매 가능한 상품인지 확인한다
                if (!_masterDataService.IsProductAvailable(product))
                    continue;

                // 사용자의 구매 정보를 조회한다
                var dailyCount = await _shopRepository.GetDailyPurchaseCountAsync(
                    userId, product.ProductId, DateTime.Now.Date);
                var totalCount = await _shopRepository.GetTotalPurchaseCountAsync(
                    userId, product.ProductId);

                // 구매 가능 여부를 확인한다
                bool canPurchase = true;
                if (product.DailyPurchaseLimit > 0 && dailyCount >= product.DailyPurchaseLimit)
                    canPurchase = false;
                if (product.TotalPurchaseLimit > 0 && totalCount >= product.TotalPurchaseLimit)
                    canPurchase = false;

                // 보상 아이템 이름을 조회한다
                string rewardItemName = null;
                if (product.RewardType == (byte)RewardType.Item && product.RewardItemId.HasValue)
                {
                    var itemMaster = _itemMasterDataService.GetItem(product.RewardItemId.Value);
                    rewardItemName = itemMaster?.ItemName;
                }

                productDtos.Add(new ShopProductDto
                {
                    ProductId = product.ProductId,
                    ProductName = product.ProductName,
                    PriceType = product.PriceType,
                    PriceAmount = product.PriceAmount,
                    RewardType = product.RewardType,
                    RewardItemId = product.RewardItemId,
                    RewardItemName = rewardItemName,
                    RewardAmount = product.RewardAmount,
                    DailyPurchaseLimit = product.DailyPurchaseLimit,
                    TotalPurchaseLimit = product.TotalPurchaseLimit,
                    UserDailyPurchaseCount = dailyCount,
                    UserTotalPurchaseCount = totalCount,
                    CanPurchase = canPurchase
                });
            }

            return new GetShopListResponse
            {
                Result = 0,
                Products = productDtos
            };
        }

        public async Task<PurchaseProductResponse> PurchaseProductAsync(long userId, int productId, int quantity)
        {
            // 상품 정보를 조회한다
            var product = _masterDataService.GetProduct(productId);
            if (product == null)
            {
                return new PurchaseProductResponse { Result = 1 }; // 존재하지 않는 상품
            }

            // 판매 가능한 상품인지 확인한다
            if (!_masterDataService.IsProductAvailable(product))
            {
                return new PurchaseProductResponse { Result = 2 }; // 판매 불가 상품
            }

            // 수량이 유효한지 확인한다
            if (quantity <= 0)
            {
                return new PurchaseProductResponse { Result = 3 }; // 잘못된 수량
            }

            // 구매 제한을 확인한다
            var dailyCount = await _shopRepository.GetDailyPurchaseCountAsync(
                userId, productId, DateTime.Now.Date);
            var totalCount = await _shopRepository.GetTotalPurchaseCountAsync(userId, productId);

            if (product.DailyPurchaseLimit > 0 && dailyCount + quantity > product.DailyPurchaseLimit)
            {
                return new PurchaseProductResponse { Result = 4 }; // 일일 구매 제한 초과
            }

            if (product.TotalPurchaseLimit > 0 && totalCount + quantity > product.TotalPurchaseLimit)
            {
                return new PurchaseProductResponse { Result = 5 }; // 총 구매 제한 초과
            }

            // 필요한 재화와 지급할 보상을 계산한다
            int totalPrice = product.PriceAmount * quantity;
            int totalReward = product.RewardAmount * quantity;

            // 트랜잭션으로 처리한다
            await _connection.OpenAsync();
            using var transaction = await _connection.BeginTransactionAsync();

            try
            {
                // 재화를 차감한다
                bool deducted = false;
                if (product.PriceType == (byte)CurrencyType.Gold)
                {
                    deducted = await _shopRepository.DeductGoldAsync(transaction, userId, totalPrice);
                }
                else if (product.PriceType == (byte)CurrencyType.Diamond)
                {
                    deducted = await _shopRepository.DeductDiamondAsync(transaction, userId, totalPrice);
                }
                else if (product.PriceType == (byte)CurrencyType.EventCurrency)
                {
                    deducted = await _shopRepository.DeductEventCurrencyAsync(transaction, userId, totalPrice);
                }

                if (!deducted)
                {
                    await transaction.RollbackAsync();
                    return new PurchaseProductResponse { Result = 6 }; // 재화 부족
                }

                // 보상을 지급한다
                if (product.RewardType == (byte)RewardType.Item)
                {
                    if (!product.RewardItemId.HasValue)
                    {
                        await transaction.RollbackAsync();
                        return new PurchaseProductResponse { Result = 7 }; // 잘못된 상품 설정
                    }

                    // 아이템을 지급한다 (트랜잭션 내에서)
                    await _inventoryRepository.AddItemAsync(userId, product.RewardItemId.Value, totalReward);
                }
                else if (product.RewardType == (byte)RewardType.Gold)
                {
                    await _shopRepository.AddGoldAsync(transaction, userId, totalReward);
                }
                else if (product.RewardType == (byte)RewardType.Diamond)
                {
                    await _shopRepository.AddDiamondAsync(transaction, userId, totalReward);
                }

                // 구매 이력을 저장한다
                var history = new UserPurchaseHistory
                {
                    UserId = userId,
                    ProductId = productId,
                    PriceType = product.PriceType,
                    PriceAmount = totalPrice,
                    RewardType = product.RewardType,
                    RewardItemId = product.RewardItemId,
                    RewardAmount = totalReward
                };

                var purchaseId = await _shopRepository.CreatePurchaseHistoryAsync(transaction, history);

                // 일일 구매 횟수를 증가시킨다
                await _shopRepository.IncrementDailyPurchaseCountAsync(
                    transaction, userId, productId, DateTime.Now.Date, quantity);

                // 트랜잭션을 커밋한다
                await transaction.CommitAsync();

                // 사용자의 현재 재화 정보를 조회한다
                var gold = await _shopRepository.GetUserGoldAsync(userId);
                var diamond = await _shopRepository.GetUserDiamondAsync(userId);
                var eventCurrency = await _shopRepository.GetUserEventCurrencyAsync(userId);

                return new PurchaseProductResponse
                {
                    Result = 0,
                    PurchaseId = purchaseId,
                    Reward = new RewardInfo
                    {
                        RewardType = product.RewardType,
                        RewardItemId = product.RewardItemId,
                        RewardAmount = totalReward
                    },
                    Currency = new UserCurrencyInfo
                    {
                        Gold = gold,
                        Diamond = diamond,
                        EventCurrency = eventCurrency
                    }
                };
            }
            catch (Exception)
            {
                await transaction.RollbackAsync();
                throw;
            }
            finally
            {
                await _connection.CloseAsync();
            }
        }
    }
}
```

## 18.12 트랜잭션의 중요성

상점 시스템에서 트랜잭션은 매우 중요하다. 재화 차감과 보상 지급이 원자적으로 처리되지 않으면 심각한 문제가 발생할 수 있다.

다음과 같은 상황을 고려해야 한다. 재화를 차감했는데 아이템 지급에 실패하면 사용자는 손해를 입는다. 아이템을 지급했는데 재화 차감에 실패하면 게임 경제가 붕괴된다. 구매 이력은 저장되었는데 실제 처리가 실패하면 데이터 정합성이 깨진다.

이러한 문제를 방지하기 위해 모든 작업을 하나의 트랜잭션으로 묶어서 처리한다.

```csharp
using var transaction = await _connection.BeginTransactionAsync();
try
{
    // 1. 재화 차감
    await DeductCurrencyAsync(transaction, ...);
    
    // 2. 보상 지급
    await GiveRewardAsync(transaction, ...);
    
    // 3. 구매 이력 저장
    await SavePurchaseHistoryAsync(transaction, ...);
    
    // 4. 일일 구매 횟수 증가
    await IncrementDailyCountAsync(transaction, ...);
    
    // 모두 성공하면 커밋한다
    await transaction.CommitAsync();
}
catch (Exception)
{
    // 하나라도 실패하면 모두 롤백한다
    await transaction.RollbackAsync();
    throw;
}
```

## 18.13 구매 제한 검증

상점 시스템에서 구매 제한을 제대로 검증하지 않으면 게임 밸런스가 무너질 수 있다. 일일 구매 제한과 총 구매 제한을 정확하게 확인해야 한다.

특히 동시 요청이 발생했을 때 구매 제한이 우회되지 않도록 주의해야 한다. 이를 위해 구매 횟수 증가도 트랜잭션 내에서 처리하고, 필요하다면 Redis 분산 락을 사용할 수 있다.

```csharp
// 구매 전 제한 확인
var dailyCount = await GetDailyPurchaseCountAsync(userId, productId);
if (dailyCount + quantity > product.DailyPurchaseLimit)
{
    return new Response { Result = 4 }; // 일일 구매 제한 초과
}

// 트랜잭션 내에서 구매 처리
using var transaction = await _connection.BeginTransactionAsync();
try
{
    // 재화 차감 및 보상 지급
    // ...
    
    // 구매 횟수 증가 (트랜잭션 내)
    await IncrementDailyPurchaseCountAsync(transaction, userId, productId, quantity);
    
    await transaction.CommitAsync();
}
catch
{
    await transaction.RollbackAsync();
    throw;
}
```
