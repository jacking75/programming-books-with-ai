메모리 정렬과 캐시 일관성은 현대 하드웨어에서 성능과 정확성에 직접적인 영향을 미친다. 잘못된 정렬은 성능 저하, 크래시, 데이터 레이스를 초래할 수 있다. C++26은 메모리 정렬을 안전하게 다루기 위한 여러 기능을 제공한다. 이 장에서는 정렬 보장, 타입 특성, false sharing 방지를 통해 안전하고 효율적인 메모리 접근을 구현하는 방법을 다룬다.

## 14.1 `std::aligned_accessor`로 정렬 보장

### 문제: 정렬되지 않은 메모리 접근의 위험성

정렬되지 않은 메모리 접근은 성능 저하나 하드웨어 예외를 발생시킨다.

**안전하지 않은 코드:**

```cpp
#include <iostream>
#include <cstring>
#include <cstdint>

// 정렬되지 않은 메모리 접근
struct UnalignedData {
    char padding;
    int64_t value;  // 1바이트 오프셋에 8바이트 정렬 필요
} __attribute__((packed));

void process_unaligned_unsafe(void* ptr) {
    // 정렬되지 않은 포인터를 int64_t*로 캐스팅
    int64_t* aligned_ptr = reinterpret_cast<int64_t*>(ptr);
    *aligned_ptr = 42;  // 정렬되지 않은 접근 - 성능 저하 또는 크래시
}

// SIMD 연산에서의 정렬 문제
void simd_unaligned_unsafe(float* data, size_t size) {
    // data가 16바이트 정렬되어 있다고 가정
    __m128* simd_ptr = reinterpret_cast<__m128*>(data);
    
    for (size_t i = 0; i < size / 4; ++i) {
        // 정렬되지 않은 경우 크래시 가능
        __m128 val = _mm_load_ps(reinterpret_cast<float*>(&simd_ptr[i]));
        // ... 처리
    }
}

int main() {
    // 정렬되지 않은 할당
    char buffer[100];
    void* unaligned = &buffer[1];  // 의도적으로 정렬 깨뜨리기
    
    std::cout << "버퍼 주소: " << static_cast<void*>(buffer) << "\n";
    std::cout << "정렬되지 않은 주소: " << unaligned << "\n";
    
    // 위험한 연산
    process_unaligned_unsafe(unaligned);
    
    // SIMD 연산
    float data[16];  // 정렬 보장 없음
    simd_unaligned_unsafe(data, 16);
    
    return 0;
}
```

이 코드의 문제점은 다음과 같다.

- 정렬되지 않은 메모리 접근으로 성능이 저하된다
- 일부 아키텍처에서는 정렬되지 않은 접근이 하드웨어 예외를 발생시킨다
- SIMD 명령어는 특정 정렬을 요구하며, 위반 시 크래시가 발생한다
- 컴파일러 최적화가 정렬을 가정하여 잘못된 코드를 생성할 수 있다

### 해결책: C++26의 정렬 보장 기능

C++26은 `std::aligned_accessor`와 관련 기능으로 정렬을 보장한다.

**안전한 코드:**

```cpp
#include <iostream>
#include <memory>
#include <span>
#include <cstdint>
#include <bit>

// C++26 aligned_accessor (표준 라이브러리 구현 대기 중)
// 여기서는 개념적 구현을 보여준다
template<typename T, size_t Alignment = alignof(T)>
class aligned_accessor {
    static_assert(std::has_single_bit(Alignment), 
                  "정렬은 2의 거듭제곱이어야 한다");
    static_assert(Alignment >= alignof(T), 
                  "정렬은 타입의 기본 정렬보다 크거나 같아야 한다");
    
    T* ptr_;
    
public:
    aligned_accessor() : ptr_(nullptr) {}
    
    explicit aligned_accessor(T* p) : ptr_(p) {
        if (ptr_ && reinterpret_cast<uintptr_t>(ptr_) % Alignment != 0) {
            throw std::runtime_error("포인터가 요구된 정렬을 만족하지 않는다");
        }
    }
    
    T& operator*() const noexcept {
        return *std::assume_aligned<Alignment>(ptr_);
    }
    
    T* operator->() const noexcept {
        return std::assume_aligned<Alignment>(ptr_);
    }
    
    T& operator[](size_t index) const noexcept {
        return std::assume_aligned<Alignment>(ptr_)[index];
    }
    
    T* get() const noexcept {
        return std::assume_aligned<Alignment>(ptr_);
    }
    
    explicit operator bool() const noexcept {
        return ptr_ != nullptr;
    }
};

// 정렬된 메모리 할당 헬퍼
template<typename T, size_t Alignment = alignof(T)>
class aligned_allocator {
public:
    using value_type = T;
    static constexpr size_t alignment = Alignment;
    
    T* allocate(size_t n) {
        void* ptr = std::aligned_alloc(Alignment, n * sizeof(T));
        if (!ptr) {
            throw std::bad_alloc();
        }
        return static_cast<T*>(ptr);
    }
    
    void deallocate(T* ptr, size_t) noexcept {
        std::free(ptr);
    }
};

// 안전한 정렬 메모리 접근
template<typename T, size_t Alignment>
void process_aligned_safe(aligned_accessor<T, Alignment> accessor, size_t count) {
    for (size_t i = 0; i < count; ++i) {
        accessor[i] = T{};  // 정렬이 보장된 접근
    }
}

// SIMD를 위한 안전한 정렬 보장
template<typename T>
class simd_aligned_buffer {
    static constexpr size_t simd_alignment = 32;  // AVX2 요구사항
    
    T* data_;
    size_t size_;
    
public:
    simd_aligned_buffer(size_t size) : size_(size) {
        data_ = static_cast<T*>(std::aligned_alloc(simd_alignment, size * sizeof(T)));
        if (!data_) {
            throw std::bad_alloc();
        }
    }
    
    ~simd_aligned_buffer() {
        std::free(data_);
    }
    
    simd_aligned_buffer(const simd_aligned_buffer&) = delete;
    simd_aligned_buffer& operator=(const simd_aligned_buffer&) = delete;
    
    aligned_accessor<T, simd_alignment> get_accessor() {
        return aligned_accessor<T, simd_alignment>(data_);
    }
    
    T* data() noexcept { 
        return std::assume_aligned<simd_alignment>(data_); 
    }
    
    size_t size() const noexcept { return size_; }
};

int main() {
    // 정렬된 메모리 할당
    constexpr size_t alignment = 64;  // 캐시 라인 크기
    constexpr size_t count = 16;
    
    aligned_allocator<int64_t, alignment> allocator;
    int64_t* aligned_mem = allocator.allocate(count);
    
    std::cout << "할당된 주소: " << static_cast<void*>(aligned_mem) << "\n";
    std::cout << "주소 % " << alignment << " = " 
              << (reinterpret_cast<uintptr_t>(aligned_mem) % alignment) << "\n";
    
    // aligned_accessor 사용
    try {
        aligned_accessor<int64_t, alignment> accessor(aligned_mem);
        
        for (size_t i = 0; i < count; ++i) {
            accessor[i] = static_cast<int64_t>(i * 100);
        }
        
        std::cout << "\n정렬된 데이터: ";
        for (size_t i = 0; i < count; ++i) {
            std::cout << accessor[i] << " ";
        }
        std::cout << "\n";
        
    } catch (const std::runtime_error& e) {
        std::cerr << "정렬 오류: " << e.what() << "\n";
    }
    
    allocator.deallocate(aligned_mem, count);
    
    // SIMD 버퍼 사용
    simd_aligned_buffer<float> simd_buf(32);
    auto simd_accessor = simd_buf.get_accessor();
    
    for (size_t i = 0; i < simd_buf.size(); ++i) {
        simd_accessor[i] = static_cast<float>(i) * 1.5f;
    }
    
    std::cout << "\nSIMD 버퍼 데이터: ";
    for (size_t i = 0; i < 8; ++i) {
        std::cout << simd_accessor[i] << " ";
    }
    std::cout << "...\n";
    
    return 0;
}
```

### 동적 정렬 검증과 자동 조정

런타임에 정렬을 검증하고 필요시 조정하는 메커니즘을 구현할 수 있다.

```cpp
#include <iostream>
#include <memory>
#include <vector>
#include <cstring>
#include <bit>

// 정렬 상태를 추적하는 스마트 포인터
template<typename T, size_t Alignment = alignof(T)>
class aligned_unique_ptr {
    T* ptr_;
    bool owns_memory_;
    
public:
    aligned_unique_ptr() : ptr_(nullptr), owns_memory_(false) {}
    
    explicit aligned_unique_ptr(size_t count) {
        ptr_ = static_cast<T*>(std::aligned_alloc(Alignment, count * sizeof(T)));
        if (!ptr_) {
            throw std::bad_alloc();
        }
        owns_memory_ = true;
        
        // 배치 new로 객체 생성
        for (size_t i = 0; i < count; ++i) {
            new (&ptr_[i]) T();
        }
    }
    
    ~aligned_unique_ptr() {
        if (owns_memory_ && ptr_) {
            std::free(ptr_);
        }
    }
    
    aligned_unique_ptr(const aligned_unique_ptr&) = delete;
    aligned_unique_ptr& operator=(const aligned_unique_ptr&) = delete;
    
    aligned_unique_ptr(aligned_unique_ptr&& other) noexcept 
        : ptr_(other.ptr_), owns_memory_(other.owns_memory_) {
        other.ptr_ = nullptr;
        other.owns_memory_ = false;
    }
    
    aligned_unique_ptr& operator=(aligned_unique_ptr&& other) noexcept {
        if (this != &other) {
            if (owns_memory_ && ptr_) {
                std::free(ptr_);
            }
            ptr_ = other.ptr_;
            owns_memory_ = other.owns_memory_;
            other.ptr_ = nullptr;
            other.owns_memory_ = false;
        }
        return *this;
    }
    
    T* get() const noexcept {
        return std::assume_aligned<Alignment>(ptr_);
    }
    
    T& operator[](size_t index) const noexcept {
        return get()[index];
    }
    
    bool is_aligned() const noexcept {
        return ptr_ && (reinterpret_cast<uintptr_t>(ptr_) % Alignment == 0);
    }
    
    size_t get_alignment() const noexcept {
        if (!ptr_) return 0;
        uintptr_t addr = reinterpret_cast<uintptr_t>(ptr_);
        return addr & -addr;  // 최하위 1비트의 위치 = 실제 정렬
    }
};

// 정렬 요구사항을 가진 데이터 구조
template<typename T, size_t CacheLineSize = 64>
class cache_aligned_array {
    struct alignas(CacheLineSize) aligned_element {
        T value;
        // 나머지는 패딩
    };
    
    std::vector<aligned_element> data_;
    
public:
    explicit cache_aligned_array(size_t size) : data_(size) {}
    
    T& operator[](size_t index) {
        return data_[index].value;
    }
    
    const T& operator[](size_t index) const {
        return data_[index].value;
    }
    
    size_t size() const { return data_.size(); }
    
    // 각 원소가 캐시 라인에 정렬되었는지 확인
    bool verify_alignment() const {
        for (size_t i = 0; i < data_.size(); ++i) {
            uintptr_t addr = reinterpret_cast<uintptr_t>(&data_[i]);
            if (addr % CacheLineSize != 0) {
                return false;
            }
        }
        return true;
    }
};

int main() {
    // aligned_unique_ptr 사용
    constexpr size_t alignment = 64;
    constexpr size_t count = 16;
    
    aligned_unique_ptr<double, alignment> aligned_data(count);
    
    std::cout << "정렬 확인: " << std::boolalpha << aligned_data.is_aligned() << "\n";
    std::cout << "실제 정렬: " << aligned_data.get_alignment() << " 바이트\n";
    
    // 데이터 초기화
    for (size_t i = 0; i < count; ++i) {
        aligned_data[i] = static_cast<double>(i) * 3.14;
    }
    
    std::cout << "\n정렬된 double 배열: ";
    for (size_t i = 0; i < 5; ++i) {
        std::cout << aligned_data[i] << " ";
    }
    std::cout << "...\n";
    
    // 캐시 라인 정렬 배열
    cache_aligned_array<int, 64> cache_array(8);
    
    std::cout << "\n캐시 라인 정렬 검증: " 
              << cache_array.verify_alignment() << "\n";
    
    for (size_t i = 0; i < cache_array.size(); ++i) {
        cache_array[i] = static_cast<int>(i * 10);
    }
    
    std::cout << "캐시 정렬 배열: ";
    for (size_t i = 0; i < cache_array.size(); ++i) {
        std::cout << cache_array[i] << " ";
    }
    std::cout << "\n";
    
    return 0;
}
```

## 14.2 `std::is_sufficiently_aligned` 타입 특성

### 문제: 정렬 요구사항의 불일치

타입의 정렬 요구사항과 실제 메모리 정렬이 일치하지 않으면 문제가 발생한다.

**안전하지 않은 코드:**

```cpp
#include <iostream>
#include <new>
#include <memory>

// 큰 정렬을 요구하는 타입
struct alignas(32) Vector256 {
    float data[8];
};

// 정렬 검사 없이 배치 new 사용
void unsafe_placement_new(void* buffer) {
    // buffer가 32바이트 정렬되어 있다고 가정
    Vector256* vec = new (buffer) Vector256();  // 위험!
    
    // 정렬되지 않은 경우 미정의 동작
    vec->data[0] = 1.0f;
}

// 정렬 검사 없는 타입 변환
template<typename T>
T* unsafe_reinterpret(void* ptr) {
    // ptr이 T의 정렬 요구사항을 만족한다고 가정
    return reinterpret_cast<T*>(ptr);
}

int main() {
    // 정렬되지 않은 버퍼
    char buffer[100];
    void* unaligned = &buffer[5];  // 의도적으로 정렬 깨뜨리기
    
    std::cout << "버퍼 주소: " << static_cast<void*>(buffer) << "\n";
    std::cout << "정렬되지 않은 주소: " << unaligned << "\n";
    
    // 위험한 배치 new
    unsafe_placement_new(unaligned);
    
    // 위험한 타입 변환
    Vector256* vec = unsafe_reinterpret<Vector256>(unaligned);
    
    return 0;
}
```

문제점은 다음과 같다.

- 배치 new가 정렬 요구사항을 검사하지 않는다
- reinterpret_cast가 정렬을 보장하지 않는다
- 컴파일 타임에 정렬 불일치를 감지할 수 없다

### 해결책: 정렬 검증 타입 특성

C++26은 정렬을 검증하는 타입 특성을 제공한다.

**안전한 코드:**

```cpp
#include <iostream>
#include <memory>
#include <type_traits>
#include <bit>
#include <stdexcept>

// C++26 is_sufficiently_aligned 구현 (표준 구현 예정)
template<typename T>
constexpr bool is_sufficiently_aligned(void* ptr) noexcept {
    return reinterpret_cast<uintptr_t>(ptr) % alignof(T) == 0;
}

template<typename T, size_t Alignment>
constexpr bool is_sufficiently_aligned_for(void* ptr) noexcept {
    static_assert(Alignment >= alignof(T), 
                  "요구 정렬이 타입의 기본 정렬보다 작다");
    return reinterpret_cast<uintptr_t>(ptr) % Alignment == 0;
}

// 안전한 배치 new
template<typename T, typename... Args>
T* safe_placement_new(void* buffer, Args&&... args) {
    if (!is_sufficiently_aligned<T>(buffer)) {
        throw std::runtime_error(
            "버퍼가 타입의 정렬 요구사항을 만족하지 않는다"
        );
    }
    return new (buffer) T(std::forward<Args>(args)...);
}

// 안전한 타입 변환
template<typename T>
T* safe_cast(void* ptr) {
    if (!ptr) {
        return nullptr;
    }
    
    if (!is_sufficiently_aligned<T>(ptr)) {
        throw std::runtime_error(
            "포인터가 타입의 정렬 요구사항을 만족하지 않는다"
        );
    }
    
    return static_cast<T*>(ptr);
}

// 컴파일 타임 정렬 검증
template<typename T, size_t BufferAlignment>
concept sufficiently_aligned = (BufferAlignment >= alignof(T));

template<typename T, size_t BufferAlignment>
    requires sufficiently_aligned<T, BufferAlignment>
class aligned_buffer {
    alignas(BufferAlignment) std::byte storage_[sizeof(T)];
    bool constructed_ = false;
    
public:
    template<typename... Args>
    T& construct(Args&&... args) {
        if (constructed_) {
            throw std::logic_error("객체가 이미 생성되었다");
        }
        
        T* ptr = new (storage_) T(std::forward<Args>(args)...);
        constructed_ = true;
        return *ptr;
    }
    
    void destroy() {
        if (constructed_) {
            std::launder(reinterpret_cast<T*>(storage_))->~T();
            constructed_ = false;
        }
    }
    
    T& get() {
        if (!constructed_) {
            throw std::logic_error("객체가 생성되지 않았다");
        }
        return *std::launder(reinterpret_cast<T*>(storage_));
    }
    
    ~aligned_buffer() {
        destroy();
    }
};

// 정렬 요구사항을 가진 타입들
struct alignas(16) SSEVector {
    float data[4];
};

struct alignas(32) AVXVector {
    float data[8];
};

struct alignas(64) CacheLineData {
    int64_t values[8];
};

int main() {
    // 안전한 배치 new 테스트
    alignas(32) char buffer1[sizeof(AVXVector)];
    
    try {
        AVXVector* vec1 = safe_placement_new<AVXVector>(buffer1);
        std::cout << "AVXVector 생성 성공\n";
        vec1->data[0] = 1.0f;
        vec1->~AVXVector();
    } catch (const std::runtime_error& e) {
        std::cerr << "오류: " << e.what() << "\n";
    }
    
    // 정렬되지 않은 버퍼 테스트
    char buffer2[100];
    void* unaligned = &buffer2[1];
    
    try {
        AVXVector* vec2 = safe_placement_new<AVXVector>(unaligned);
    } catch (const std::runtime_error& e) {
        std::cout << "예상된 오류: " << e.what() << "\n";
    }
    
    // 안전한 타입 변환
    alignas(64) char buffer3[sizeof(CacheLineData)];
    
    try {
        CacheLineData* data = safe_cast<CacheLineData>(buffer3);
        std::cout << "CacheLineData 변환 성공\n";
        data->values[0] = 42;
    } catch (const std::runtime_error& e) {
        std::cerr << "오류: " << e.what() << "\n";
    }
    
    // aligned_buffer 사용 (컴파일 타임 검증)
    aligned_buffer<SSEVector, 16> sse_buf;
    SSEVector& vec = sse_buf.construct();
    vec.data[0] = 3.14f;
    std::cout << "SSEVector 데이터: " << vec.data[0] << "\n";
    
    // 컴파일 오류 발생 (주석 처리)
    // aligned_buffer<AVXVector, 16> wrong_buf;  // 16 < 32 (AVXVector 정렬)
    
    return 0;
}
```

### 정렬 정보를 포함한 스마트 포인터

정렬 정보를 타입 시스템에 인코딩하는 방법을 구현할 수 있다.

```cpp
#include <iostream>
#include <memory>
#include <type_traits>

// 정렬 정보를 포함한 포인터 래퍼
template<typename T, size_t Alignment = alignof(T)>
class aligned_ptr {
    static_assert(std::has_single_bit(Alignment), 
                  "정렬은 2의 거듭제곱이어야 한다");
    static_assert(Alignment >= alignof(T),
                  "정렬은 타입의 기본 정렬 이상이어야 한다");
    
    T* ptr_;
    
    void verify_alignment() const {
        if (ptr_ && !is_sufficiently_aligned_for<T, Alignment>(ptr_)) {
            throw std::runtime_error("포인터 정렬 불일치");
        }
    }
    
public:
    static constexpr size_t alignment = Alignment;
    
    aligned_ptr() : ptr_(nullptr) {}
    
    explicit aligned_ptr(T* p) : ptr_(p) {
        verify_alignment();
    }
    
    // 더 큰 정렬에서 변환 가능
    template<size_t OtherAlignment>
        requires (OtherAlignment >= Alignment)
    aligned_ptr(aligned_ptr<T, OtherAlignment> other) 
        : ptr_(other.get()) {}
    
    T* get() const noexcept {
        return std::assume_aligned<Alignment>(ptr_);
    }
    
    T& operator*() const noexcept {
        return *get();
    }
    
    T* operator->() const noexcept {
        return get();
    }
    
    explicit operator bool() const noexcept {
        return ptr_ != nullptr;
    }
    
    // 정렬 정보 쿼리
    static constexpr size_t get_alignment() noexcept {
        return Alignment;
    }
    
    bool is_aligned() const noexcept {
        return !ptr_ || is_sufficiently_aligned_for<T, Alignment>(ptr_);
    }
};

// 정렬별 타입 별칭
template<typename T>
using cache_aligned_ptr = aligned_ptr<T, 64>;

template<typename T>
using simd_aligned_ptr = aligned_ptr<T, 32>;

// 정렬을 보장하는 팩토리 함수
template<typename T, size_t Alignment = alignof(T), typename... Args>
aligned_ptr<T, Alignment> make_aligned(Args&&... args) {
    void* mem = std::aligned_alloc(Alignment, sizeof(T));
    if (!mem) {
        throw std::bad_alloc();
    }
    
    T* ptr = new (mem) T(std::forward<Args>(args)...);
    return aligned_ptr<T, Alignment>(ptr);
}
```

## 14.3 false sharing 방지 기법

### 문제: False Sharing으로 인한 성능 저하

여러 스레드가 같은 캐시 라인의 다른 변수를 수정하면 캐시 일관성 프로토콜로 인해 성능이 크게 저하된다.

**안전하지 않은 코드:**

```cpp
#include <iostream>
#include <thread>
#include <vector>
#include <chrono>
#include <atomic>

// false sharing이 발생하는 구조
struct CountersFalseSharing {
    std::atomic<int64_t> counter1{0};  // 같은 캐시 라인
    std::atomic<int64_t> counter2{0};  // 같은 캐시 라인
    std::atomic<int64_t> counter3{0};  // 같은 캐시 라인
    std::atomic<int64_t> counter4{0};  // 같은 캐시 라인
};

void benchmark_false_sharing() {
    CountersFalseSharing counters;
    constexpr int64_t iterations = 10'000'000;
    
    auto start = std::chrono::high_resolution_clock::now();
    
    std::vector<std::thread> threads;
    
    // 각 스레드가 다른 카운터를 증가
    threads.emplace_back([&]() {
        for (int64_t i = 0; i < iterations; ++i) {
            counters.counter1.fetch_add(1, std::memory_order_relaxed);
        }
    });
    
    threads.emplace_back([&]() {
        for (int64_t i = 0; i < iterations; ++i) {
            counters.counter2.fetch_add(1, std::memory_order_relaxed);
        }
    });
    
    threads.emplace_back([&]() {
        for (int64_t i = 0; i < iterations; ++i) {
            counters.counter3.fetch_add(1, std::memory_order_relaxed);
        }
    });
    
    threads.emplace_back([&]() {
        for (int64_t i = 0; i < iterations; ++i) {
            counters.counter4.fetch_add(1, std::memory_order_relaxed);
        }
    });
    
    for (auto& t : threads) {
        t.join();
    }
    
    auto end = std::chrono::high_resolution_clock::now();
    auto duration = std::chrono::duration_cast<std::chrono::milliseconds>(end - start);
    
    std::cout << "False sharing 있음: " << duration.count() << "ms\n";
    std::cout << "결과: " << counters.counter1 << ", " << counters.counter2 
              << ", " << counters.counter3 << ", " << counters.counter4 << "\n";
}

int main() {
    benchmark_false_sharing();
    return 0;
}
```

문제점은 다음과 같다.

- 독립적인 변수들이 같은 캐시 라인을 공유한다
- 한 스레드의 쓰기가 다른 스레드의 캐시 라인을 무효화한다
- 캐시 일관성 트래픽으로 성능이 크게 저하된다

### 해결책: 캐시 라인 정렬과 패딩

각 변수를 캐시 라인 경계에 정렬하여 false sharing을 방지한다.

**안전한 코드:**

```cpp
    operator const T&() const noexcept { return value; }

public:

    auto sum() const {
        typename U::value_type total{};
        for (size_t i = 0; i < MaxThreads; ++i) {
            total += data_[i].get().load(std::memory_order_relaxed);
        }
        return total;
    }
```

### 동적 데이터 구조에서 false sharing 방지

동적으로 할당되는 데이터에서도 false sharing을 방지할 수 있다.

```cpp
#include <iostream>
#include <vector>
#include <memory>
#include <thread>
#include <atomic>
#include <new>

constexpr size_t cache_line_size = std::hardware_destructive_interference_size;

// 캐시 라인 정렬 할당자
template<typename T>
class cache_line_allocator {
public:
    using value_type = T;
    
    cache_line_allocator() = default;
    
    template<typename U>
    cache_line_allocator(const cache_line_allocator<U>&) noexcept {}
    
    T* allocate(size_t n) {
        // 각 객체를 캐시 라인에 정렬
        size_t aligned_size = ((sizeof(T) + cache_line_size - 1) / cache_line_size) 
                             * cache_line_size;
        void* ptr = std::aligned_alloc(cache_line_size, n * aligned_size);
        
        if (!ptr) {
            throw std::bad_alloc();
        }
        
        return static_cast<T*>(ptr);
    }
    
    void deallocate(T* ptr, size_t) noexcept {
        std::free(ptr);
    }
};

// 작업 큐 (false sharing 방지)
template<typename T>
class work_queue {
    struct alignas(cache_line_size) queue_node {
        std::atomic<T*> item{nullptr};
        std::atomic<queue_node*> next{nullptr};
    };
    
    alignas(cache_line_size) std::atomic<queue_node*> head_{nullptr};
    alignas(cache_line_size) std::atomic<queue_node*> tail_{nullptr};
    alignas(cache_line_size) std::atomic<size_t> size_{0};
    
public:
    work_queue() {
        auto dummy = new queue_node();
        head_.store(dummy, std::memory_order_relaxed);
        tail_.store(dummy, std::memory_order_relaxed);
    }
    
    ~work_queue() {
        T* item;
        while (try_dequeue(item)) {
            delete item;
        }
        
        auto head = head_.load(std::memory_order_relaxed);
        while (head) {
            auto next = head->next.load(std::memory_order_relaxed);
            delete head;
            head = next;
        }
    }
    
    void enqueue(T* item) {
        auto node = new queue_node();
        node->item.store(item, std::memory_order_relaxed);
        
        while (true) {
            auto tail = tail_.load(std::memory_order_acquire);
            auto next = tail->next.load(std::memory_order_acquire);
            
            if (tail == tail_.load(std::memory_order_acquire)) {
                if (next == nullptr) {
                    if (tail->next.compare_exchange_weak(next, node,
                        std::memory_order_release,
                        std::memory_order_acquire)) {
                        tail_.compare_exchange_weak(tail, node,
                            std::memory_order_release,
                            std::memory_order_acquire);
                        size_.fetch_add(1, std::memory_order_relaxed);
                        return;
                    }
                } else {
                    tail_.compare_exchange_weak(tail, next,
                        std::memory_order_release,
                        std::memory_order_acquire);
                }
            }
        }
    }
    
    bool try_dequeue(T*& item) {
        while (true) {
            auto head = head_.load(std::memory_order_acquire);
            auto tail = tail_.load(std::memory_order_acquire);
            auto next = head->next.load(std::memory_order_acquire);
            
            if (head == head_.load(std::memory_order_acquire)) {
                if (head == tail) {
                    if (next == nullptr) {
                        return false;
                    }
                    tail_.compare_exchange_weak(tail, next,
                        std::memory_order_release,
                        std::memory_order_acquire);
                } else {
                    item = next->item.load(std::memory_order_relaxed);
                    if (head_.compare_exchange_weak(head, next,
                        std::memory_order_release,
                        std::memory_order_acquire)) {
                        delete head;
                        size_.fetch_sub(1, std::memory_order_relaxed);
                        return true;
                    }
                }
            }
        }
    }
    
    size_t size() const {
        return size_.load(std::memory_order_relaxed);
    }
};

// 병렬 축약 (reduction) - false sharing 방지
template<typename T>
class parallel_reducer {
    struct alignas(cache_line_size) padded_value {
        T value{};
    };
    
    std::vector<padded_value> thread_values_;
    
public:
    explicit parallel_reducer(size_t num_threads) 
        : thread_values_(num_threads) {}
    
    void accumulate(size_t thread_id, const T& value) {
        thread_values_[thread_id].value += value;
    }
    
    T reduce() const {
        T result{};
        for (const auto& padded : thread_values_) {
            result += padded.value;
        }
        return result;
    }
    
    void reset() {
        for (auto& padded : thread_values_) {
            padded.value = T{};
        }
    }
};

int main() {
    std::cout << "=== Work Queue 테스트 ===\n";
    
    work_queue<int> queue;
    constexpr int num_producers = 4;
    constexpr int items_per_producer = 10000;
    
    // 생산자 스레드
    std::vector<std::thread> producers;
    for (int i = 0; i < num_producers; ++i) {
        producers.emplace_back([&queue, i]() {
            for (int j = 0; j < items_per_producer; ++j) {
                queue.enqueue(new int(i * items_per_producer + j));
            }
        });
    }
    
    // 소비자 스레드
    std::atomic<int> consumed{0};
    std::vector<std::thread> consumers;
    for (int i = 0; i < 2; ++i) {
        consumers.emplace_back([&queue, &consumed]() {
            int* item;
            while (consumed.load(std::memory_order_relaxed) 
                   < num_producers * items_per_producer) {
                if (queue.try_dequeue(item)) {
                    delete item;
                    consumed.fetch_add(1, std::memory_order_relaxed);
                }
            }
        });
    }
    
    for (auto& t : producers) {
        t.join();
    }
    for (auto& t : consumers) {
        t.join();
    }
    
    std::cout << "처리된 항목: " << consumed.load() << "\n";
    std::cout << "남은 항목: " << queue.size() << "\n\n";
    
    // 병렬 축약 테스트
    std::cout << "=== Parallel Reducer 테스트 ===\n";
    
    constexpr size_t num_threads = 4;
    constexpr int64_t iterations = 10'000'000;
    
    parallel_reducer<int64_t> reducer(num_threads);
    
    auto start = std::chrono::high_resolution_clock::now();
    
    std::vector<std::thread> threads;
    for (size_t tid = 0; tid < num_threads; ++tid) {
        threads.emplace_back([&reducer, tid, iterations]() {
            for (int64_t i = 0; i < iterations; ++i) {
                reducer.accumulate(tid, 1);
            }
        });
    }
    
    for (auto& t : threads) {
        t.join();
    }
    
    auto end = std::chrono::high_resolution_clock::now();
    auto duration = std::chrono::duration_cast<std::chrono::milliseconds>(end - start);
    
    std::cout << "소요 시간: " << duration.count() << "ms\n";
    std::cout << "총합: " << reducer.reduce() << "\n";
    
    return 0;
}
```

## 실전 예제: 고성능 메트릭 수집 시스템

메모리 정렬과 캐시 안전성의 모든 개념을 통합한 실전 예제를 살펴본다.

```cpp
#include <iostream>
#include <thread>
#include <vector>
#include <atomic>
#include <chrono>
#include <string>
#include <new>
#include <iomanip>

constexpr size_t cache_line_size = std::hardware_destructive_interference_size;

// 메트릭 타입
enum class MetricType {
    Counter,
    Gauge,
    Histogram
};

// 캐시 라인 정렬된 메트릭
template<MetricType Type>
struct alignas(cache_line_size) Metric {
    std::atomic<int64_t> value{0};
    std::atomic<int64_t> count{0};  // Histogram용
    
    void increment(int64_t delta = 1) {
        if constexpr (Type == MetricType::Counter) {
            value.fetch_add(delta, std::memory_order_relaxed);
        }
    }
    
    void set(int64_t new_value) {
        if constexpr (Type == MetricType::Gauge) {
            value.store(new_value, std::memory_order_relaxed);
        }
    }
    
    void observe(int64_t observation) {
        if constexpr (Type == MetricType::Histogram) {
            value.fetch_add(observation, std::memory_order_relaxed);
            count.fetch_add(1, std::memory_order_relaxed);
        }
    }
    
    int64_t get() const {
        return value.load(std::memory_order_relaxed);
    }
    
    double average() const {
        if constexpr (Type == MetricType::Histogram) {
            auto c = count.load(std::memory_order_relaxed);
            if (c == 0) return 0.0;
            return static_cast<double>(value.load(std::memory_order_relaxed)) / c;
        }
        return 0.0;
    }
};

// 스레드별 메트릭 컬렉터
class MetricsCollector {
    static constexpr size_t max_threads = 64;
    
    // 각 메트릭 타입별 스레드 로컬 저장소
    struct alignas(cache_line_size) ThreadMetrics {
        Metric<MetricType::Counter> requests;
        Metric<MetricType::Counter> errors;
        Metric<MetricType::Gauge> active_connections;
        Metric<MetricType::Histogram> latency;
    };
    
    ThreadMetrics metrics_[max_threads];
    std::atomic<size_t> thread_count_{0};
    
public:
    size_t register_thread() {
        return thread_count_.fetch_add(1, std::memory_order_relaxed);
    }
    
    void record_request(size_t thread_id) {
        metrics_[thread_id].requests.increment();
    }
    
    void record_error(size_t thread_id) {
        metrics_[thread_id].errors.increment();
    }
    
    void set_active_connections(size_t thread_id, int64_t count) {
        metrics_[thread_id].active_connections.set(count);
    }
    
    void record_latency(size_t thread_id, int64_t microseconds) {
        metrics_[thread_id].latency.observe(microseconds);
    }
    
    // 전체 메트릭 집계
    struct AggregatedMetrics {
        int64_t total_requests;
        int64_t total_errors;
        int64_t total_connections;
        double avg_latency;
    };
    
    AggregatedMetrics aggregate() const {
        AggregatedMetrics result{0, 0, 0, 0.0};
        
        int64_t total_latency = 0;
        int64_t total_observations = 0;
        
        size_t active_threads = thread_count_.load(std::memory_order_relaxed);
        
        for (size_t i = 0; i < active_threads; ++i) {
            result.total_requests += metrics_[i].requests.get();
            result.total_errors += metrics_[i].errors.get();
            result.total_connections += metrics_[i].active_connections.get();
            
            total_latency += metrics_[i].latency.get();
            total_observations += metrics_[i].latency.count.load(
                std::memory_order_relaxed);
        }
        
        if (total_observations > 0) {
            result.avg_latency = static_cast<double>(total_latency) 
                               / total_observations;
        }
        
        return result;
    }
    
    void print_metrics() const {
        auto metrics = aggregate();
        
        std::cout << std::fixed << std::setprecision(2);
        std::cout << "=== 메트릭 집계 ===\n";
        std::cout << "총 요청: " << metrics.total_requests << "\n";
        std::cout << "총 오류: " << metrics.total_errors << "\n";
        std::cout << "오류율: " 
                  << (metrics.total_requests > 0 
                      ? (100.0 * metrics.total_errors / metrics.total_requests)
                      : 0.0) 
                  << "%\n";
        std::cout << "활성 연결: " << metrics.total_connections << "\n";
        std::cout << "평균 지연: " << metrics.avg_latency << " μs\n";
    }
};

// 시뮬레이션 워커
void simulate_worker(MetricsCollector& collector, size_t worker_id, 
                    int duration_ms) {
    size_t thread_id = collector.register_thread();
    
    auto start = std::chrono::steady_clock::now();
    auto end = start + std::chrono::milliseconds(duration_ms);
    
    std::atomic<int64_t> active_conn{0};
    
    while (std::chrono::steady_clock::now() < end) {
        // 요청 시뮬레이션
        auto req_start = std::chrono::high_resolution_clock::now();
        
        collector.record_request(thread_id);
        active_conn.fetch_add(1, std::memory_order_relaxed);
        collector.set_active_connections(thread_id, 
            active_conn.load(std::memory_order_relaxed));
        
        // 가짜 작업
        std::this_thread::sleep_for(std::chrono::microseconds(10 + (worker_id % 50)));
        
        // 10% 확률로 오류
        if (worker_id % 10 == 0) {
            collector.record_error(thread_id);
        }
        
        auto req_end = std::chrono::high_resolution_clock::now();
        auto latency = std::chrono::duration_cast<std::chrono::microseconds>(
            req_end - req_start).count();
        
        collector.record_latency(thread_id, latency);
        
        active_conn.fetch_sub(1, std::memory_order_relaxed);
        collector.set_active_connections(thread_id, 
            active_conn.load(std::memory_order_relaxed));
    }
}

int main() {
    std::cout << "캐시 라인 크기: " << cache_line_size << " 바이트\n\n";
    
    MetricsCollector collector;
    
    constexpr size_t num_workers = 8;
    constexpr int duration_ms = 1000;
    
    std::cout << "워커 " << num_workers << "개로 " 
              << duration_ms << "ms 동안 시뮬레이션 실행\n\n";
    
    auto sim_start = std::chrono::high_resolution_clock::now();
    
    std::vector<std::thread> workers;
    for (size_t i = 0; i < num_workers; ++i) {
        workers.emplace_back(simulate_worker, 
            std::ref(collector), i, duration_ms);
    }
    
    for (auto& worker : workers) {
        worker.join();
    }
    
    auto sim_end = std::chrono::high_resolution_clock::now();
    auto sim_duration = std::chrono::duration_cast<std::chrono::milliseconds>(
        sim_end - sim_start);
    
    std::cout << "시뮬레이션 완료 (" << sim_duration.count() << "ms)\n\n";
    
    collector.print_metrics();
    
    return 0;
}
```

## 핵심 정리

이 장에서는 메모리 정렬과 캐시 안전성을 위한 세 가지 핵심 기법을 다루었다.

**`aligned_accessor`**는 메모리 정렬을 컴파일 타임과 런타임에 보장한다. SIMD 연산, 하드웨어 접근, 성능 최적화에 필수적이다.

**`is_sufficiently_aligned`** 타입 특성은 정렬 요구사항을 검증한다. 배치 new, 타입 변환, 커스텀 할당자에서 안전성을 보장한다.

**False sharing 방지**는 캐시 라인 정렬과 패딩으로 달성된다. 멀티스레드 환경에서 독립적인 변수들을 분리하여 캐시 일관성 오버헤드를 제거한다.
