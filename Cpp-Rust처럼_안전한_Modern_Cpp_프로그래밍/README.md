# Rust처럼 안전한 Modern C++ 프로그래밍

<pre>
- Rust처럼 안전한 C++ 프로그래밍을 하는 방법을 알려주는 책을 만들려고 해
- C++26 표준처럼 최신 표준 문법을 기반으로 하고 싶어
- C++26은 현재 최신 컴파일러에서 구현한 기능만 다루어줘
- 안전한 C++ 프로그래밍을 학습하기 위해 C++ 기능을 열거하고, 이 기능을 사용하지 않을 때와 사용할 때의 차이, 활용 예를 보여줘
- 목차를 만들어줘 
- 꼭 한글로 해줘
이 책은 다음과 같은 특징을 가진다:

1. **실용성 중심**: 모든 개념을 실제 코드 예제로 설명
2. **비교 학습**: 안전하지 않은 방법과 안전한 방법을 대비하여 설명
3. **점진적 학습**: 기초부터 고급까지 체계적으로 구성
4. **최신 표준**: C++26의 최신 기능들을 적극 활용
5. **실전 응용**: 각 장마다 실제 프로젝트에 적용할 수 있는 예제 제공

C++26 기반의 안전한 C++프로그래밍 책 목차를 만들었다. 현재 컴파일러에서 구현되거나 구현 예정인 C++26 기능들을 중심으로 구성했다.

C++26의 주요 안전성 기능들은 다음과 같다:

**핵심 안전성 기능들:**
- **Contracts**: 사전조건, 사후조건, 불변식을 코드에서 명시적으로 검증
- **Pack Indexing**: Template 매개변수 팩을 안전하게 접근
- **Erroneous Behavior**: 초기화되지 않은 메모리 읽기를 컴파일 타임에 감지
- **Trivial Relocatability**: 객체 이동 최적화와 안전성 보장
- **Hazard Pointer & RCU**: Lock-free 프로그래밍을 위한 안전한 메모리 관리

**라이브러리 안전성 기능들:**
- **std::execution**: 타입 안전한 비동기 프로그래밍
- **std::inplace_vector**: 스택 기반 동적 배열
- **Hardened Mode**: 컨테이너의 런타임 경계 검사
- **Saturation Arithmetic**: 정수 오버플로우 방지

현재 GCC 14+와 Clang에서 실험적으로 구현되고 있는 기능들을 중심으로 실용적인 예제와 함께 구성했다. 각 장에서는 "안전하지 않은 방법 → 문제점 → 안전한 해결책"의 구조로 설명하여 학습 효과를 높였다.

목차의 특징:
- **7개 파트 20개 장**으로 체계적 구성
- **기초부터 고급까지** 점진적 학습 구조  
- **실전 프로젝트** 중심의 응용 예제
- **최신 컴파일러 지원 현황** 반영
- **성능과 안전성의 균형** 추구

각 장마다 구체적인 코드 예제와 "Before/After" 비교를 통해 안전한 C++ 프로그래밍 기법을 습득할 수 있도록 설계했다.
