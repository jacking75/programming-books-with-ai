# 4장: Pack Indexing과 Template 안전성

## 4.1 기존 Template 메타프로그래밍의 문제점
C++에서 가변 인자 템플릿(variadic template)은 강력한 기능이지만, 특정 인덱스의 타입이나 값에 접근하는 것은 전통적으로 복잡하고 오류가 발생하기 쉬운 작업이었다. 기존 방식은 재귀적 템플릿 인스턴스화를 통해 구현되었고, 이는 코드를 읽기 어렵게 만들며 컴파일 시간도 증가시켰다.

### 전통적인 방식의 문제점

```cpp
// C++20 이전: 재귀 템플릿으로 N번째 타입 추출
template<size_t N, typename T, typename... Ts>
struct nth_type_impl {
    using type = typename nth_type_impl<N - 1, Ts...>::type;
};

template<typename T, typename... Ts>
struct nth_type_impl<0, T, Ts...> {
    using type = T;
};

template<size_t N, typename... Ts>
using nth_type = typename nth_type_impl<N, Ts...>::type;

// 사용 예시
template<typename... Args>
void process(Args... args) {
    // 두 번째 인자의 타입을 얻기 위해 복잡한 메타프로그래밍 필요
    using SecondType = nth_type<1, Args...>;
    // ...
}
```

이 코드는 여러 문제점을 가지고 있다:

1. **가독성 저하**: 간단한 인덱싱 작업을 위해 여러 개의 템플릿 특수화가 필요하다
2. **컴파일 시간 증가**: 재귀적 템플릿 인스턴스화는 컴파일러에 부담을 준다
3. **오류 메시지 복잡성**: 잘못된 인덱스 사용 시 이해하기 어려운 오류 메시지가 출력된다
4. **런타임 값 접근 어려움**: 특정 인덱스의 값을 얻기 위해서는 추가적인 헬퍼 함수가 필요하다

### 실제 문제 사례

```cpp
// 안전하지 않은 예시: 경계 검사 없음
template<typename... Args>
void unsafe_print_third(Args... args) {
    // 세 번째 인자가 없으면 컴파일 에러
    using ThirdType = nth_type<2, Args...>;  // 범위 초과 시 복잡한 오류
    // 값 접근은 더욱 복잡...
}

// 호출
unsafe_print_third(1, 2);  // 컴파일 에러: 긴 에러 메시지 출력
```

컴파일러는 다음과 같은 길고 이해하기 어려운 에러 메시지를 출력한다:

```
error: implicit instantiation of undefined template 
'nth_type_impl<2, int, int>'
note: template is declared here
note: in instantiation of template class 
'nth_type_impl<1, int>' requested here
...
```

## 4.2 Pack Indexing으로 안전한 Template 작성
C++26에서는 pack indexing 기능이 도입되어 이러한 문제들을 근본적으로 해결한다. 이 기능을 통해 파라미터 팩의 요소에 직접적이고 안전하게 접근할 수 있다.

### Pack Indexing 기본 문법

```cpp
// C++26: 직관적인 pack indexing
template<typename... Args>
void modern_process(Args... args) {
    // 타입 인덱싱
    using FirstType = Args...[0];
    using SecondType = Args...[1];
    
    // 값 인덱싱
    auto first_value = args...[0];
    auto second_value = args...[1];
    
    std::cout << "첫 번째 값: " << first_value << '\n';
    std::cout << "두 번째 값: " << second_value << '\n';
}

// 사용
modern_process(42, 3.14, "hello");  // 정상 작동
```

### 컴파일 타임 인덱스 검증
C++26의 pack indexing은 컴파일 타임에 인덱스의 유효성을 검증한다. 범위를 벗어난 인덱스는 명확한 에러 메시지와 함께 거부된다.

```cpp
// 안전한 예시: 컴파일 타임 경계 검사
template<typename... Args>
void safe_print_with_check(Args... args) {
    // constexpr if로 안전하게 검증
    if constexpr (sizeof...(Args) >= 3) {
        auto third = args...[2];
        std::cout << "세 번째 값: " << third << '\n';
    } else {
        std::cout << "인자가 3개 미만입니다\n";
    }
}

// 호출
safe_print_with_check(1, 2);           // "인자가 3개 미만입니다" 출력
safe_print_with_check(1, 2, 3, 4);     // "세 번째 값: 3" 출력
```

### 안전하지 않은 방법과 안전한 방법의 비교

```cpp
// === 안전하지 않은 방법 (C++20) ===
template<typename... Args>
void unsafe_tuple_like_access(Args... args) {
    // std::tuple로 변환 필요 - 추가 오버헤드
    auto t = std::make_tuple(args...);
    auto second = std::get<1>(t);  // 범위 검사는 되지만 복사 발생
    
    // 또는 fold expression으로 복잡하게 구현
    size_t index = 0;
    auto result = ((index++ == 1 ? args : 0) + ...);  // 타입 안전하지 않음
}

// === 안전한 방법 (C++26) ===
template<typename... Args>
void safe_direct_access(Args... args) {
    // 직접 접근, 복사 없음, 타입 안전
    if constexpr (sizeof...(Args) > 1) {
        auto& second = args...[1];  // 참조로 접근 가능
        second = modify(second);     // 원본 수정 가능
    }
}
```

```cpp
template<typename... Args>
void safe_print(const char* format, Args... args) {
    size_t arg_count = sizeof...(Args);
    size_t format_specifiers = count_format_specifiers(format);
    
    static_assert(sizeof...(Args) > 0, "최소 하나의 인자가 필요하다");
```

```cpp
// 안전한 인덱스 접근 래퍼
template<size_t Index, typename... Args>

constexpr auto& get_pack_element(Args&... args) {
    return args...[Index];

// 사용 예시
template<typename... Args>

    constexpr size_t count = sizeof...(Args);
```

## 4.3 Structured Binding 확장
C++26은 structured binding을 확장하여 pack 도입과 조건문에서의 사용을 지원한다. 이를 통해 더욱 표현력 있고 안전한 코드 작성이 가능하다.

### Pack 도입 Structured Binding

```cpp
// 기본 예시: 첫 요소와 나머지 분리
template<typename... Args>
void process_first_and_rest(Args... args) {
    // Pack을 structured binding으로 분해
    auto [first, ...rest] = std::make_tuple(args...);
    
    std::cout << "첫 번째: " << first << '\n';
    std::cout << "나머지 개수: " << sizeof...(rest) << '\n';
    
    // 나머지 요소들 처리
    (..., (std::cout << rest << " "));
    std::cout << '\n';
}

// 사용
process_first_and_rest(1, 2, 3, 4, 5);
// 출력:
// 첫 번째: 1
// 나머지 개수: 4
// 2 3 4 5
```

### 안전하지 않은 방법과 안전한 방법의 비교

```cpp
// === 안전하지 않은 방법 (C++20) ===
template<typename Container>
void unsafe_split_container(const Container& c) {
    // 반복자를 사용한 수동 분리
    if (!c.empty()) {
        auto first = *c.begin();
        
        // 나머지 요소를 얻기 위해 복사 필요
        Container rest;
        std::copy(std::next(c.begin()), c.end(), 
                  std::back_inserter(rest));
        
        // 크기 검사 누락 시 문제 발생 가능
        process_first(first);
        process_rest(rest);  // 불필요한 복사
    }
}

// === 안전한 방법 (C++26) ===
template<typename... Args>
void safe_split_args(Args... args) {
    if constexpr (sizeof...(Args) > 0) {
        // 구조적 바인딩으로 명확하게 분리
        auto [...all] = std::forward_as_tuple(args...);
        auto [first, ...rest] = std::forward_as_tuple(args...);
        
        // 복사 없이 참조로 작업
        process_first(first);
        (..., process_item(rest));  // fold expression으로 간결하게
    } else {
        std::cout << "인자가 없다\n";
    }
}
```

### 조건문에서의 Structured Binding
C++26은 if 문과 switch 문에서 structured binding을 직접 사용할 수 있도록 확장했다.

```cpp
// if 문에서 structured binding
struct Result {
    bool success;
    int value;
    std::string message;
};

Result compute() {
    // 계산 수행
    return {true, 42, "성공"};
}

void handle_result() {
    // 조건문 내에서 직접 분해
    if (auto [success, value, msg] = compute(); success) {
        std::cout << "성공: " << value << " - " << msg << '\n';
    } else {
        std::cout << "실패: " << msg << '\n';
    }
}

// 안전한 옵셔널 값 처리
std::optional<std::pair<int, std::string>> fetch_data() {
    return std::make_pair(100, "데이터");
}

void process_optional() {
    if (auto opt = fetch_data(); opt) {
        // opt가 값을 가질 때만 분해
        auto [id, name] = *opt;
        std::cout << "ID: " << id << ", 이름: " << name << '\n';
    }
}
```

### 실용적인 활용 예제

```cpp
// 예제 1: 안전한 함수 오버로딩 대체
class DataProcessor {
public:
    // 하나의 템플릿으로 모든 경우 처리
    template<typename... Args>
    void process(Args... args) {
        if constexpr (sizeof...(Args) == 0) {
            // 인자 없음
            process_empty();
        } else if constexpr (sizeof...(Args) == 1) {
            auto [single] = std::forward_as_tuple(args...);
            process_single(std::forward<decltype(single)>(single));
        } else {
            auto [first, ...rest] = std::forward_as_tuple(args...);
            process_first(std::forward<decltype(first)>(first));
            (..., process_item(std::forward<decltype(rest)>(rest)));
        }
    }
    
private:
    void process_empty() {
        std::cout << "빈 데이터\n";
    }
    
    template<typename T>
    void process_single(T&& item) {
        std::cout << "단일 항목: " << item << '\n';
    }
    
    template<typename T>
    void process_first(T&& item) {
        std::cout << "첫 항목: " << item << '\n';
    }
    
    template<typename T>
    void process_item(T&& item) {
        std::cout << "추가 항목: " << item << '\n';
    }
};

// 사용
DataProcessor dp;
dp.process();                    // 빈 데이터
dp.process(42);                  // 단일 항목: 42
dp.process(1, 2, 3);             // 첫 항목: 1, 추가 항목: 2, 추가 항목: 3

// 예제 2: 타입 안전한 이벤트 핸들러
template<typename... EventArgs>
class EventHandler {
    using CallbackType = std::function<void(EventArgs...)>;
    std::vector<CallbackType> callbacks;
    
public:
    void subscribe(CallbackType callback) {
        callbacks.push_back(std::move(callback));
    }
    
    void trigger(EventArgs... args) {
        // 모든 콜백에 안전하게 전달
        for (auto& callback : callbacks) {
            // pack indexing으로 타입 검증
            if constexpr (sizeof...(EventArgs) > 0) {
                callback(args...);
            } else {
                callback();
            }
        }
    }
    
    // 조건부 트리거
    template<typename Predicate>
    void trigger_if(Predicate pred, EventArgs... args) {
        if constexpr (sizeof...(EventArgs) > 0) {
            auto [first, ...rest] = std::forward_as_tuple(args...);
            
            // 첫 번째 인자로 조건 검사
            if (pred(first)) {
                trigger(args...);
            }
        }
    }
};

// 사용
EventHandler<int, std::string> handler;

handler.subscribe([](int code, const std::string& msg) {
    std::cout << "이벤트: " << code << " - " << msg << '\n';
});

handler.trigger(200, "OK");
handler.trigger_if([](int code) { return code >= 200; }, 
                   404, "Not Found");

// 예제 3: 안전한 튜플 연산
template<typename... Ts>
class SafeTuple {
    std::tuple<Ts...> data;
    
public:
    SafeTuple(Ts... values) : data(std::move(values)...) {}
    
    // 안전한 인덱스 접근
    template<size_t I>
        requires (I < sizeof...(Ts))
    auto& get() {
        return std::get<I>(data);
    }
    
    // pack indexing으로 부분 튜플 추출
    template<size_t... Is>
    auto extract() {
        return std::make_tuple(std::get<Is>(data)...);
    }
    
    // 타입 기반 검색
    template<typename T>
    auto find_first() -> std::optional<T> {
        std::optional<T> result;
        
        [&]<size_t... Is>(std::index_sequence<Is...>) {
            (..., [&] {
                using CurrentType = Ts...[Is];
                if constexpr (std::is_same_v<CurrentType, T>) {
                    if (!result) {
                        result = std::get<Is>(data);
                    }
                }
            }());
        }(std::make_index_sequence<sizeof...(Ts)>{});
        
        return result;
    }
    
    // 모든 요소 출력
    void print() const {
        std::cout << "튜플: (";
        [&]<size_t... Is>(std::index_sequence<Is...>) {
            (..., (std::cout << (Is > 0 ? ", " : "") 
                            << std::get<Is>(data)));
        }(std::make_index_sequence<sizeof...(Ts)>{});
        std::cout << ")\n";
    }
};

// 사용
SafeTuple<int, double, std::string> tuple(42, 3.14, "hello");
tuple.print();  // 튜플: (42, 3.14, hello)

auto value = tuple.get<1>();  // 3.14
auto sub = tuple.extract<0, 2>();  // std::tuple<int, std::string>

if (auto found = tuple.find_first<std::string>()) {
    std::cout << "문자열 발견: " << *found << '\n';
}
```

### 실전 응용: 로깅 시스템

```cpp
// 타입 안전하고 유연한 로깅 시스템
enum class LogLevel { Debug, Info, Warning, Error };

template<typename... Args>
class Logger {
public:
    void log(LogLevel level, Args... args) {
        print_timestamp();
        print_level(level);
        
        if constexpr (sizeof...(Args) == 0) {
            std::cout << "빈 메시지\n";
        } else if constexpr (sizeof...(Args) == 1) {
            auto [msg] = std::forward_as_tuple(args...);
            std::cout << msg << '\n';
        } else {
            // 첫 번째는 포맷 문자열, 나머지는 인자
            auto [format, ...params] = std::forward_as_tuple(args...);
            format_and_print(format, params...);
        }
    }
    
    // 조건부 로깅
    void log_if(bool condition, LogLevel level, Args... args) {
        if (condition) {
            log(level, args...);
        }
    }
    
    // 디버그 빌드에서만 로깅
    void debug_log(Args... args) {
        #ifdef DEBUG
        log(LogLevel::Debug, args...);
        #endif
    }
    
private:
    void print_timestamp() {
        auto now = std::chrono::system_clock::now();
        auto time = std::chrono::system_clock::to_time_t(now);
        std::cout << "[" << std::put_time(std::localtime(&time), "%T") << "] ";
    }
    
    void print_level(LogLevel level) {
        switch (level) {
            case LogLevel::Debug:   std::cout << "[DEBUG] "; break;
            case LogLevel::Info:    std::cout << "[INFO] "; break;
            case LogLevel::Warning: std::cout << "[WARN] "; break;
            case LogLevel::Error:   std::cout << "[ERROR] "; break;
        }
    }
    
    template<typename Format, typename... Params>
    void format_and_print(Format&& format, Params&&... params) {
        // 간단한 포맷팅 (실제로는 std::format 사용 권장)
        std::cout << format;
        (..., (std::cout << " " << params));
        std::cout << '\n';
    }
};

// 사용
Logger<std::string, int, double> logger;
```
