- 실전 WinRT: 네트워크와 멀티스레드 프로그래밍
- 모던 Windows 멀티스레딩: 게임 서버 개발자를 위한 고성능 동시성 프로그래밍
- 게임 서버 개발: Socket 방식

	- SuperSocketLite를 이용한 C# 게임 서버 프로그래밍
- FPS 게임 매칭 시스템 만들기

- 게임 서버 개발: API 서버 방식
