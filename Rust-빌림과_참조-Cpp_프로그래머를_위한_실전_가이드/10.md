# 10장. 실전 프로젝트: GUI 애플리케이션
GUI 애플리케이션은 빌림과 참조 문제가 집약적으로 나타나는 대표적인 사례다. 위젯 트리, 이벤트 처리, 상태 관리가 모두 복잡하게 얽혀있어 C++ 방식으로 접근하면 빌림 검사기와 끊임없이 충돌한다. 이 장에서는 간단한 GUI 프레임워크를 처음부터 구축하면서 실전에서 마주치는 문제들을 해결해본다.

## 10.1 위젯 트리 구조 설계
GUI의 핵심은 위젯 트리다. 버튼, 패널, 윈도우 등이 부모-자식 관계로 구성되며, 각 위젯은 서로를 참조해야 한다.

### 첫 번째 시도: 직접 참조 (실패)
C++에서는 부모가 자식을 포인터로 소유하고, 자식이 부모를 포인터로 역참조하는 구조를 흔히 사용한다.

```rust
// 컴파일되지 않는 코드
struct Widget {
    id: String,
    bounds: Rect,
    parent: Option<&Widget>,      // 라이프타임 문제
    children: Vec<Box<Widget>>,
}

#[derive(Debug, Clone, Copy)]
struct Rect {
    x: i32,
    y: i32,
    width: u32,
    height: u32,
}
```

이 접근법은 라이프타임 문제로 실패한다. `Widget`이 다른 `Widget`에 대한 참조를 가지려면 라이프타임을 명시해야 하는데, 이는 실용적이지 않다.

### 두 번째 시도: Rc와 RefCell (제한적 성공)

```rust
use std::rc::{Rc, Weak};
use std::cell::RefCell;

type WidgetRef = Rc<RefCell<Widget>>;
type WeakWidgetRef = Weak<RefCell<Widget>>;

#[derive(Debug, Clone, Copy)]
struct Rect {
    x: i32,
    y: i32,
    width: u32,
    height: u32,
}

struct Widget {
    id: String,
    bounds: Rect,
    parent: Option<WeakWidgetRef>,
    children: Vec<WidgetRef>,
}

impl Widget {
    fn new(id: String, bounds: Rect) -> WidgetRef {
        Rc::new(RefCell::new(Widget {
            id,
            bounds,
            parent: None,
            children: Vec::new(),
        }))
    }
    
    fn add_child(parent: &WidgetRef, child: WidgetRef) {
        child.borrow_mut().parent = Some(Rc::downgrade(parent));
        parent.borrow_mut().children.push(child);
    }
    
    fn get_absolute_position(&self) -> (i32, i32) {
        let mut x = self.bounds.x;
        let mut y = self.bounds.y;
        
        if let Some(parent_weak) = &self.parent {
            if let Some(parent_rc) = parent_weak.upgrade() {
                let parent = parent_rc.borrow();
                let (px, py) = parent.get_absolute_position();
                x += px;
                y += py;
            }
        }
        
        (x, y)
    }
}

fn main() {
    let window = Widget::new("window".to_string(), Rect {
        x: 0, y: 0, width: 800, height: 600,
    });
    
    let panel = Widget::new("panel".to_string(), Rect {
        x: 10, y: 10, width: 300, height: 200,
    });
    
    let button = Widget::new("button".to_string(), Rect {
        x: 20, y: 20, width: 100, height: 30,
    });
    
    Widget::add_child(&window, Rc::clone(&panel));
    Widget::add_child(&panel, Rc::clone(&button));
    
    let button_pos = button.borrow().get_absolute_position();
    println!("Button absolute position: {:?}", button_pos);
}
```

이 방식은 작동하지만 `Rc<RefCell<T>>`를 남발하면 런타임 오버헤드가 크고, 빌림 규칙 위반으로 패닉이 발생할 위험이 있다.

### 세 번째 시도: 인덱스 기반 (권장)
9장에서 배운 인덱스 패턴을 적용한다.

```rust
use std::collections::HashMap;

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
struct WidgetId(usize);

#[derive(Debug, Clone, Copy)]
struct Rect {
    x: i32,
    y: i32,
    width: u32,
    height: u32,
}

#[derive(Debug, Clone, Copy, PartialEq)]
enum Alignment {
    Start,
    Center,
    End,
}

#[derive(Debug)]
enum WidgetKind {
    Window { title: String },
    Panel { padding: u32 },
    Button { label: String, enabled: bool },
    Label { text: String },
}

struct Widget {
    id: WidgetId,
    kind: WidgetKind,
    bounds: Rect,
    parent: Option<WidgetId>,
    children: Vec<WidgetId>,
    visible: bool,
}

struct WidgetTree {
    widgets: HashMap<WidgetId, Widget>,
    next_id: usize,
    root: Option<WidgetId>,
}

impl WidgetTree {
    fn new() -> Self {
        WidgetTree {
            widgets: HashMap::new(),
            next_id: 0,
            root: None,
        }
    }
    
    fn create_widget(&mut self, kind: WidgetKind, bounds: Rect) -> WidgetId {
        let id = WidgetId(self.next_id);
        self.next_id += 1;
        
        self.widgets.insert(id, Widget {
            id,
            kind,
            bounds,
            parent: None,
            children: Vec::new(),
            visible: true,
        });
        
        if self.root.is_none() {
            self.root = Some(id);
        }
        
        id
    }
    
    fn add_child(&mut self, parent_id: WidgetId, child_id: WidgetId) {
        if let Some(child) = self.widgets.get_mut(&child_id) {
            child.parent = Some(parent_id);
        }
        
        if let Some(parent) = self.widgets.get_mut(&parent_id) {
            parent.children.push(child_id);
        }
    }
    
    fn get_absolute_bounds(&self, id: WidgetId) -> Option<Rect> {
        let widget = self.widgets.get(&id)?;
        let mut bounds = widget.bounds;
        
        let mut current = widget.parent;
        while let Some(parent_id) = current {
            if let Some(parent) = self.widgets.get(&parent_id) {
                bounds.x += parent.bounds.x;
                bounds.y += parent.bounds.y;
                current = parent.parent;
            } else {
                break;
            }
        }
        
        Some(bounds)
    }
    
    fn set_bounds(&mut self, id: WidgetId, bounds: Rect) {
        if let Some(widget) = self.widgets.get_mut(&id) {
            widget.bounds = bounds;
        }
    }
    
    fn is_point_inside(&self, id: WidgetId, x: i32, y: i32) -> bool {
        if let Some(bounds) = self.get_absolute_bounds(id) {
            x >= bounds.x && x < bounds.x + bounds.width as i32
                && y >= bounds.y && y < bounds.y + bounds.height as i32
        } else {
            false
        }
    }
    
    fn find_widget_at(&self, x: i32, y: i32) -> Option<WidgetId> {
        self.find_widget_at_recursive(self.root?, x, y)
    }
    
    fn find_widget_at_recursive(&self, id: WidgetId, x: i32, y: i32) -> Option<WidgetId> {
        let widget = self.widgets.get(&id)?;
        
        if !widget.visible {
            return None;
        }
        
        // 자식을 역순으로 검사 (나중에 그려진 것이 위에 있음)
        for &child_id in widget.children.iter().rev() {
            if let Some(found) = self.find_widget_at_recursive(child_id, x, y) {
                return Some(found);
            }
        }
        
        // 자식에서 찾지 못했으면 자신을 검사
        if self.is_point_inside(id, x, y) {
            Some(id)
        } else {
            None
        }
    }
    
    fn print_tree(&self, id: WidgetId, depth: usize) {
        if let Some(widget) = self.widgets.get(&id) {
            let indent = "  ".repeat(depth);
            let kind_str = match &widget.kind {
                WidgetKind::Window { title } => format!("Window({})", title),
                WidgetKind::Panel { .. } => "Panel".to_string(),
                WidgetKind::Button { label, .. } => format!("Button({})", label),
                WidgetKind::Label { text } => format!("Label({})", text),
            };
            
            println!("{}{:?}: {} at {:?}", indent, widget.id, kind_str, widget.bounds);
            
            for &child_id in &widget.children {
                self.print_tree(child_id, depth + 1);
            }
        }
    }
}

fn main() {
    let mut tree = WidgetTree::new();
    
    let window = tree.create_widget(
        WidgetKind::Window { title: "My App".to_string() },
        Rect { x: 0, y: 0, width: 800, height: 600 }
    );
    
    let panel = tree.create_widget(
        WidgetKind::Panel { padding: 10 },
        Rect { x: 50, y: 50, width: 300, height: 200 }
    );
    
    let button1 = tree.create_widget(
        WidgetKind::Button { label: "OK".to_string(), enabled: true },
        Rect { x: 20, y: 150, width: 80, height: 30 }
    );
    
    let button2 = tree.create_widget(
        WidgetKind::Button { label: "Cancel".to_string(), enabled: true },
        Rect { x: 110, y: 150, width: 80, height: 30 }
    );
    
    tree.add_child(window, panel);
    tree.add_child(panel, button1);
    tree.add_child(panel, button2);
    
    println!("Widget tree:");
    if let Some(root) = tree.root {
        tree.print_tree(root, 0);
    }
    
    println!("\nAbsolute bounds of button1: {:?}", tree.get_absolute_bounds(button1));
    
    let click_x = 90;
    let click_y = 220;
    if let Some(clicked) = tree.find_widget_at(click_x, click_y) {
        println!("\nWidget at ({}, {}): {:?}", click_x, click_y, clicked);
    }
}
```
  

## 10.2 이벤트 핸들러와 콜백
이벤트 처리는 GUI에서 가장 복잡한 부분이다. 위젯이 자신의 상태를 변경하면서 동시에 다른 위젯에 영향을 줘야 하는 경우가 많다.

### 문제 상황: 콜백에서 트리 수정하기

```rust
// 이벤트 타입
#[derive(Debug, Clone)]
enum Event {
    MouseDown { x: i32, y: i32 },
    MouseUp { x: i32, y: i32 },
    MouseMove { x: i32, y: i32 },
    KeyPress { key: char },
}

// 이벤트 핸들러 타입
type EventHandler = Box<dyn FnMut(&mut WidgetTree, WidgetId, &Event)>;

// Widget에 핸들러 추가
struct Widget {
    id: WidgetId,
    kind: WidgetKind,
    bounds: Rect,
    parent: Option<WidgetId>,
    children: Vec<WidgetId>,
    visible: bool,
    on_click: Option<EventHandler>,
}
```

문제는 `EventHandler`가 `&mut WidgetTree`를 받는데, 이 핸들러를 `Widget` 안에 저장하면 순환 참조가 발생한다는 것이다.

### 해결책 1: 이벤트 큐 패턴
이벤트를 즉시 처리하지 않고 큐에 모았다가 나중에 일괄 처리한다.

```rust
#[derive(Debug, Clone)]
enum Event {
    MouseDown { x: i32, y: i32, button: u8 },
    MouseUp { x: i32, y: i32, button: u8 },
    Click { widget_id: WidgetId },
    ValueChanged { widget_id: WidgetId, value: String },
}

#[derive(Debug, Clone)]
enum Command {
    SetVisible { widget_id: WidgetId, visible: bool },
    SetEnabled { widget_id: WidgetId, enabled: bool },
    SetText { widget_id: WidgetId, text: String },
    Close,
}

struct EventQueue {
    events: Vec<Event>,
    commands: Vec<Command>,
}

impl EventQueue {
    fn new() -> Self {
        EventQueue {
            events: Vec::new(),
            commands: Vec::new(),
        }
    }
    
    fn push_event(&mut self, event: Event) {
        self.events.push(event);
    }
    
    fn push_command(&mut self, command: Command) {
        self.commands.push(command);
    }
    
    fn take_events(&mut self) -> Vec<Event> {
        std::mem::take(&mut self.events)
    }
    
    fn take_commands(&mut self) -> Vec<Command> {
        std::mem::take(&mut self.commands)
    }
}

impl Widget {
    fn handle_event(&mut self, event: &Event, queue: &mut EventQueue) {
        match &mut self.kind {
            WidgetKind::Button { label, enabled } => {
                if *enabled {
                    if let Event::Click { widget_id } = event {
                        if *widget_id == self.id {
                            println!("Button '{}' clicked!", label);
                            // 명령을 큐에 추가
                            queue.push_command(Command::SetText {
                                widget_id: self.id,
                                text: format!("{} (clicked)", label),
                            });
                        }
                    }
                }
            }
            _ => {}
        }
    }
}

impl WidgetTree {
    fn process_event(&mut self, event: Event, queue: &mut EventQueue) {
        // 이벤트를 처리할 위젯 찾기
        let target = match &event {
            Event::MouseDown { x, y, .. } | Event::MouseUp { x, y, .. } => {
                self.find_widget_at(*x, *y)
            }
            Event::Click { widget_id } => Some(*widget_id),
            Event::ValueChanged { widget_id, .. } => Some(*widget_id),
        };
        
        if let Some(widget_id) = target {
            if let Some(widget) = self.widgets.get_mut(&widget_id) {
                widget.handle_event(&event, queue);
            }
        }
    }
    
    fn execute_command(&mut self, command: Command) {
        match command {
            Command::SetVisible { widget_id, visible } => {
                if let Some(widget) = self.widgets.get_mut(&widget_id) {
                    widget.visible = visible;
                }
            }
            Command::SetEnabled { widget_id, enabled } => {
                if let Some(widget) = self.widgets.get_mut(&widget_id) {
                    if let WidgetKind::Button { enabled: ref mut e, .. } = widget.kind {
                        *e = enabled;
                    }
                }
            }
            Command::SetText { widget_id, text } => {
                if let Some(widget) = self.widgets.get_mut(&widget_id) {
                    match &mut widget.kind {
                        WidgetKind::Button { label, .. } => *label = text,
                        WidgetKind::Label { text: t } => *t = text,
                        _ => {}
                    }
                }
            }
            Command::Close => {
                println!("Application closing...");
            }
        }
    }
}

fn main() {
    let mut tree = WidgetTree::new();
    let mut queue = EventQueue::new();
    
    let window = tree.create_widget(
        WidgetKind::Window { title: "Event Demo".to_string() },
        Rect { x: 0, y: 0, width: 400, height: 300 }
    );
    
    let button = tree.create_widget(
        WidgetKind::Button { 
            label: "Click Me".to_string(),
            enabled: true 
        },
        Rect { x: 150, y: 130, width: 100, height: 40 }
    );
    
    tree.add_child(window, button);
    
    // 마우스 클릭 시뮬레이션
    queue.push_event(Event::MouseDown { x: 200, y: 150, button: 0 });
    
    // 이벤트 처리
    for event in queue.take_events() {
        println!("Processing event: {:?}", event);
        
        // Click 이벤트 생성
        if let Event::MouseDown { x, y, .. } = event {
            if let Some(widget_id) = tree.find_widget_at(x, y) {
                let click_event = Event::Click { widget_id };
                tree.process_event(click_event, &mut queue);
            }
        }
    }
    
    // 명령 실행
    for command in queue.take_commands() {
        println!("Executing command: {:?}", command);
        tree.execute_command(command);
    }
}
```

### 해결책 2: 콜백 ID 패턴

콜백을 위젯에 직접 저장하는 대신 ID로 관리한다.

```rust
use std::collections::HashMap;

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
struct CallbackId(usize);

type Callback = Box<dyn FnMut(&mut WidgetTree, WidgetId) -> Vec<Command>>;

struct CallbackRegistry {
    callbacks: HashMap<CallbackId, Callback>,
    next_id: usize,
}

impl CallbackRegistry {
    fn new() -> Self {
        CallbackRegistry {
            callbacks: HashMap::new(),
            next_id: 0,
        }
    }
    
    fn register<F>(&mut self, callback: F) -> CallbackId
    where
        F: FnMut(&mut WidgetTree, WidgetId) -> Vec<Command> + 'static,
    {
        let id = CallbackId(self.next_id);
        self.next_id += 1;
        self.callbacks.insert(id, Box::new(callback));
        id
    }
    
    fn invoke(&mut self, id: CallbackId, tree: &mut WidgetTree, widget_id: WidgetId) -> Vec<Command> {
        if let Some(callback) = self.callbacks.get_mut(&id) {
            callback(tree, widget_id)
        } else {
            Vec::new()
        }
    }
}

struct WidgetWithCallbacks {
    id: WidgetId,
    kind: WidgetKind,
    bounds: Rect,
    parent: Option<WidgetId>,
    children: Vec<WidgetId>,
    visible: bool,
    on_click: Option<CallbackId>,
}

struct Application {
    tree: WidgetTree,
    callbacks: CallbackRegistry,
    queue: EventQueue,
}

impl Application {
    fn new() -> Self {
        Application {
            tree: WidgetTree::new(),
            callbacks: CallbackRegistry::new(),
            queue: EventQueue::new(),
        }
    }
    
    fn create_button_with_action<F>(
        &mut self,
        label: String,
        bounds: Rect,
        on_click: F,
    ) -> WidgetId
    where
        F: FnMut(&mut WidgetTree, WidgetId) -> Vec<Command> + 'static,
    {
        let callback_id = self.callbacks.register(on_click);
        let button_id = self.tree.create_widget(
            WidgetKind::Button { label, enabled: true },
            bounds
        );
        
        // 콜백 ID 저장 (실제 구현에서는 Widget 구조체 확장 필요)
        button_id
    }
    
    fn handle_click(&mut self, widget_id: WidgetId, callback_id: CallbackId) {
        let commands = self.callbacks.invoke(callback_id, &mut self.tree, widget_id);
        for command in commands {
            self.tree.execute_command(command);
        }
    }
}
```

## 10.3 상태 관리와 UI 업데이트
GUI 애플리케이션의 상태 관리는 복잡하다. 모델-뷰 분리 패턴을 적용해본다.

### MVC 패턴 구현

```rust
use std::collections::HashMap;

// 모델: 애플리케이션의 비즈니스 로직과 데이터
#[derive(Debug, Clone)]
struct TodoItem {
    id: usize,
    text: String,
    completed: bool,
}

struct TodoModel {
    items: Vec<TodoItem>,
    next_id: usize,
}

impl TodoModel {
    fn new() -> Self {
        TodoModel {
            items: Vec::new(),
            next_id: 0,
        }
    }
    
    fn add_item(&mut self, text: String) -> usize {
        let id = self.next_id;
        self.next_id += 1;
        
        self.items.push(TodoItem {
            id,
            text,
            completed: false,
        });
        
        id
    }
    
    fn toggle_item(&mut self, id: usize) {
        if let Some(item) = self.items.iter_mut().find(|i| i.id == id) {
            item.completed = !item.completed;
        }
    }
    
    fn remove_item(&mut self, id: usize) {
        self.items.retain(|item| item.id != id);
    }
    
    fn get_items(&self) -> &[TodoItem] {
        &self.items
    }
}

// 뷰: UI 표현
struct TodoView {
    tree: WidgetTree,
    list_panel: WidgetId,
    input_field: WidgetId,
    add_button: WidgetId,
    item_widgets: HashMap<usize, WidgetId>, // 모델 ID -> 위젯 ID
}

impl TodoView {
    fn new() -> Self {
        let mut tree = WidgetTree::new();
        
        let window = tree.create_widget(
            WidgetKind::Window { title: "Todo App".to_string() },
            Rect { x: 0, y: 0, width: 400, height: 500 }
        );
        
        let list_panel = tree.create_widget(
            WidgetKind::Panel { padding: 10 },
            Rect { x: 10, y: 10, width: 380, height: 400 }
        );
        
        let input_field = tree.create_widget(
            WidgetKind::Label { text: String::new() },
            Rect { x: 10, y: 420, width: 280, height: 30 }
        );
        
        let add_button = tree.create_widget(
            WidgetKind::Button { 
                label: "Add".to_string(),
                enabled: true 
            },
            Rect { x: 300, y: 420, width: 80, height: 30 }
        );
        
        tree.add_child(window, list_panel);
        tree.add_child(window, input_field);
        tree.add_child(window, add_button);
        
        TodoView {
            tree,
            list_panel,
            input_field,
            add_button,
            item_widgets: HashMap::new(),
        }
    }
    
    fn update_from_model(&mut self, model: &TodoModel) {
        // 기존 아이템 위젯 제거
        if let Some(panel) = self.tree.widgets.get_mut(&self.list_panel) {
            for &widget_id in panel.children.clone().iter() {
                self.tree.widgets.remove(&widget_id);
            }
            panel.children.clear();
        }
        
        self.item_widgets.clear();
        
        // 새 아이템 위젯 생성
        let items = model.get_items();
        for (i, item) in items.iter().enumerate() {
            let y = (i * 40) as i32;
            
            let checkbox = self.tree.create_widget(
                WidgetKind::Button {
                    label: if item.completed { "[X]" } else { "[ ]" }.to_string(),
                    enabled: true,
                },
                Rect { x: 10, y, width: 30, height: 30 }
            );
            
            let label = self.tree.create_widget(
                WidgetKind::Label { text: item.text.clone() },
                Rect { x: 50, y, width: 250, height: 30 }
            );
            
            let delete_btn = self.tree.create_widget(
                WidgetKind::Button {
                    label: "Delete".to_string(),
                    enabled: true,
                },
                Rect { x: 310, y, width: 60, height: 30 }
            );
```

```rust
        }
        
        self.listeners = listeners;

        self.notify_listeners();
        id
    }
    
    fn toggle_item(&mut self, id: usize) {
        self.inner.toggle_item(id);
        self.notify_listeners();
    }
    
    fn remove_item(&mut self, id: usize) {
        self.inner.remove_item(id);
        self.notify_listeners();
    }
}
```

```rust
// ============ 1. 기반 계층 ============

#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash)]
struct WidgetId(usize);

#[derive(Debug, Clone, Copy)]
struct Rect {
    x: i32,
    y: i32,
    width: u32,
    height: u32,
}

#[derive(Debug)]
enum WidgetKind {
    Window { title: String },
    Panel { padding: u32 },
    Button { label: String, enabled: bool },
    Label { text: String },
    TextInput { text: String, placeholder: String },
    Checkbox { label: String, checked: bool },
}

struct Widget {
    id: WidgetId,
    kind: WidgetKind,
    bounds: Rect,
    parent: Option<WidgetId>,
    children: Vec<WidgetId>,
    visible: bool,
}

struct WidgetTree {
    widgets: HashMap<WidgetId, Widget>,
    next_id: usize,
    root: Option<WidgetId>,
}

impl WidgetTree {
    fn new() -> Self {
        WidgetTree {
            widgets: HashMap::new(),
            next_id: 0,
            root: None,
        }
    }
    
    fn create_widget(&mut self, kind: WidgetKind, bounds: Rect) -> WidgetId {
        let id = WidgetId(self.next_id);
        self.next_id += 1;
        
        self.widgets.insert(id, Widget {
            id,
            kind,
            bounds,
            parent: None,
            children: Vec::new(),
            visible: true,
        });
        
        if self.root.is_none() {
            self.root = Some(id);
        }
        
        id
    }
    
    fn add_child(&mut self, parent_id: WidgetId, child_id: WidgetId) {
        if let Some(child) = self.widgets.get_mut(&child_id) {
            child.parent = Some(parent_id);
        }
        if let Some(parent) = self.widgets.get_mut(&parent_id) {
            parent.children.push(child_id);
        }
    }
    
    fn get_widget(&self, id: WidgetId) -> Option<&Widget> {
        self.widgets.get(&id)
    }
    
    fn get_widget_mut(&mut self, id: WidgetId) -> Option<&mut Widget> {
        self.widgets.get_mut(&id)
    }
}

// ============ 2. 이벤트 계층 ============

#[derive(Debug, Clone)]
enum UiEvent {
    Click { widget_id: WidgetId },
    TextChanged { widget_id: WidgetId, text: String },
    CheckboxToggled { widget_id: WidgetId, checked: bool },
}

#[derive(Debug, Clone)]
enum UiCommand {
    SetText { widget_id: WidgetId, text: String },
    SetEnabled { widget_id: WidgetId, enabled: bool },
    SetVisible { widget_id: WidgetId, visible: bool },
    SetChecked { widget_id: WidgetId, checked: bool },
}

struct EventBus {
    events: Vec<UiEvent>,
    commands: Vec<UiCommand>,
}

impl EventBus {
    fn new() -> Self {
        EventBus {
            events: Vec::new(),
            commands: Vec::new(),
        }
    }
    
    fn emit_event(&mut self, event: UiEvent) {
        self.events.push(event);
    }
    
    fn emit_command(&mut self, command: UiCommand) {
        self.commands.push(command);
    }
    
    fn take_events(&mut self) -> Vec<UiEvent> {
        std::mem::take(&mut self.events)
    }
    
    fn take_commands(&mut self) -> Vec<UiCommand> {
        std::mem::take(&mut self.commands)
    }
}

// ============ 3. 애플리케이션 계층 ============

struct FormData {
    name: String,
    email: String,
    subscribe: bool,
}

impl FormData {
    fn new() -> Self {
        FormData {
            name: String::new(),
            email: String::new(),
            subscribe: false,
        }
    }
    
    fn is_valid(&self) -> bool {
        !self.name.is_empty() && self.email.contains('@')
    }
}

struct FormApp {
    tree: WidgetTree,
    event_bus: EventBus,
    data: FormData,
    
    // 위젯 ID들
    name_input: WidgetId,
    email_input: WidgetId,
    subscribe_checkbox: WidgetId,
    submit_button: WidgetId,
    status_label: WidgetId,
}

impl FormApp {
    fn new() -> Self {
        let mut tree = WidgetTree::new();
        
        let window = tree.create_widget(
            WidgetKind::Window { title: "Registration Form".to_string() },
            Rect { x: 0, y: 0, width: 400, height: 300 }
        );
        
        let name_label = tree.create_widget(
            WidgetKind::Label { text: "Name:".to_string() },
            Rect { x: 20, y: 20, width: 100, height: 30 }
        );
        
        let name_input = tree.create_widget(
            WidgetKind::TextInput {
                text: String::new(),
                placeholder: "Enter your name".to_string(),
            },
            Rect { x: 130, y: 20, width: 250, height: 30 }
        );
        
        let email_label = tree.create_widget(
            WidgetKind::Label { text: "Email:".to_string() },
            Rect { x: 20, y: 60, width: 100, height: 30 }
        );
        
        let email_input = tree.create_widget(
            WidgetKind::TextInput {
                text: String::new(),
                placeholder: "Enter your email".to_string(),
            },
            Rect { x: 130, y: 60, width: 250, height: 30 }
        );
        
        let subscribe_checkbox = tree.create_widget(
            WidgetKind::Checkbox {
                label: "Subscribe to newsletter".to_string(),
                checked: false,
            },
            Rect { x: 20, y: 100, width: 250, height: 30 }
        );
        
        let submit_button = tree.create_widget(
            WidgetKind::Button {
                label: "Submit".to_string(),
                enabled: false,
            },
            Rect { x: 150, y: 150, width: 100, height: 40 }
        );
        
        let status_label = tree.create_widget(
            WidgetKind::Label { text: String::new() },
            Rect { x: 20, y: 210, width: 360, height: 30 }
        );
        
        tree.add_child(window, name_label);
        tree.add_child(window, name_input);
        tree.add_child(window, email_label);
        tree.add_child(window, email_input);
        tree.add_child(window, subscribe_checkbox);
        tree.add_child(window, submit_button);
        tree.add_child(window, status_label);
        
        FormApp {
            tree,
            event_bus: EventBus::new(),
            data: FormData::new(),
            name_input,
            email_input,
            subscribe_checkbox,
            submit_button,
            status_label,
        }
    }
    
    fn handle_event(&mut self, event: &UiEvent) {
        match event {
            UiEvent::TextChanged { widget_id, text } => {
                if *widget_id == self.name_input {
                    self.data.name = text.clone();
                } else if *widget_id == self.email_input {
                    self.data.email = text.clone();
                }
                self.update_submit_button();
            }
            
            UiEvent::CheckboxToggled { widget_id, checked } => {
                if *widget_id == self.subscribe_checkbox {
                    self.data.subscribe = *checked;
                }
            }
            
            UiEvent::Click { widget_id } => {
                if *widget_id == self.submit_button {
                    self.handle_submit();
                }
            }
        }
    }
    
    fn update_submit_button(&mut self) {
        let enabled = self.data.is_valid();
        self.event_bus.emit_command(UiCommand::SetEnabled {
            widget_id: self.submit_button,
            enabled,
        });
    }
    
    fn handle_submit(&mut self) {
        if self.data.is_valid() {
            let message = format!(
                "Registered: {} ({}{})",
                self.data.name,
                self.data.email,
                if self.data.subscribe { ", subscribed" } else { "" }
            );
            
            self.event_bus.emit_command(UiCommand::SetText {
                widget_id: self.status_label,
                text: message,
            });
            
            println!("Form submitted successfully!");
            println!("  Name: {}", self.data.name);
            println!("  Email: {}", self.data.email);
            println!("  Subscribe: {}", self.data.subscribe);
        }
    }
    
    fn execute_command(&mut self, command: &UiCommand) {
        match command {
            UiCommand::SetText { widget_id, text } => {
                if let Some(widget) = self.tree.get_widget_mut(*widget_id) {
                    match &mut widget.kind {
                        WidgetKind::Label { text: t } => *t = text.clone(),
                        WidgetKind::TextInput { text: t, .. } => *t = text.clone(),
                        _ => {}
                    }
                }
            }
            
            UiCommand::SetEnabled { widget_id, enabled } => {
                if let Some(widget) = self.tree.get_widget_mut(*widget_id) {
                    if let WidgetKind::Button { enabled: e, .. } = &mut widget.kind {
                        *e = *enabled;
                    }
                }
            }
            
            UiCommand::SetVisible { widget_id, visible } => {
                if let Some(widget) = self.tree.get_widget_mut(*widget_id) {
                    widget.visible = *visible;
                }
            }
            
            UiCommand::SetChecked { widget_id, checked } => {
                if let Some(widget) = self.tree.get_widget_mut(*widget_id) {
                    if let WidgetKind::Checkbox { checked: c, .. } = &mut widget.kind {
                        *c = *checked;
                    }
                }
            }
        }
    }
    
    fn run_frame(&mut self) {
        // 이벤트 처리
        for event in self.event_bus.take_events() {
            self.handle_event(&event);
        }
        
        // 명령 실행
        for command in self.event_bus.take_commands() {
            self.execute_command(&command);
        }
    }
    
    fn print_state(&self) {
        println!("\n=== Form State ===");
        println!("Name: '{}'", self.data.name);
        println!("Email: '{}'", self.data.email);
        println!("Subscribe: {}", self.data.subscribe);
        println!("Valid: {}", self.data.is_valid());
    }
}

fn main() {
    let mut app = FormApp::new();
    
    println!("=== Simulating User Interaction ===\n");
    
    // 사용자가 이름 입력
    app.event_bus.emit_event(UiEvent::TextChanged {
        widget_id: app.name_input,
        text: "John Doe".to_string(),
    });
    app.run_frame();
    app.print_state();
    
    // 사용자가 이메일 입력
    app.event_bus.emit_event(UiEvent::TextChanged {
        widget_id: app.email_input,
        text: "john@example.com".to_string(),
    });
    app.run_frame();
    app.print_state();
    
    // 사용자가 체크박스 토글
    app.event_bus.emit_event(UiEvent::CheckboxToggled {
        widget_id: app.subscribe_checkbox,
        checked: true,
    });
    app.run_frame();
    app.print_state();
    
    // 사용자가 제출 버튼 클릭
    app.event_bus.emit_event(UiEvent::Click {
        widget_id: app.submit_button,
    });
    app.run_frame();
    
    println!("\n=== Final Widget Tree ===");
    if let Some(root) = app.tree.root {
        app.tree.print_tree(root, 0);
    }
}
```

### 핵심 설계 원칙 정리
이 장에서 구현한 GUI 애플리케이션은 다음 원칙들을 따른다:

**1. 인덱스 기반 참조**: 위젯 ID를 통해 간접 참조하여 빌림 문제 회피

**2. 이벤트-명령 분리**: 이벤트는 무엇이 일어났는지를 표현하고, 명령은 무엇을 해야 하는지를 표현

**3. 레이어 분리**: 
- 기반 계층 (위젯 트리)
- 이벤트 계층 (이벤트 버스)
- 애플리케이션 계층 (비즈니스 로직)

**4. 명시적 업데이트 사이클**: `run_frame()`에서 이벤트 처리와 명령 실행을 분리

**5. 불변성 우선**: 가능한 한 데이터를 불변으로 유지하고, 변경이 필요할 때만 명령을 통해 수정

이러한 패턴들은 Rust의 빌림 검사기와 협력하면서도 복잡한 GUI 애플리케이션을 구축할 수 있게 해준다. C++의 자유로운 포인터 사용에 비하면 제약이 많아 보이지만, 컴파일 타임에 많은 버그를 잡아내고 런타임 안전성을 보장받을 수 있다는 큰 장점이 있다.
