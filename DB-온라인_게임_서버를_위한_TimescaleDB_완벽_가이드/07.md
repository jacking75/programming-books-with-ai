# Chapter 7: 게임 로그 수집 및 분석

게임 서버의 메트릭을 수집하는 것만으로는 충분하지 않다. CPU 사용률이나 메모리 사용량은 서버의 건강 상태를 알려주지만, 플레이어가 실제로 게임 안에서 무엇을 하고 있는지는 알려주지 않는다. 플레이어가 어떤 던전을 클리어했는가? 어떤 아이템을 구매했는가? 어디서 이탈했는가? 이런 질문에 답하려면 게임 내에서 발생하는 모든 중요한 이벤트를 로그로 기록해야 한다.

게임 로그는 단순한 기록이 아니다. 이는 게임 운영의 핵심 자산이다. 로그 데이터를 제대로 분석하면 플레이어가 어디서 즐거움을 느끼고 어디서 좌절하는지 알 수 있다. 어떤 콘텐츠가 인기 있고 어떤 기능이 사용되지 않는지 파악할 수 있다. 어뷰징 행위를 탐지하고, 버그를 추적하며, 게임 밸런스를 조정하는 모든 결정이 로그 데이터에 기반한다.

이번 챕터에서는 게임 내 다양한 이벤트 로그를 TimescaleDB에 저장하고 분석하는 완전한 시스템을 구축한다. 구조화된 로그 설계부터 JSON을 활용한 유연한 데이터 저장, 플레이어 행동 패턴 분석, 아이템 경제 추적, 그리고 어뷰징 탐지까지 실전에서 필요한 모든 것을 다룬다.

## 7.1 게임 로그의 종류와 중요성

온라인 게임에서 발생하는 로그는 크게 여러 카테고리로 나눌 수 있다. 각 카테고리마다 추적해야 하는 정보와 분석 목적이 다르므로, 먼저 어떤 종류의 로그가 있는지 이해하는 것이 중요하다.

**플레이어 세션 로그**는 플레이어의 접속과 종료를 기록한다. 로그인 시각, 로그아웃 시각, 접속 IP 주소, 디바이스 정보, 플레이 시간 등을 포함한다. 이 데이터로 동시 접속자 수(CCU), 일일 활성 사용자(DAU), 월간 활성 사용자(MAU) 같은 핵심 지표를 계산한다.

**게임 플레이 로그**는 플레이어가 게임 안에서 수행하는 모든 액션을 추적한다. 던전 입장, 몬스터 처치, 레벨 업, 스킬 사용, 퀘스트 완료 등이 포함된다. 이 로그를 분석하면 플레이어가 어떤 콘텐츠를 즐기는지, 어디서 막히는지, 어느 레벨에서 이탈하는지 파악할 수 있다.

**아이템 로그**는 게임 내 경제의 모든 흐름을 기록한다. 아이템 획득, 사용, 판매, 구매, 강화, 분해 등 아이템과 관련된 모든 이벤트를 추적한다. 이는 게임 경제의 인플레이션이나 디플레이션을 감지하고, 아이템 밸런스를 조정하는 데 필수적이다.

**재화 로그**는 게임 내 화폐(골드, 다이아몬드 등)의 흐름을 추적한다. 재화의 획득 경로, 소비 경로, 잔액 변화를 기록한다. 이 데이터는 게임 경제의 건강도를 측정하고, 불법 거래나 해킹을 탐지하는 데 사용된다.

**소셜 로그**는 플레이어 간 상호작용을 기록한다. 채팅, 친구 추가, 길드 가입, 파티 플레이, 거래 등이 포함된다. 이를 통해 커뮤니티의 활성도를 측정하고, 유해한 행위를 감지할 수 있다.

**결제 로그**는 실제 돈이 오가는 모든 거래를 기록한다. 구매한 상품, 결제 금액, 결제 수단, 결제 성공/실패 여부 등을 추적한다. 이는 매출 분석과 결제 시스템 안정성 모니터링에 사용된다.

각 로그 유형마다 중요도와 보관 기간이 다르다. 예를 들어 결제 로그는 법적 요구사항 때문에 최소 5년 이상 보관해야 하지만, 일반적인 플레이 로그는 3~6개월 정도만 보관해도 충분할 수 있다. TimescaleDB의 데이터 보관 정책을 활용하면 이런 차별화된 관리가 가능하다.

## 7.2 구조화된 로그 설계하기

로그를 효과적으로 분석하려면 일관된 구조로 데이터를 저장해야 한다. 게임 서버에서 발생하는 다양한 이벤트를 하나의 통합된 스키마로 저장하는 것이 관리와 분석 측면에서 유리하다.

먼저 게임 이벤트를 저장할 메인 테이블을 설계한다. pgAdmin을 열고 다음 SQL을 실행한다.

```sql
-- 게임 이벤트 로그 테이블
CREATE TABLE game_events (
    time TIMESTAMPTZ NOT NULL,
    event_id BIGSERIAL,
    player_id BIGINT NOT NULL,
    event_type VARCHAR(50) NOT NULL,
    event_category VARCHAR(30) NOT NULL,
    event_data JSONB,
    server_id VARCHAR(20),
    session_id VARCHAR(50),
    client_version VARCHAR(20),
    PRIMARY KEY (time, event_id)
);

-- Hypertable로 변환
SELECT create_hypertable('game_events', 'time');

-- 자주 사용하는 조회를 위한 인덱스
CREATE INDEX idx_game_events_player_time 
ON game_events (player_id, time DESC);

CREATE INDEX idx_game_events_type_time 
ON game_events (event_type, time DESC);

CREATE INDEX idx_game_events_category_time 
ON game_events (event_category, time DESC);

-- JSONB 필드 내 특정 키 검색을 위한 GIN 인덱스
CREATE INDEX idx_game_events_data 
ON game_events USING GIN (event_data);

-- 특정 아이템 관련 이벤트를 빠르게 찾기 위한 인덱스
CREATE INDEX idx_game_events_item_id 
ON game_events ((event_data->>'item_id')) 
WHERE event_category = 'item';
```

이 스키마의 핵심은 **고정 컬럼**과 **유연한 JSON 컬럼**의 조합이다. `time`, `player_id`, `event_type` 같은 필수 정보는 고정 컬럼으로 정의하여 빠른 인덱싱과 조회를 보장한다. 반면 이벤트마다 다를 수 있는 세부 정보는 `event_data` JSONB 컬럼에 저장하여 스키마 변경 없이 새로운 이벤트 타입을 추가할 수 있다.

`event_category`는 이벤트를 대분류로 구분한다. 예를 들어 `session`, `gameplay`, `item`, `currency`, `social`, `payment` 등이 될 수 있다. `event_type`은 구체적인 이벤트 종류를 나타낸다. 예를 들어 `item` 카테고리 안에 `item_acquired`, `item_used`, `item_sold` 같은 타입이 있을 수 있다.

C# 모델 클래스를 작성한다.

```csharp
using System;
using System.Collections.Generic;
using Newtonsoft.Json;

namespace GameServerLogging.Models
{
    /// <summary>
    /// 게임 이벤트의 카테고리
    /// </summary>
    public enum EventCategory
    {
        Session,      // 세션 관련 (로그인/로그아웃)
        Gameplay,     // 게임플레이 (던전, 전투, 퀘스트)
        Item,         // 아이템 관련
        Currency,     // 재화 관련
        Social,       // 소셜 (채팅, 거래, 길드)
        Payment,      // 결제
        Achievement,  // 업적
        System        // 시스템 (에러, 경고)
    }

    /// <summary>
    /// 게임 이벤트를 표현하는 모델
    /// </summary>
    public class GameEvent
    {
        /// <summary>
        /// 이벤트 발생 시각 (UTC)
        /// </summary>
        public DateTime Time { get; set; }

        /// <summary>
        /// 이벤트 고유 ID (자동 생성)
        /// </summary>
        public long EventId { get; set; }

        /// <summary>
        /// 플레이어 ID
        /// </summary>
        public long PlayerId { get; set; }

        /// <summary>
        /// 이벤트 타입 (예: "dungeon_clear", "item_acquired")
        /// </summary>
        public string EventType { get; set; }

        /// <summary>
        /// 이벤트 카테고리
        /// </summary>
        public EventCategory EventCategory { get; set; }

        /// <summary>
        /// 이벤트 세부 데이터 (JSON)
        /// </summary>
        public Dictionary<string, object> EventData { get; set; }

        /// <summary>
        /// 서버 ID
        /// </summary>
        public string ServerId { get; set; }

        /// <summary>
        /// 세션 ID
        /// </summary>
        public string SessionId { get; set; }

        /// <summary>
        /// 클라이언트 버전
        /// </summary>
        public string ClientVersion { get; set; }

        public GameEvent()
        {
            Time = DateTime.UtcNow;
            EventData = new Dictionary<string, object>();
        }

        /// <summary>
        /// 이벤트 데이터를 JSON 문자열로 변환
        /// </summary>
        public string GetEventDataJson()
        {
            return JsonConvert.SerializeObject(EventData);
        }
    }

    /// <summary>
    /// 특정 이벤트 타입을 위한 빌더 클래스들
    /// </summary>
    public static class GameEventBuilder
    {
        /// <summary>
        /// 로그인 이벤트 생성
        /// </summary>
        public static GameEvent CreateLoginEvent(
            long playerId, 
            string sessionId, 
            string ipAddress,
            string deviceType)
        {
            return new GameEvent
            {
                PlayerId = playerId,
                EventType = "player_login",
                EventCategory = EventCategory.Session,
                SessionId = sessionId,
                EventData = new Dictionary<string, object>
                {
                    { "ip_address", ipAddress },
                    { "device_type", deviceType },
                    { "login_method", "normal" }
                }
            };
        }

        /// <summary>
        /// 던전 클리어 이벤트 생성
        /// </summary>
        public static GameEvent CreateDungeonClearEvent(
            long playerId,
            string sessionId,
            int dungeonId,
            int clearTimeSeconds,
            int rewardGold,
            List<int> rewardItems)
        {
            return new GameEvent
            {
                PlayerId = playerId,
                EventType = "dungeon_clear",
                EventCategory = EventCategory.Gameplay,
                SessionId = sessionId,
                EventData = new Dictionary<string, object>
                {
                    { "dungeon_id", dungeonId },
                    { "clear_time_seconds", clearTimeSeconds },
                    { "reward_gold", rewardGold },
                    { "reward_items", rewardItems },
                    { "difficulty", "normal" }
                }
            };
        }

        /// <summary>
        /// 아이템 획득 이벤트 생성
        /// </summary>
        public static GameEvent CreateItemAcquiredEvent(
            long playerId,
            string sessionId,
            int itemId,
            int quantity,
            string source)
        {
            return new GameEvent
            {
                PlayerId = playerId,
                EventType = "item_acquired",
                EventCategory = EventCategory.Item,
                SessionId = sessionId,
                EventData = new Dictionary<string, object>
                {
                    { "item_id", itemId },
                    { "quantity", quantity },
                    { "source", source }, // "drop", "quest", "purchase" 등
                    { "rarity", "rare" }
                }
            };
        }

        /// <summary>
        /// 재화 변동 이벤트 생성
        /// </summary>
        public static GameEvent CreateCurrencyChangeEvent(
            long playerId,
            string sessionId,
            string currencyType,
            long amountChange,
            long balanceAfter,
            string reason)
        {
            return new GameEvent
            {
                PlayerId = playerId,
                EventType = "currency_change",
                EventCategory = EventCategory.Currency,
                SessionId = sessionId,
                EventData = new Dictionary<string, object>
                {
                    { "currency_type", currencyType }, // "gold", "diamond" 등
                    { "amount_change", amountChange },
                    { "balance_before", balanceAfter - amountChange },
                    { "balance_after", balanceAfter },
                    { "reason", reason }
                }
            };
        }

        /// <summary>
        /// 결제 이벤트 생성
        /// </summary>
        public static GameEvent CreatePaymentEvent(
            long playerId,
            string sessionId,
            string transactionId,
            decimal amount,
            string currency,
            string productId,
            bool success)
        {
            return new GameEvent
            {
                PlayerId = playerId,
                EventType = "payment",
                EventCategory = EventCategory.Payment,
                SessionId = sessionId,
                EventData = new Dictionary<string, object>
                {
                    { "transaction_id", transactionId },
                    { "amount", amount },
                    { "currency", currency }, // "USD", "KRW" 등
                    { "product_id", productId },
                    { "success", success },
                    { "payment_method", "credit_card" }
                }
            };
        }
    }
}
```

## 7.3 JSON 컬럼으로 유연한 로그 저장

JSONB 컬럼을 사용하는 것은 로그 시스템의 유연성을 크게 높인다. 새로운 이벤트 타입을 추가하거나 기존 이벤트에 새로운 필드를 추가할 때 테이블 스키마를 변경할 필요가 없다. 하지만 JSONB를 효과적으로 사용하려면 몇 가지 모범 사례를 따라야 한다.

게임 이벤트를 저장하고 조회하는 리포지토리 클래스를 작성한다.

```csharp
using System;
using System.Collections.Generic;
using System.Threading.Tasks;
using Npgsql;
using Newtonsoft.Json;
using NpgsqlTypes;

namespace GameServerLogging.Repositories
{
    /// <summary>
    /// 게임 이벤트 저장소
    /// </summary>
    public class GameEventRepository
    {
        private readonly string _connectionString;

        public GameEventRepository(string connectionString)
        {
            _connectionString = connectionString;
        }

        /// <summary>
        /// 단일 이벤트를 저장한다
        /// </summary>
        public async Task InsertEventAsync(GameEvent gameEvent)
        {
            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var sql = @"
                INSERT INTO game_events 
                (time, player_id, event_type, event_category, event_data, 
                 server_id, session_id, client_version)
                VALUES 
                (@time, @playerId, @eventType, @eventCategory, @eventData::jsonb,
                 @serverId, @sessionId, @clientVersion)";

            await using var command = new NpgsqlCommand(sql, connection);
            command.Parameters.AddWithValue("time", gameEvent.Time);
            command.Parameters.AddWithValue("playerId", gameEvent.PlayerId);
            command.Parameters.AddWithValue("eventType", gameEvent.EventType);
            command.Parameters.AddWithValue("eventCategory", gameEvent.EventCategory.ToString().ToLower());
            command.Parameters.AddWithValue("eventData", gameEvent.GetEventDataJson());
            command.Parameters.AddWithValue("serverId", gameEvent.ServerId ?? (object)DBNull.Value);
            command.Parameters.AddWithValue("sessionId", gameEvent.SessionId ?? (object)DBNull.Value);
            command.Parameters.AddWithValue("clientVersion", gameEvent.ClientVersion ?? (object)DBNull.Value);

            await command.ExecuteNonQueryAsync();
        }

        /// <summary>
        /// 배치로 여러 이벤트를 저장한다
        /// </summary>
        public async Task InsertEventsBatchAsync(List<GameEvent> events)
        {
            if (events == null || events.Count == 0)
                return;

            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var copyCommand = @"
                COPY game_events 
                (time, player_id, event_type, event_category, event_data, 
                 server_id, session_id, client_version)
                FROM STDIN (FORMAT BINARY)";

            await using var writer = await connection.BeginBinaryImportAsync(copyCommand);
            
            foreach (var evt in events)
            {
                await writer.StartRowAsync();
                await writer.WriteAsync(evt.Time, NpgsqlDbType.TimestampTz);
                await writer.WriteAsync(evt.PlayerId, NpgsqlDbType.Bigint);
                await writer.WriteAsync(evt.EventType, NpgsqlDbType.Varchar);
                await writer.WriteAsync(evt.EventCategory.ToString().ToLower(), NpgsqlDbType.Varchar);
                await writer.WriteAsync(evt.GetEventDataJson(), NpgsqlDbType.Jsonb);
                await writer.WriteAsync(evt.ServerId, NpgsqlDbType.Varchar);
                await writer.WriteAsync(evt.SessionId, NpgsqlDbType.Varchar);
                await writer.WriteAsync(evt.ClientVersion, NpgsqlDbType.Varchar);
            }

            await writer.CompleteAsync();
            Console.WriteLine($"✓ {events.Count}개 이벤트 저장 완료");
        }

        /// <summary>
        /// 특정 플레이어의 이벤트를 조회한다
        /// </summary>
        public async Task<List<GameEvent>> GetPlayerEventsAsync(
            long playerId,
            DateTime startTime,
            DateTime endTime,
            string eventType = null,
            int limit = 1000)
        {
            var events = new List<GameEvent>();

            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var sql = @"
                SELECT time, event_id, player_id, event_type, event_category, 
                       event_data, server_id, session_id, client_version
                FROM game_events
                WHERE player_id = @playerId
                  AND time BETWEEN @startTime AND @endTime";

            if (!string.IsNullOrEmpty(eventType))
            {
                sql += " AND event_type = @eventType";
            }

            sql += " ORDER BY time DESC LIMIT @limit";

            await using var command = new NpgsqlCommand(sql, connection);
            command.Parameters.AddWithValue("playerId", playerId);
            command.Parameters.AddWithValue("startTime", startTime);
            command.Parameters.AddWithValue("endTime", endTime);
            command.Parameters.AddWithValue("limit", limit);
            
            if (!string.IsNullOrEmpty(eventType))
            {
                command.Parameters.AddWithValue("eventType", eventType);
            }

            await using var reader = await command.ExecuteReaderAsync();
            while (await reader.ReadAsync())
            {
                var evt = new GameEvent
                {
                    Time = reader.GetDateTime(0),
```

## 7.4 플레이어 행동 패턴 분석

```csharp
using System.Threading.Tasks;

        /// 플레이어의 평균 세션 길이를 계산한다
        /// </summary>

        {
            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            // 더 안전한 방법: 파라미터 바인딩 사용
            var sql = @"
                WITH session_durations AS (
                    SELECT 
                        player_id,
                        session_id,
                        MIN(time) AS session_start,
                        MAX(time) AS session_end,

                    WHERE time >= NOW() - MAKE_INTERVAL(days => @days)

                    GROUP BY player_id, session_id

                    MAX(duration_minutes) AS max_duration_minutes
                FROM session_durations
                GROUP BY day
                ORDER BY day DESC";

        /// <summary>
        /// 플레이어 레벨별 진행 속도를 분석한다
        /// </summary>
        public async Task<List<dynamic>> GetPlayerProgressionAsync()
        {
            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var sql = @"
                WITH player_first_login AS (
                    SELECT 
                        player_id,
                        MIN(time) AS first_login

                    GROUP BY player_id

        /// 플레이어의 이탈 지점을 분석한다 (마지막 활동 후 일정 시간 지난 플레이어)
        /// </summary>
        public async Task<List<dynamic>> GetChurnAnalysisAsync(int inactiveDays = 7)
        {
            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var sql = @"
                WITH player_last_activity AS (
                    SELECT 
                        player_id,
                        MAX(time) AS last_activity,
                        MAX(event_data->>'level') FILTER (WHERE event_type = 'level_up') AS last_level,
                        MAX(event_data->>'dungeon_id') FILTER (WHERE event_type = 'dungeon_clear') AS last_dungeon
                    FROM game_events
                    WHERE time >= NOW() - INTERVAL '30 days'
                    GROUP BY player_id
                ),
                churned_players AS (
                    SELECT 
                        player_id,
                        last_activity,
                        last_level,
                        last_dungeon,
                        EXTRACT(EPOCH FROM (NOW() - last_activity)) / 86400.0 AS days_inactive
                    FROM player_last_activity
                    WHERE last_activity < NOW() - INTERVAL '@inactiveDays days'
                )
                SELECT 
                    CASE 
                        WHEN last_level::int BETWEEN 1 AND 10 THEN '1-10'
                        WHEN last_level::int BETWEEN 11 AND 20 THEN '11-20'
                        WHEN last_level::int BETWEEN 21 AND 30 THEN '21-30'
                        WHEN last_level::int BETWEEN 31 AND 40 THEN '31-40'
                        ELSE '41+'
                    END AS level_range,
                    COUNT(*) AS churned_count,
                    AVG(days_inactive) AS avg_days_inactive,
                    ARRAY_AGG(DISTINCT last_dungeon) FILTER (WHERE last_dungeon IS NOT NULL) AS common_last_dungeons
                FROM churned_players
                WHERE last_level IS NOT NULL
                GROUP BY level_range
                ORDER BY level_range";

            var results = new List<dynamic>();
            await using var command = new NpgsqlCommand(
                sql.Replace("@inactiveDays", inactiveDays.ToString()), 
                connection);
            await using var reader = await command.ExecuteReaderAsync();
            
            while (await reader.ReadAsync())

                results.Add(new
                {
                    LevelRange = reader.GetString(0),
                    ChurnedCount = reader.GetInt64(1),
                    AvgDaysInactive = reader.GetDouble(2),
                    CommonLastDungeons = reader.IsDBNull(3) ? new string[0] : (string[])reader.GetValue(3)
                });
            }

            return results;
        }

        /// <summary>
        /// 플레이어 코호트별 리텐션을 계산한다
        /// </summary>
        public async Task<List<dynamic>> GetCohortRetentionAsync()
        {
            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var sql = @"
                WITH player_cohorts AS (
                    SELECT 
                        player_id,
                        DATE_TRUNC('week', MIN(time)) AS cohort_week
                    FROM game_events
                    WHERE event_type = 'player_login'
                      AND time >= NOW() - INTERVAL '12 weeks'
                    GROUP BY player_id
                ),
                player_weekly_activity AS (
                    SELECT DISTINCT
                        e.player_id,
                        DATE_TRUNC('week', e.time) AS activity_week
                    FROM game_events e
                    WHERE e.time >= NOW() - INTERVAL '12 weeks'
                )
                SELECT 
                    c.cohort_week,
                    COUNT(DISTINCT c.player_id) AS cohort_size,
                    COUNT(DISTINCT CASE WHEN a.activity_week = c.cohort_week THEN c.player_id END) AS week_0,
                    COUNT(DISTINCT CASE WHEN a.activity_week = c.cohort_week + INTERVAL '1 week' THEN c.player_id END) AS week_1,
                    COUNT(DISTINCT CASE WHEN a.activity_week = c.cohort_week + INTERVAL '2 weeks' THEN c.player_id END) AS week_2,
                    COUNT(DISTINCT CASE WHEN a.activity_week = c.cohort_week + INTERVAL '3 weeks' THEN c.player_id END) AS week_3,
                    COUNT(DISTINCT CASE WHEN a.activity_week = c.cohort_week + INTERVAL '4 weeks' THEN c.player_id END) AS week_4
                FROM player_cohorts c
                LEFT JOIN player_weekly_activity a ON c.player_id = a.player_id
                GROUP BY c.cohort_week
                ORDER BY c.cohort_week DESC";

            var results = new List<dynamic>();
            await using var command = new NpgsqlCommand(sql, connection);
            await using var reader = await command.ExecuteReaderAsync();

            {
                var cohortSize = reader.GetInt64(1);
                results.Add(new
                {
                    CohortWeek = reader.GetDateTime(0),
                    CohortSize = cohortSize,
                    Week0Retention = (double)reader.GetInt64(2) / cohortSize * 100,
                    Week1Retention = (double)reader.GetInt64(3) / cohortSize * 100,
                    Week2Retention = (double)reader.GetInt64(4) / cohortSize * 100,
                    Week3Retention = (double)reader.GetInt64(5) / cohortSize * 100,
                    Week4Retention = (double)reader.GetInt64(6) / cohortSize * 100
                });
            }

            return results;
        }
```

## 7.5 아이템 경제 추적 및 분석

게임 내 경제 시스템의 건강도를 유지하는 것은 매우 중요하다. 아이템과 재화의 흐름을 추적하여 인플레이션, 디플레이션, 불균형을 조기에 발견해야 한다.

아이템 경제 분석 서비스를 작성한다.

```csharp
using System;

namespace GameServerLogging.Analytics

        /// <summary>
        /// 시간대별 재화 유입/유출을 분석한다

        {

            await connection.OpenAsync();

            var sql = @"
                SELECT 
                    time_bucket('1 hour', time) AS hour,
                    SUM((event_data->>'amount_change')::bigint) FILTER (
                        WHERE (event_data->>'amount_change')::bigint > 0
                    ) AS inflow,
                    ABS(SUM((event_data->>'amount_change')::bigint) FILTER (
                        WHERE (event_data->>'amount_change')::bigint < 0
                    )) AS outflow,
                    SUM((event_data->>'amount_change')::bigint) AS net_flow,
                    COUNT(*) AS transaction_count,
                    COUNT(DISTINCT player_id) AS unique_players
                FROM game_events
                WHERE event_type = 'currency_change'
                  AND event_data->>'currency_type' = @currencyType
                  AND time >= NOW() - INTERVAL '@days days'
                GROUP BY hour
                ORDER BY hour DESC";

            var results = new List<dynamic>();
            await using var command = new NpgsqlCommand(
                sql.Replace("@days", days.ToString()), 
                connection);
            command.Parameters.AddWithValue("currencyType", currencyType);
            
            await using var reader = await command.ExecuteReaderAsync();
            while (await reader.ReadAsync())
            {
                results.Add(new
                {
                    Hour = reader.GetDateTime(0),
                    Inflow = reader.IsDBNull(1) ? 0 : reader.GetInt64(1),
                    Outflow = reader.IsDBNull(2) ? 0 : reader.GetInt64(2),
                    NetFlow = reader.IsDBNull(3) ? 0 : reader.GetInt64(3),

        /// <summary>

                WHERE event_type = 'currency_change'
                  AND event_data->>'currency_type' = @currencyType
                  AND (event_data->>'amount_change')::bigint > 0
                  AND time >= NOW() - INTERVAL '@days days'
                GROUP BY event_data->>'reason'
                ORDER BY total_amount DESC";

            await using var reader = await command.ExecuteReaderAsync();
            while (await reader.ReadAsync())
            {
                results.Add(new

                WITH latest_balances AS (
                    SELECT DISTINCT ON (player_id)
                        player_id,

                    FROM game_events

                        WHEN balance < 100000 THEN '50k-100k'

                        WHEN balance < 1000000 THEN '500k-1M'
                        ELSE '1M+'
                    END AS balance_range,
                    COUNT(*) AS player_count,
                    AVG(balance) AS avg_balance,
                    PERCENTILE_CONT(0.5) WITHIN GROUP (ORDER BY balance) AS median_balance
                FROM latest_balances
                GROUP BY balance_range
                ORDER BY

                        WHEN '0-10k' THEN 1
                        WHEN '10k-50k' THEN 2
                        WHEN '50k-100k' THEN 3
                        WHEN '100k-500k' THEN 4
```

```csharp
        /// <summary>
        /// 비정상적으로 빠른 재화 획득을 탐지한다
        /// </summary>
        public async Task<List<dynamic>> DetectAbnormalCurrencyGainAsync(
            double thresholdMultiplier = 10.0,
            int hours = 24)
        {
            await using var connection = new NpgsqlConnection(_connectionString);
            await connection.OpenAsync();

            var sql = @"
                WITH player_hourly_gain AS (
                    SELECT 
                        player_id,
                        time_bucket('1 hour', time) AS hour,
                        SUM((event_data->>'amount_change')::bigint) FILTER (
                            WHERE (event_data->>'amount_change')::bigint > 0
                        ) AS hourly_gain
                    FROM game_events
                    WHERE event_type = 'currency_change'
                      AND event_data->>'currency_type' = 'gold'
                      AND time >= NOW() - INTERVAL '@hours hours'
                    GROUP BY player_id, hour
                ),
                avg_gain AS (
                    SELECT AVG(hourly_gain) AS platform_avg_gain
                    FROM player_hourly_gain
                    WHERE hourly_gain > 0
                )
                SELECT 
                    p.player_id,
                    p.hour,
                    p.hourly_gain,
                    a.platform_avg_gain,
                    p.hourly_gain / NULLIF(a.platform_avg_gain, 0) AS gain_ratio
                FROM player_hourly_gain p
                CROSS JOIN avg_gain a
                WHERE p.hourly_gain > a.platform_avg_gain * @thresholdMultiplier
                ORDER BY p.hourly_gain DESC
                LIMIT 100";

            var results = new List<dynamic>();
            await using var command = new NpgsqlCommand(
                sql.Replace("@hours", hours.ToString()), 
                connection);
            command.Parameters.AddWithValue("thresholdMultiplier", thresholdMultiplier);
            
            await using var reader = await command.ExecuteReaderAsync();
            while (await reader.ReadAsync())

            var sql = @"
                WITH player_action_intervals AS (
                    SELECT 
                        player_id,
                        time,
                        LAG(time) OVER (PARTITION BY player_id ORDER BY time) AS prev_time,
                        EXTRACT(EPOCH FROM (time - LAG(time) OVER (PARTITION BY player_id ORDER BY time))) AS interval_seconds
                    FROM game_events
                    WHERE event_category = 'gameplay'
                      AND time >= NOW() - INTERVAL '@hours hours'
                ),
                player_stats AS (
                    SELECT 
                        player_id,
                        COUNT(*) AS action_count,
                        AVG(interval_seconds) AS avg_interval,
                        STDDEV(interval_seconds) AS stddev_interval,
                        STDDEV(interval_seconds) / NULLIF(AVG(interval_seconds), 0) AS coefficient_of_variation
                    FROM player_action_intervals
                    WHERE interval_seconds IS NOT NULL
                      AND interval_seconds > 0
                    GROUP BY player_id
                    HAVING COUNT(*) >= 50  -- 최소 50개 이상의 액션
                )
                SELECT 
                    player_id,
                    action_count,
                    avg_interval,
                    stddev_interval,
                    coefficient_of_variation
                FROM player_stats
                WHERE coefficient_of_variation < @varianceThreshold
                  AND avg_interval BETWEEN 1 AND 10  -- 1~10초 간격
                ORDER BY coefficient_of_variation
                LIMIT 100";
```
